HTML <body> 요소는 HTML 문서의 내용을 나타냅니다. 한 문서에 하나의 <body> 요소만 존재할 수 있습니다.

콘텐츠 카테고리 구획 루트.
가능한 콘텐츠 플로우 콘텐츠.
태그 생략 태그 내의 첫번째 것이 공백, 주석, <script>요소, <style>요소가 아닐경우 시작태그를 생략할수 있습니다. body 요소가 시작태그를 가지고있고, 바로 뒤에 주석이 따라오지 않는다면 종료 태그를 생략할수 있습니다.
가능한 부모 요소 <html> 요소의 두번째 요소여야 합니다.
암시적 ARIA 역할 document
가능한 ARIA 역할 없음
DOM 인터페이스 HTMLBodyElement
  • <body> 요소는 HTMLBodyElement 인터페이스를 노출합니다.
  • document.body 속성을 통해 <body> 요소에 접근할 수 있습니다.

특성

이 요소는 전역 특성을 포함합니다.

alink
선택한 하이퍼텍스트 링크의 색. 이 방법은 적합하지 않습니다. CSS color 속성과 :active의 조합을 대신 사용하세요.
background
배경 이미지의 URI. 이 방법은 적합하지 않습니다. CSS background 속성을 대신 적용하세요.
bgcolor
배경색. 이 방법은 적합하지 않습니다. CSS background-color 속성을 대신 적용하세요.
bottommargin
아래쪽 여백. 이 방법은 적합하지 않습니다. CSS margin-bottom 속성을 대신 적용하세요.
leftmargin
왼쪽 여백. 이 방법은 적합하지 않습니다. CSS margin-left 속성을 대신 적용하세요.
link
방문하지 않은 하이퍼텍스트 링크의 색. 이 방법은 적합하지 않습니다. CSS color 속성과 :link의 조합을 대신 사용하세요.
onafterprint
사용자가 문서를 출력한 뒤 호출할 함수.
onbeforeprint
사용자가 문서 출력을 요청했을 때 호출할 함수.
onbeforeunload
문서를 처분unload하기 직전에 호출할 함수.
onblur
문서가 포커스를 상실했을 때 호출할 함수.
onerror
문서를 제대로 불러오지 못했을 때 호출할 함수.
onfocus
문서가 포커스를 받을 때 호출할 함수.
onhashchange
문서의 현재 주소 중 Fragment identifier part(해시('#') 문자로 시작)가 변경됐을 때 호출할 함수.
onlanguagechange
선호 언어 변경 시 호출할 함수.
onload
문서를 다 불러왔을 때 호출할 함수.
onmessage
문서가 메시지를 받았을 때 호출할 함수.
onoffline
네트워크 연결이 실패했을 때 호출할 함수.
ononline
네트워크 연결을 복구했을 때 호출할 함수.
onpopstate
사용자가 세션 기록을 따라 이동했을 때 호출할 함수.
onredo
사용자가 되돌리기 변경 기록(undo transaction history)에서 다시 이후로 이동했을 때 호출할 함수.
onresize
문서의 크기가 바뀔 때 호출할 함수.
onstorage
저장 영역이 변경되었을 때 호출할 함수.
onundo
사용자가 되돌리기 변경 기록(undo transaction history)에서 이전으로 이동했을 때 호출할 함수.
onunload
문서를 처분(unload) 중일 때 호출할 함수.
rightmargin
오른쪽 여백. 이 방법은 적합하지 않습니다. CSS margin-right 속성을 대신 적용하세요.
text
글자의 전경색. 이 방법은 적합하지 않습니다. CSS color 속성을 대신 적용하세요.
topmargin
위쪽 여백. 이 방법은 적합하지 않습니다. CSS margin-top 속성을 대신 적용하세요.
vlink
방문했던 하이퍼텍스트 링크의 색. 이 방법은 적합하지 않습니다. CSS color 속성과 :visited의 조합을 대신 사용하세요.

예제

<html>
  <head>
    <title>문서 제목</title>
  </head>
  <body>
    <p>문단입니다</p>
  </body>
</html>
