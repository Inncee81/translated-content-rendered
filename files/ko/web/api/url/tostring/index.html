URL.toString() 문자열화 메서드는 전체 URL을 담은 USVString을 반환합니다. 사실상 읽기 전용 URL.href인 셈입니다.

주의: 이 기능은 Web Worker에서 사용할 수 있습니다.

구문

href = url.toString()

반환 값

USVString.

예제

const url = new URL("https://developer.mozilla.org/ko/docs/Web/API/URL/toString");
url.toString(); // should return the URL as a string
