웹 콘솔에 스택 추적을 출력합니다.

주의: 이 기능은 Web Worker에서 사용할 수 있습니다.

자세한 내용과 예제는 console 내의 스택 추적을 확인하세요.

문법

console.trace();

예제

function foo() {
  function bar() {
    console.trace();
  }
  bar();
}

foo();

콘솔에 다음과 같은 추적이 표시됩니다.

bar
foo
<anonymous>
