인코딩 API는 character encodings, UTF-8이 아닌 레거시 인코딩을 포함한 다양한 문자열을 제어할 수 있는 매커니즘을 제공한다.

인코딩 API는 4가지 인터페이스를 제공한다. TextDecoder, TextEncoder, TextDecoderStream, TextEncoderStream 등 이다.

Interfaces

  • TextDecoder
  • TextEncoder
  • TextDecoderStream
  • TextEncoderStream

Tutorials & tools

  • A shim allowing to use this interface in browsers that don't support it.
