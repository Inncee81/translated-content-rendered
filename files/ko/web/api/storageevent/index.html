특성

특성 유형 설명
key DOMString 변경된 키. 저장소가 clear() 메서드로 인해 바뀐 경우 null입니다. 읽기 전용.
newValue DOMString key의 새로운 값. 저장소가 clear() 메서드로 바뀐 경우, 또는 변경점이 key를 저장소에서 삭제한 것인 경우 newValue는 null입니다. 읽기 전용.
oldValue DOMString key의 이전 값. key를 저장소에 새로 추가한 경우 이전 값이 존재하지 않으므로 oldValue는 null입니다. 읽기 전용.
storageArea nsIDOMStorage 변경된 Storage 객체입니다. 읽기 전용.
url USVString key를 바꾼 문서의 URL입니다. 읽기 전용.
