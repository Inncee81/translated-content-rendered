USVString은 유니코드 스칼라 값의 모든 가능한 시퀀스 집합에 해당합니다. USVString이 JavaScript로 전달될 땐 String으로 매핑되며, 보통 유니코드 스칼라 값 문자열을 필요로 하는 텍스트 처리 API에서만 사용합니다. USVString은 짝짓지 않은 서로게이트 코드포인트를 허용하지 않는 점을 제외하면 DOMString과 같습니다. USVString의 해당 코드포인트는 브라우저가 유니코드 "대체 문자" U+FFFD(�)로 변환합니다.
