ClipboardEvent.clipboardData 속성은 DataTransfer 객체를 가지고 있으며, 아래와 같은 목적으로 사용될 수 있습니다:

  • cut 과 copy 이벤트 핸들러로부터 어떤 데이터가 클립보드에 들어있는지, 일반적으로 setData(format, data) 함수를 호출하여 확인하는 것;
  • paste 이벤트 핸들러로부터 일반적으로 getData(format) 함수를 호출하여 붙여넣어진 데이터를 얻는 것.

더 많은 정보를 확인하려면 cut, copy, and paste 이벤트에 관한 문서를 참고하세요.

Syntax

data = ClipboardEvent.clipboardData
