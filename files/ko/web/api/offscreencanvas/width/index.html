width 프로퍼티는 OffscreenCanvas 객체에 할당된 너비를 반환합니다.

구문

var pxl = offscreen.width;
offscreen.width = pxl;

예시

새로운 offscreen 캔버스를 생성하고 높이를 설정하거나 반환합니다.

var offscreen = new OffscreenCanvas(256, 256);
offscreen.width; // 256
offscreen.width = 512;
