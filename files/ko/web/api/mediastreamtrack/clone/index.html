MediaStreamTrack 인터페이스의 clone() 메서드는 MediaStreamTrack 객체의 사본을 반환합니다. 사본은 새로운 id를 제외하면 모든 값이 원본과 동일합니다.

구문

const newTrack = track.clone()

반환 값

clone()을 호출한 MediaStreamTrack에서 id를 제외하면 똑같은 새로운 MediaStreamTrack.
