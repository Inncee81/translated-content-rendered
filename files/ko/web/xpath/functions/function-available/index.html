XSLT/XPath 참고 문서: XSLT 요소, XPath 함수, XPath 축


function-available 함수는 인수로 준 함수가 이용 가능한 지를 결정하고 boolean true나 false를 반환합니다.

구문

function-available(name )

인수

name
검사할 함수 이름.

반환

Boolean true나 false.

정의

XPath 1.0 15
