XSLT/XPath 참고 문서: XSLT 요소, XPath 함수, XPath 축


floor 함수는 10진수를 평가하고 10진수보다 크지 않는(작거나 같은) 가장 큰 정수를 반환합니다.

구문

floor(number )

인수

number
평가할 10진수.

반환

number 보다 크지 않는 가장 큰 정수.

정의

XPath 1.0 4.4
