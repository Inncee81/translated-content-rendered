특정한 컴퓨터 프로그래밍 언어에서 문자를 표현하는 데 사용되는, 문자 열 시퀀스이다.

JavaScript에서 String은 원시 값들 중 하나이고 String객체는 String primitive를 둘러싼 wrapper다.
