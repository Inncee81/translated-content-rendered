Microsoft Edge는 Microsoft Windows에 포함되어 제공되는 무료 그래픽 웹 브라우저입니다. Microsoft 는 오랜 기간 사용했던 Microsoft Internet Explorer를 대체하기 위해, 2014년부터 초기 코드명 스파르탄, 즉 Edge를 개발했습니다.
