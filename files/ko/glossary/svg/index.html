Scalable Vector Graphics (SVG)는  XML 신텍스에 기반을 둔 2D 벡터 이미지 형식(포맷)이다.

W3C는 1990년 후반부터 SVG를 사용하기 시작했지만, SVG를 지원하는 Internet Explorer 9가 출시된 후에야 널리 사용되기 시작했다. 현재 주요 웹브라우저(browsers)들은 SVG를 지원한다.

XML 신텍스를 기반으로 한 SVG는 CSS를 통해 스타일 처리가 가능하고,  자바스크립트(JavaScript)를 사용해 상호소통이 가능하다. 현재 HTML5는  SVGtags가 HTML문서에서 직접 사용 되도록 지원한다.

벡터 이미지 형식 (vector image format)으로써, SVG 그래픽은 무한정적인 크기를 가질 수 있고, 이러한 점 때문에 어떠한 화면 크기에도 적용 가능한  인터페이스 요소와 그래픽을 만들 수 있어 반응형 디자인(responsive design) 에서 아주 유용하게 여겨진다. 또한 SVG는 클리핑(clipping), 마스킹(masking), 필터(filter), 애니매이션(animation) 등 여러가지 유용한 툴 셋들을 제공한다.
