OOP (객체 지향적 프로그래밍) 는 데이터가 객체 내에 캡슐화되고 구성 요소 부분이 아닌 객체 자체가 운용되는 프로그래밍 방식이다

JavaScript 는 매우 객체 지향적인 언어이다. JavaScript는  클래스 기반의 것과는 대조적으로 프로토타입 기반의 모델을 따른다 .
