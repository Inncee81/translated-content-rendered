A propriedade Number.MAX_VALUE representa o maior valor numérico representável em JavaScript.

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

Property attributes of Number.MAX_VALUE
Writable no
Enumerable no
Configurable no

Descrição

A propriedade MAX_VALUE tem o valor de aproximadamente 1.79E+308, ou 21024. Valores maiores que MAX_VALUE são representados como Infinity.

Por MAX_VALUE ser uma propriedade estática de Number, você sempre deve usar como Number.MAX_VALUE, ao invés de uma propriedade do objeto Number que você criou. 

Exemplos

Usando MAX_VALUE

O código a seguir multiplica dois valores numéricos. Se o resultado é menor ou igual a MAX_VALUE, a função func1 é chamada; caso contrário, a função func2 é chamada.

if (num1 * num2 <= Number.MAX_VALUE) {
  func1();
} else {
  func2();
}
