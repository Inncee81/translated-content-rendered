 A função Math.cos() ,que é estatica, retorna o cosseno de um ângulo, que deve estar em radianos. A expressão é length adjacent length hypotenuse .

Sintaxe

Math.cos(x)

Parâmetros

x
Ângulo em radianos

Valor de retorno

O cosseno do número: x

Descrição

Math.cos() é um método que retorna um número entre -1 e 1, que representa o cosseno de um ângulo.

Já que cos() é um método estático de Math, você sempre deve usar Math.cos(), ao invez de criar um objeto de Math (Math não tem construtor).

Exemplos

Usando Math.cos()

Math.cos(0);           // 1
Math.cos(1);           // 0.5403023058681398

Math.cos(Math.PI);     // -1
Math.cos(2 * Math.PI); // 1
