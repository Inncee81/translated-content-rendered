A palavra-chave CSS initial CSS se aplica ao valor inicial (ou padrão) de uma propriedade para um elemento. Este valor inicial é definido pelo navegador. Este valor pode ser aplicado em qualquer propriedade CSS. Isto inclui também o atalho CSS all, no qual o initial pode ser utilizado para restaurar todas as propriedades CSS para o seu estado inicial.

Nota: Em propriedades herdadas, O valor inicial pode ser inesperado. Neste caso, considere o uso dos termos {cssxref("inherit")}}, unset, ou revert.

Exemplo

HTML

<p>
  <span>Este texto é vermelho.</span>
  <em>Este texto está em sua cor inicial (normalmente preto).</em>
  <span>Este texto está em vermelho de novo :) </span>
</p>

CSS

p {
  color: red;
}

em {
  color: initial;
}

Veja também

  • Utilize unset para definir uma propriedade para o seu valor herdado caso haja herança, ou para o seu valor inicial, caso não haja..
  • Utilize revert para redefinir uma propriedade ao valor estabelecido pela folha de estilo user-agent  (ou por seus estilos de usuário, se existir algum).
  • Utilize inherit para fazer uma propriedade de elemento a mesma de seu elemento pai.
  • A propriedade all permite que você redefina todas as propriedades para os seus estados initial, inherited, reverted, or unset de uma vez.
