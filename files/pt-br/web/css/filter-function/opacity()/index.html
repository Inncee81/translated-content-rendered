A função CSS opacity() aplica transparência às amostras na imagem inputada. Seu resultado é uma <filter-function>.

Nota: Essa função é similar à propriedade opacity mais estabelecida. A diferença é que com filtros, alguns browsers fornecem aceleração de hardware para uma melhor performance.

Sintaxe

opacity(amount)

Parâmetros

amount
A quantidade de conversão, especificada com um <number> ou uma <percentage>. Um valor de 0% é completamente transparente, enquanto que um valor de 100% mantém o input inalterado. Valores entre 0% e 100% são multiplicadores lineares do efeito. O valor de lacuna para interpolação é 1.

Exemplos

opacity(0%)   /* Completamente transparente */
opacity(50%)  /* 50% transparente */
opacity(1)    /* Nenhum efeito */
