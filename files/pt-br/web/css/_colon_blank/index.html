Rascunho
Esta página está incompleta.

Esta é uma tecnologia experimental
Verifique a tabela de compatibilidade entre Navegadores cuidadosamente antes de usar essa funcionalidade em produção.

Note: The :blank selector is considered at risk, as the CSSWG keeps changing it.

See CSSWG issue #1967.

A pseudo-classe CSS :blank seleciona elementos de entrada do usuário vazios (ex. <input> ou <textarea>).

Sintaxe

:blank
