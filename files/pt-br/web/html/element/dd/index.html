O elemento HTML <dd> fornece detalhes ou uma definição mais completa do termo precedente (definido por <dt>) numa lista de descrições (<dl>).
 
Categorias de conteúdo Nenhuma.
Conteúdo permitido Conteúdo de fluxo.
Omissão de Tag

A Tag inicial (<dd>) é obrigatória.
A Tag final (</dd>) pode ser omitida se seguida imediatamente por outro elemento <dd>, ou se o elemento pai não tiver mais conteúdo.
Ancestrais permitidos <dl> ou (em WHATWG HTML) uma <div> que está dentro de uma <dl>.
Tags irmãs <dt> ou outro elemento <dd> .
Papeis ARIA permitidos Nenhum.
Tipo de elemento DOM HTMLElement

Atributos

Este elemento inclui os atributos globais como seus.

nowrap
Se o valor deste atributo for yes, o texto de definição não será envolvido. O valor padrão é no.
