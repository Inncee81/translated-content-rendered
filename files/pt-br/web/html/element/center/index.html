Obsolete
This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

O elemento Center do HTML (<center>) se tornou obsoleto, é um elemento de nível de bloco que exibe o conteúdo em nível de bloco ou em linha centralizado horizontalmente dentro do elemento que o contém. O contêiner <body> geralmente está presente, mas não é necessário. Essa tag foi descontinuada no HTML 4 (e XHTML 1) em favor da propriedade CSS text-align, que pode ser aplicada ao elemento <div> ou à um paragráfo indivídual <p>. Para blocos de centralização, use outras propriedades CSS como margin-left e margin-right e defina-as como auto (ou defina margin para 0 auto).

Interface DOM

Este elemento implementa a interface HTMLElement.

Nota de implementação: até o Gecko 1.9.2 inclusive, o Firefox implementa a interface HTMLSpanElement para este elemento.

Exemplo 1

<center>Esse texto vai ficar centralizado.
<p>Este parágrafo também.</p></center>

O efeito da centralização vai acontecer até onde o elemento center for fechado.

Exemplo 2 (Alternativa CSS)

<div style="text-align:center">Esse texto vai ficar centralizado.
<p>Este parágrafo também.</p></div>

Isso acontece por os textos que sofreram as alterações estão dentro dos elementos que foi aplicado a propriedade.

Exemplo 3 (Alternativa CSS)

<p style="text-align:center">Essa linha vai ficar centralizada.<br>
E esta linha também.</p>

Nota

Aplicar text-align:center a um elemento <div> ou <p> centraliza o conteúdo desses elementos, deixando suas dimensões gerais inalteradas.

Compatibilidade de navegadores

A tabela de compatibilidade nesta página é gerada a partir de dados estruturados. Se você gostaria de contribuir com os dados, verifique https://github.com/mdn/browser-compat-data e nos envie uma solicitação de pull.

Apesar de obsoleto, o elemento é aceito em todos os navegadores do mercado.
