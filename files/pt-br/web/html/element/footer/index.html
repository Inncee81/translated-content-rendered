Resumo

O elemento HTML de Rodapé (<footer>) representa um rodapé para o seu sectioning content (conteúdo de seção) mais próximo ou sectioning root elemento (ou seja, seu parente mais próximo <article>, <aside>, <nav>, <section>, <blockquote>, <body>, <details>, <fieldset>, <figure>, <td>). Normalmente um rodapé contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.

Notas de Uso:

  • Coloque informações sobre o autor em um elemento <address> que pode ser incluído no elemento <footer>.
  • O elemento <footer> não é sectioning content portanto, não introduz uma nova seção no outline.

Contexto de uso:

Conteúdo permitido Flow content, mas não o <footer> ou <header> descendentes.
Omissão  de tag Nenhuma, ambas as tags de início e fim são obrigatórias.
Parent Elements permitidos

Qualquer elemento que aceite flow content. Note-se que um elemento <footer> não deve ser um descendente de um <address>, <header> ou outro elemento <footer>.
Documento normativo HTML5, section 4.4.9

Atributos

Este elemento não tem outros atributos que os global attributes, comuns a todos os elementos.

DOM Interface

Este elemento implementa a interface HTMLElement.

Exemplos

<footer>
  Algumas informações de copyright ou talvez alguma informação do autor de um <article>?
</footer>
