Obsolete
This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

O elemento de diretório HTML obsoleto (<dir>) é usado como um contêiner para um diretório de arquivos e/ou pastas, potencialmente com estilos e ícones aplicados pelo user agent. Não use este elemento obsoleto; em vez disso, você deve usar o <ul> elemento para listas, incluindo listas de arquivos.

Nota de uso: não use este elemento. Embora presente nas primeiras especificações do HTML, foi descontinuado no HTML 4 e, desde então, foi totalmente removido. Nenhum dos principais navegadores suporta esse elemento.

DOM interface

 

Este elemento implementa a interface HTMLDirectoryElement.

Atributos

Como todos os outros elementos HTML, este elemento suporta os global attributes.

compact
Este atributo booleano indica que a lista deve ser renderizada em um estilo compacto. A interpretação deste atributo depende do agente do usuário e não funciona em todos os navegadores.
Nota de uso: não use este atributo, pois ele foi descontinuado: o <dir> elemento deve ser estilizado usando  CSS. Para dar um efeito semelhante ao alcançado com o  atributo compact, a propriedade CSS line-height pode ser usado com um valor de 80%.

Compatibilidade do navegador

 
A tabela de compatibilidade nesta página é gerada a partir de dados estruturados. Se você quiser contribuir com os dados, confira https://github.com/mdn/browser-compat-data e envie-nos um pedido de pull.

Veja também

  • Outros relacionados a lista de elementos HTML: <ol>, <ul>, <li>, e <menu>;
  • Propriedades CSS que podem ser especialmente úteis para estilizar o elemento <dir> :
    • A propriedade list-style, útil para escolher a maneira como o ordinal é exibido.
    • CSS counters, útil para lidar com listas aninhadas complexas.
    • A propriedade line-height, útil para simular o atributo obsoleto compact.
    • A propriedade margin, útil para controlar o recuo da lista.
