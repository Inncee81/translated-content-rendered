O elemento linearGradient permite que os autores definam gradientes lineares para preenchimento (fill) ou contornos (stroke) de elementos gráficos.

Usage context

CategoriasElemento gradiente
Conteúdo permitidoQualquer número dos seguintes elementos, em qualquer ordem:
Elementos descritivos
<animate>, <animateTransform>, <set>, <stop>
