O elemento marker define os gráficos quem devem ser usados para desenhar setas or polymarkers em um determinado elemento <path>, <line>, <polyline> ou <polygon>.

Contexto de uso

CategoriasElemento recipiente
Conteúdo permitidoQualquer número dos seguintes elementos, em qualquer ordem:
Elementos de animação
Elementos descritivos
Elementos de forma
Elementos estruturais
Elementos de gradiente
<a>, <altGlyphDef>, <clipPath>, <color-profile>, <cursor>, <filter>, <font>, <font-face>, <foreignObject>, <image>, <marker>, <mask>, <pattern>, <script>, <style>, <switch>, <text>, <view>
