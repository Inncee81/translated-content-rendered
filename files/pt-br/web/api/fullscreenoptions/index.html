O dicionário FullscreenOptions é usado para prover configurações quando chamado requestFullscreen() em um elemento para colocar este elmento em modo full-screen (tela inteira).

Propriedades

navigationUIOptional
Uma string controlando se deve ou não manter elementos da interface do usuário do navegador visíveis enquanto estiver no modo full-screen. O modo padrão, "auto", deixa que o navegador faça esta decisão.
