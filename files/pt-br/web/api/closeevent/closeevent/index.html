O construtor CloseEvent()cria uma nova instância CloseEvent.

Sintaxe

var event = new CloseEvent(typeArg, closeEventInit);

Valores

typeArg
É uma DOMString que representa o nome do evento.
closeEventInit Optional
É uma coleção CloseEventInit, que possui os seguintes campos:
  • "wasClean", opcional e inicialmente de valor false, do tipo long, indica se uma conexão foi encerrada de forma limpa ou não.
  • "code", opcional e inicialmente de valor 0, do tipo unsigned short, este é o código de encerramento da conexão enviado pelo servidor.
  • "reason", opcional e inicialmente de valor '', do tipo DOMString, esta é uma razão humanamente compreensível do porque o servidor encerrou a conexão.

A coleção CloseEventInit também aceita campos vindos da coleção EventInit.
