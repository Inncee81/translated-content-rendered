A interface ClipboardEvent representa eventos correlatos à modificação da área de transferência, i. e., eventos de cortar (cut), de copiar (copy), e de colar (paste).

Construtor

ClipboardEvent()
Cria um evento ClipboardEvent com os parâmetros fornecidos.

Propriedades

Ainda, propriedades herdadas de sua superclasse Event.

ClipboardEvent.clipboardData Somente leitura
é um objeto DataTransfer contendo informação oriunda das operações de cut, de copy, ou de paste iniciadas pelo usuário, juntamente com sua espécie/tipo MIME.

Methods

Não há método próprio; métodos herdados de Event.
