A propriedade onerror da interface EventSource é um EventHandler chamado quando um erro ocorre e um evento error é despachado para o objeto EventSource.

Sintaxe

eventSource.onerror = function

Exemplos

evtSource.onerror = function() {
  console.log("EventSource failed.");
};

Nota: Você pode encontrar um exemplo completo no GitHub — veja Simple SSE demo using PHP.
