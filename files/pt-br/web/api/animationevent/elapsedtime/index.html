Sumário

O AnimationEvent.elapsedTime é uma propriedade só de leitura com um ponto flutuante dando a quantidade de tempo que animação esteve rodando, em segundos, quando o evento termina, excluirá o tempo em que ela esteve em pausa. Para um evento "animationstart" , elapsedTime é 0.0 a não ser que tenha um valor negativo para animation-delay, nesse caso o evento irá terminar com elapsedTime contendo (-1 * delay).

Síntaxe

time = AnimationEvent.elapsedTime
