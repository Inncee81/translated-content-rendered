O Event() cria uma nova Event.

Sintaxe

 event = new Event(typeArg, eventInit);

Valores

typeArg
É uma DOMString representa o nome do evento.
eventInitOptional
É um dicionário EventInit, tendo os seguintes campos:
  • "bubbles", opcional e false por default, do tipo Boolean, indica se o evento é bubbles ou não.
  • "cancelable", opcional e false por default, do tipo Boolean, indica se o evento pode ser cancelado ou não.

Exemplo

// criar um evento com bubbles true e que não pode ser cancelado

var ev = new Event("look", {"bubbles":true, "cancelable":false});
document.dispatchEvent(ev);
