A propriedade port do SharedWorker retorna um objeto MessagePort usado para comunicar e controlar o shared worker.

 

Sintaxe

myWorker.port;

Valor

Um objeto MessagePort.

Exemplo

O snippet de código a seguir mostra a criação de um objeto SharedWorker usando o construtor SharedWorker () . Vários scripts podem acessar o worker através de um objeto do tipo MessagePort , usando a propriedade SharedWorker.port  — A porta é iniciada usando o método start() .

var myWorker = new SharedWorker('worker.js');
myWorker.port.start();
