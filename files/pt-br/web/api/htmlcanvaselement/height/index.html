A propriedade HTMLCanvasElement.height é um inteiro positivo que reflete o atributo height do elemento HTML <canvas> interpretado em pixels no CSS. Quando o atributo não é especificado, ou se for definido como um valor inválido, como um inteiro negativo, o valor padrão de 150 será usado.

Essa é uma de duas propriedades, que controla o tamanho do canvas, sendo a outra HTMLCanvasElement.width.

Sintaxe

var pxl = canvas.height;
canvas.height = pxl;

Exemplos

Dado este elemento <canvas>:

<canvas id="canvas" width="300" height="300"></canvas>

Você pode capturar a altura do canvas com o seguinte código:

var canvas = document.getElementById('canvas');
console.log(canvas.height); // 300
