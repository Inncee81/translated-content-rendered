Esta é uma tecnologia experimental
Verifique a tabela de compatibilidade entre Navegadores cuidadosamente antes de usar essa funcionalidade em produção.

O NavigatorPlugins mixin adiciona na interface do Navigator métodos e propriedades para descobrir e interagir com plugins instalados no navegador.

 

Propriedades

NavigatorPlugins.mimeTypes Somente leitura
Retorna um MimeTypeArray listando os tipos MIME suportados pelo navegador.
NavigatorPlugins.plugins Somente leitura
Retorna um PluginArray listando os plugins instalados no navegador.

Métodos

A interface NavigatorPlugins NavigatorPlugins não herda nenhum método.

NavigatorPlugins.javaEnabled() Somente leitura
Retorna uma flag Boolean indicando se o navegador do host tem o Java ativo ou não.
