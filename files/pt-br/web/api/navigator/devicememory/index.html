A propriedade de somente-leitura deviceMemory da interface navigator retorna a quantidade de memória do dispositivo em gigabytes. Este valor é uma aproximação por arredondamento da potência mais próxima de 2 e dividindo este número por 1024.

Sintaxe

var memory = navigator.deviceMemory

Valor

Um número de ponto flutuante.
