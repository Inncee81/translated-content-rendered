A propriedade WindowEventHandlers.onstorage contém um manipulador de eventos que executa quando o evento storage dispara. Isto ocorre quando a área de armazenamento é mudada. (ex. um novo item é armazenado.)

Sintaxe

windowObj.onstorage = function() { ... };

Exemplos

window.onstorage = function(e) {
  console.log('The ' + e.key + ' key has been changed from ' + e.oldValue + ' to ' + e.newValue + '.');
};
