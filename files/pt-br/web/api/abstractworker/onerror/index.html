Resumo

A propriedade AbstractWorker.onerror da interface AbstractWorker representa um EventHandler, que é uma função a ser chamada quando o evento error ocorre através de um balão de notificação Worker.

Sintaxe

myWorker.onerror = function() { ... };

Exemplo

O seguinte trecho de código mostra criação de um objeto Worker usando o construtor de Worker() e criando um manipulador onerror no objeto resultante:

var myWorker = new Worker("worker.js");

myWorker.onerror = function() {
  console.log('Há um erro com o seu worker!');
}
