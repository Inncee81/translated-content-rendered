A propriedade length, que é somente leitura, da interface Storage retorna um inteiro que representa o número de itens armazenados no objeto Storage.

Sintaxe

var aLength = storage.length;

Retorno

Um inteiro

Exemplo

A função a seguir adiciona três itens ('bgcolor', 'font' e 'image') ao local storage do domínio atual, em seguida retorna o número de itens no storage:

function populateStorage() {
  localStorage.setItem('bgcolor', 'yellow');
  localStorage.setItem('font', 'Helvetica');
  localStorage.setItem('image', 'cats.png');

  localStorage.length; // should return 3
}
