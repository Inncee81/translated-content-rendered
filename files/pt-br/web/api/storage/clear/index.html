O método clear() da interface Storage, quando realizado uma chamada, todas as chaves do armazenamento são esvaziadas.

Syntaxe

storage.clear();

Parametros

Nenhum parametro..

Retorno

Nenhum retorno.

 

Exemplo

 

A função abaixo cria três itens e armazenam localmente, depois remove todos utilizando a função clear().

function populateStorage() {
  localStorage.setItem('bgcolor', 'red');
  localStorage.setItem('font', 'Helvetica');
  localStorage.setItem('image', 'myCat.png');

  localStorage.clear();
}
