Esta é uma tecnologia experimental
Verifique a tabela de compatibilidade entre Navegadores cuidadosamente antes de usar essa funcionalidade em produção.

A interface XMLDocument representa um document XML. Ele herda de Document e não adiciona nenhum método específico ou propriedades para este: no entanto, alguns algoritmos tem comportamento diferente com os dois tipos de documentos.

Propridades

Esta interface não tem e nem herda nenhuma propriedade específica.

Metodos

Esta interface não adiciona nenhum método novo.

XMLDocument.load()   
Carrega um documento XML
