A interface HTMLShadowElement representa um  elemento HTML <shadow>, Ao qual é usado no Shadow DOM.

Propriedades

 Esta interface herda as propriedades do HTMLElement.

Métodos

Esta interface herda os métodos do HTMLElement.

HTMLShadowElement.getDistributedNodes()
Retorna uma estática NodeList do distributed nodes associado com este elemento <shadow>.
