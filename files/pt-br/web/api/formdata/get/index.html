O método get() da interface FormData retorna o primeiro valor associado com a chave fornecida de um objeto do tipo FormData.

Nota: Este método está disponível em Web Workers.

Sintaxe

formData.get(nome);

Parâmetros

nome
Um USVString representando o nome da chave que queira retornar.

Retorno

Um FormDataEntryValue contendo o valor.

Exemplo

A linha abaixo cria um objeto vazio do tipo FormData.

var formData = new FormData();

Se adicionarmos dois valores usuario_nome usando FormData.append:

formData.append('usuario_nome', 'Sabrina');
formData.append('usuario_nome', 'Antônio');

A função get() abaixo apenas irá retornar o primeiro valor usuario_nome incluido:

formData.get('usuario_nome'); // Retorna "Sabrina"
