L'ultimo rilascio stabile di HTML, HTML5, porta HTML da un semplice linguaggi di markup per strutturare un document a una completa piattaforma di sviluppo app. Tra le altre caratteristiche, HTML5 include nuovi elementi e nuove JavaScript APIs per migliorare l'accesso allo storage, ai multimedia e all'hardware.
