Sommario

L'elemento HTML <html> (o elemento radice HTML) rappresenta la radice di un documento HTML. Tutti gli altri elementi devono esserne discendenti.

  • Categorie di contenuti Nessuna.
  • Contenuto permesso Un elemento <head>, seguito da un elemento <body>.
  • Omissione del tag Il tag di apertura può essere omesso se la prima cosa all'interno dell'elemento <html> non è un commento.
    Il tag di chiusura può essere omesso se l'elemento <html> non è immediatamente seguito da un commento e contiene un elemento <body> che non sia vuoto o il cui tag di apertura sia presente.
  • Elementi genitore permessiCome l'elemento radice di un documento, o in qualsiasi caso un frammento di un sottodocumento sia permesso in un documento composto.
  • Interfaccia DOM HTMLhtmlElement

Attributi

Questo elemento include gli attributi globali.

manifest
Specifica l'URI di un manifesto di risorse che indica le risorse da inserire nella cache locale. Vedi Usare la cache delle applicazioni per ulteriori dettagli.
version
Specifica la versione del HTML Document Type Definition che governa il documento corrente. Questo attributo non è necessario, perchè è ridondante  con l'informazione di versione nella dichiarazione del tipo di documento.
xmlns 
Specifica il Namespace XML di un documento. il valore di default è "http://www.w3.org/1999/xhtml". E' richiesto nei documenti analizzati con un parser XML, e opzionale nei documenti text/html.

 

Esempio

Il DOCTYPE utilizzato nell'esempio seguente indica HTML5.

<!DOCTYPE html>
<html>
  <head>...</head>
  <body>...</body>
</html>
