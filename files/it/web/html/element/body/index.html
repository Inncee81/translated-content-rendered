Sommario

L'elemento HTML <body> rappresenta il contenuto di un documento HTML. Può essere presente solo un elemento <body> in un documento.

  • Categorie di contenuto Sezioni principali.
  • Contenuti permessi Contenuti di flusso.
  • Omissione del tagIl tag iniziale può essere omesso se la prima cosa all'interno di esso non è uno spazio, un commento, un elemento <script> o un elemento <style>. Il tag finale può essere omesso se l'elemento <body> ha dei contenuti o ha il tag iniziale che non sia immediatamente seguito da un commento.
  • Elementi genitore permessiDeve essere il secondo elemento figlio di un elemento <html>.
  • Interfaccia DOM HTMLBodyElement
    • L'elemento <body> espone l'interfaccia HTMLBodyElement.
    • Puoi accedere all'elemento <body> di una pagina attraverso document.body.

Attributi

Questo elemento include gli attributi globali.

alink
Il colore del testo dei link quando vengono selezionati. Questo metodo non è conforme, usa la proprietà CSS color insieme alla pseudo-classe :active sugli elementi <a>​.
background
L'URI di un'immagine da usare come sfondo. Questo metodo non è conforme, usa la proprietà CSS background sull'elemento <body>.
Nota: A partire da Gecko 7.0 (Firefox 7.0 / Thunderbird 7.0 / SeaMonkey 2.4), background non è più trattato come URI, ma come una semplice stringa.
bgcolor
Il colore di sfondo del documento. Questo metodo non è conforme, usa la proprietà CSS background-color sull'elemento <body>.
bottommargin
Il margine inferiore del corpo del documento. Questo metodo non è conforme, usa la proprietà CSS margin-bottom sull'elemento <body>.
leftmargin
Il margine sinistro del corpo del documento. Questo metodo non è conforme, usa la proprietà CSS margin-left sull'elemento <body>.
link
Il colore del testo dei link non ancora visitati. Questo metodo non è conforme, usa la proprietà CSS color insieme alla pseudo-classe :link sugli elementi <a>.
onafterprint HTML5
Una funzione da richiamare dopo che l'utente abbia stampato il documento.
onbeforeprint HTML5
Una funzione da chiamare subito prima che il documento venga stampato.
onbeforeunload HTML5
Una funzione da chiamare quando il documento sta per essere chiuso.
onblur HTML5
Una funzione da chiamare quando il documento perde il focus.
onerror HTML5
Una funzione da chiamare quando il documento non riesce a caricarsi correttamente.
onfocus HTML5
Una funzione da chiamare quando il documento riceve il focus.
onhashchange HTML5
Function to call when the fragment identifier part (starting with the hash ('#') character) of the document's current address has changed.
onlanguagechange
Una funzione da chiamare quando la lingua preferita cambia.
onload HTML5
Una funzione da chiamare quando termina il caricamento della pagina.
onmessage HTML5
Una funzione da chiamare quando il documento riceve un messaggio.
onoffline HTML5
Una funzione da chiamare quando il browser si disconnette da internet.
ononline HTML5
Una funzione da chiamare quando il browser si riconnette a internet.
onpopstate HTML5
Una funzione da chiamare quando l'utente si sposta nella cronologia.
onredo HTML5
Una funzione da chiamare quando l'utente si sposta in avanti nella cronologia.
onresize HTML5
Una funzione da chiamare quando la pagina viene ridimensionata.
onstorage HTML5
Una funzione da chiamare quando lo spazio di archiviazione a disposizione cambia.
onundo HTML5
Una funzione da chiamare quando l'utente si sposta in dietro nella cronologia.
onunload HTML5
Una funzione da chiamare quando il documento viene chiuso.
rightmargin
Il margine destro del corpo del documento. Questo metodo non è conforme, usa la proprietà CSS margin-right sull'elemento <body>.
text
Il colore del testo. Questo metodo non è conforme, usa la proprietà CSS color sull'elemento <body>.
topmargin
Il margine superiore del corpo del documento. Questo metodo non è conforme, usa la proprietà CSS margin-top sull'elemento <body>.
vlink
Il colore del testo dei link già visitati. Questo metodo non è conforme, usa la proprietà CSS color insieme alla pseudo-classe :visited sugli elementi <a>​.
