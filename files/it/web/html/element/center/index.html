Deprecato
This feature is no longer recommended. Though some browsers might still support it, it may have already been removed from the relevant web standards, may be in the process of being dropped, or may only be kept for compatibility purposes. Avoid using it, and update existing code if possible; see the compatibility table at the bottom of this page to guide your decision. Be aware that this feature may cease to work at any time.

Sommario

L'elemento HTML <center> è un elemento che può contenere paragrafi o altri elementi, che siano blocchi o inline. Il contenuto di questo elemento viene centrato orizzontalmente rispetto al suo contenitore (solitamente l'elemento <body>).

Questo tag è deprecato a partire da HTML 4 (e XHTML 1) in favore della proprietà CSS text-align, che può essere applicata ad un elemento <div> o ai singoli elementi <p>. Per centrare i blocchi usa altre proprietà CSS comemargin-left e margin-right, e impostale ad auto (o importa margin a 0 auto

Interfaccia DOM

Questo elemento implementa l'interfaccia HTMLElement.

Nota: fino a Gecko 1.9.2 incluso, FIrefox implementa l'interfaccia HTMLSpanElement.

Esempio 1

<center>Questo testo verrà centrato
<p>E anche questo paragrafo.</p></center>

Esempio 2 (usando i CSS)

<div style="text-align:center">Questo testo verrà centrato.
<p>E anche questo paragrafo.</p></div>

Esempio 3 (usando i CSS)

<p style="text-align:center">Questa linea verrà centrata.<br>
E anche questa.</p>

Nota

Applicando text-align: center a un elemento <div> o <p> si centra il contenuto di questi elementi, non gli elementi stessi.
