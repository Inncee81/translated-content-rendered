Sommario

L'elemento HTML <footer> rappresenta il footer della sua sezione più vicina. Un footer solitamente contiene informazioni sull'autore della sezione, il copyright o link a documenti correlati.

Note d'uso:

  • Racchiudere le informazioni sull'autore in un elemento <address>, che può essere posizionato nell'elemento <footer>
  • L'elemento <footer> non rappresenta una sezione del contenuto e quindi non introduce una nuova sezione nell'indice.
  • Categorie di contenuto Contenuti di flusso, contenuti palpabili.
  • Contenuti permessi Contenuti di flusso, esclusi <footer> e <header>.
  • Omissione del tag Nessuna, sia il tag di apertura che quello di chiusura sono obbligatori.
  • Elementi genitore permessi Qualsiasi elemento che accett contenuti di flusso. Notare che un elemento <footer> non può essere discendente di un elemento <address>, <header> o di un altro <footer>.
  • Interfaccia DOM HTMLElement

Attributi

Questo elemento include solo gli attributi globali.

Esempi

<footer>
  Il copyright o delle informazioni su di un articolo
</footer>
