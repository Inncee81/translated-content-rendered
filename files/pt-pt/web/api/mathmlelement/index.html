  1. MathMLElement

A interface MathMLElement representa qualquer elemento MathML.

Propiedades

Esta interface não tem propriedades, mas herda propriedades de: DocumentAndElementEventHandlers, Element, ElementCSSInlineStyle, GlobalEventHandlers, HTMLOrForeignElement

Métodos

Esta interface não tem métodos, mas herda métodos de: DocumentAndElementEventHandlers, Element, ElementCSSInlineStyle, GlobalEventHandlers, HTMLOrForeignElement

Exemplos

MathML

<math xmlns="http://www.w3.org/1998/Math/MathML">
  <msqrt>
    <mi>x</mi>
  </msqrt>
</math>

JavaScript

document.querySelector('msqrt').constructor.name; // MathMLElement
