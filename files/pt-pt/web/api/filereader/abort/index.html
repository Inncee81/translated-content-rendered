O método abort aborta a operação de leitura. Ao completar o abortamento, o readyState é DONE.

Sintaxe

instanceOfFileReader.abort();

Exceções

DOM_FILE_ABORT_ERR
Lançado quando abort é chamado enquanto não há uma operação de leitura em progresso (ou seja, quando o state não é LOADING).
