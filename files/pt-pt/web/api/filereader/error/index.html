A propriedade error de FileReader retorna o erro que ocorreu ao ler o ficheiro.

Sintaxe

var error = instanceOfFileReader.error

Valor

Um DOMError com o erro. Em Chrome 48+/Firefox 58+ esta propriedade retorna uma DOMException porque DOMError foi removido do standard do DOM.
