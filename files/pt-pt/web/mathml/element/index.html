Isto é uma lista alfabética de elementos de presentação MathML.

O termo presentation markup é usado para descrever o layout da estrutura da notação matemática enquanto content markup oferece o significado matemático e não é suposto ser visualizado pelo processador MathML (veja bug 276028). Se deseja aprender mais sobre content markup deve ver Chapter 4 no MathML 3 specification.

Elementos MathML implementam a classe MathMLElement.

Elementos de apresentação MathML de A a Z

math

  • <math> (Elemento de raiz)

A

  • <maction> (Ações associadas a subexpressões)
  • <maligngroup> (Grupo de alinhamento)
  • <malignmark> (Pontos de alinhamento)

E

  • <menclose> (Conteúdos enclausurados)
  • <merror> (Conteúdos enclausurados de sintaxe de mensagens de erro)

F

  • <mfenced> (Parênteses)
  • <mfrac> (Fração)

G

  • <mglyph> (Apresentar símbolos não standard)

I

  • <mi> (Identificador)

L

  • <mlabeledtr> (Fila marcada numa tabela ou matriz)
  • <mlongdiv> (Notação de divisão longa)

M

  • <mmultiscripts> (Prescripts e índices de tensores)

N

  • <mn> (Numero)

O

  • <mo> (Operador)
  • <mover> (Overscript)

P

  • <mpadded> (Espaço à volta de conteúdo)
  • <mphantom> (Conteúdo invisível com espaço reservado)

R

  • <mroot> (Radical com índice especidicado)
  • <mrow> (Sub-expressõs agrupadas)

S

  • <ms> (String literal)
  • <mscarries> (Anotações como carries)
  • <mscarry> (Carry, elemento pertencente a <mscarries>)
  • <msgroup> (Filas agrupadas dos elementos <mstack> e <mlongdiv>)
  • <msline> (Linhas horizontais dentro de elementos <mstack>)
  • <mspace> (Espaço)
  • <msqrt> (Raiz quadrada sem índice)
  • <msrow> (Filas dentro de elementos <mstack>)
  • <mstack> (Alinhamento em pilha)
  • <mstyle> (Mudar estilo)
  • <msub> (Subscript)
  • <msup> (Superscript)
  • <msubsup> (Par de subscript-superscript)

T

  • <mtable> (Tabela ou matriz)
  • <mtd> (Célula numa tabela ou matriz)
  • <mtext> (Texto)
  • <mtr> (Fila numa tabela ou matriz)

U

  • <munder> (Underscript)
  • <munderover> (Par de underscript-overscript)

Outros elementos

  • <semantics> (Contentor para anotações semânticas)
  • <annotation> (Anotações de dados)
  • <annotation-xml> (Anotações de XML)
