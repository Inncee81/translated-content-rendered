O elemento MathML <mlabeledtr> é utilizado para representar um rótulo/etiqueta numa fila quer do lado esquerdo, quer do direito (determinado utilizando o atributo side no elemento <mtable>). Os elementos criança de <mlabeledtr> devem ser <mtd> elementos. A primeira criança é o rótulo, enquanto todas as outras crianças representam conteúdos de linha e são tratadas de forma idêntica às crianças de elementos <mtr>.

Atributos

class, id, style
Para uso com stylesheets.
columnalign
Indica o alinhamento horizontal desta célula e sobrepõe o valor especificado por <mtable>.
Os valores possíveis são: left, center e right.
groupalign
Unimplemented
href
Usado para criar uma hiperligação para um URI.
mathbackground
A cor de fundo. Pode usar #rgb, #rrggbb e nomes de cores de HTML.
mathcolor
A cor do texto. Pode usar #rgb, #rrggbb e nomes de cores de HTML.
rowalign
Indica o alinhamento vertical desta célula e sobrepõe o valor especificado por <mtable>.
Os valores possíveis são: axis, baseline, bottom, center e top.

Exemplos

Exemplo: 

<math>
  <mtable>

    <mlabeledtr>
      <mtd><mn>1</mn></mtd> <!-- label -->
      <mtd><mi>A</mi></mtd>
      <mtd><mi>B</mi></mtd>
    </mlabeledtr>

    <mtr>
      <mtd><mi>C</mi></mtd>
      <mtd><mi>D</mi></mtd>
      <mtd><mi>E</mi></mtd>
    </mtr>

  </mtable>
</math>

Notas para Gecko

A começar com Gecko 9.0 (Firefox 9.0 / Thunderbird 9.0 / SeaMonkey 2.6) etiquetas são escondidas por pré-definição (mlabeledtr > mtd:first-child {display: none}) e não estragam a apresentação de tabelas por completo (ver bug 356870). Para ver a implementação deste elemento veja bug 689641.
