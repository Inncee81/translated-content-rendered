1 MathML Landing, MathML, Referencia, Web, XML
MathML (Linguagem de Marcação Matemática) é um dialeto de XML para descrever a notação matemática e capturar ambos na sua estrutura e conteúdo.

10 <math> MathML, MathML:Element, Referência MathML
O elemento top-level em MathML é <math>. Qualquer instancia valida de MathML tem de ser envolto por <math>. Ademais não se deve por um elemento de <math> dentro de outro, mas pode se ter qualquer numero de outros elementos dentro desse elemento.
11 <menclose> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento de MathML <menclose> apresenta os seus conteúdos dentro da notação indicada pelo atributo notation.
12 <merror> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento de MathML <merror> é usado para mostrar os seus conteúdos como mensagens de erro. No Firefox esta mensagem é renderizada de forma parecida à mensagem de erro de XML. Note que este erro não aparece quando a linguagem de marcação (markup) está errada ou o XML está mal formatado. Irá receber um erro de processamento do XML (no caso de notação XHTML de MathML), que não é relacionado ao  <merror>.
13 <mfenced> Depreciado, MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento depreciado <mfenced> costumava oferecer a possibilidade de adicionar parênteses e separadores personalizados a uma expressão. Foi removido das normas mais recente de MathML e navegadores modernos já não o apoiam.
14 <mfrac> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento MathML <mfrac> é usado para apresentar frações.
15 <mglyph> MathML, MathML:Element, Referência de MathML
O elemento de MathML <mglyph> é usado para apresentar símbolos sem carateres de Unicode disponíveis. Pode ser usado dentro de token elements.
16 <mi> MathML, MathML:Element, MathML:Token Elements, Referência de MathML
O elemento de MathML <mi> indica se o conteúdo deve ser apresentado como um identificador como nomes de funções, variáveis ou constantes simbólicas. Também pode ter texto arbitrário para a marcação de termos.
17 <mlabeledtr> MathML, MathML:Element, MathML:Tabular Math, Referência de MathML
O elemento MathML <mlabeledtr> é utilizado para representar um rótulo/etiqueta numa fila quer do lado esquerdo, quer do direito (determinado utilizando o atributo side no elemento <mtable>). Os elementos criança de <mlabeledtr> devem ser <mtd> elementos. A primeira criança é o rótulo, enquanto todas as outras crianças representam conteúdos de linha e são tratadas de forma idêntica às crianças de elementos <mtr>.
18 <mmultiscripts> MathML, MathML:Element, MathML:Script and Limit Schemata, Referência de MathML
O elemento MathML <mmultiscripts> permite a criação de objetos de tipo tensor. Tensores são entidades geométricas introduzidas na matemática e na física para generalizar a noção de escalares, vetores e matrizes. O grau de um tensor depende das dimensões duma matriz representativa. Por exemplo, um número é uma matriz de 0 dimensões, ou um tensor de 0ª ordem. Uma matriz unidimensional (por exemplo, vetores) é um tensor de 1ª ordem, por isso são necessários tensores de 2ª ordem para representar matrizes quadradas.
19 <mn> MathML, MathML:Element, MathML:Token Elements, Referência de MathML
O elemento de MathML <mn> representa uma numeric literal que tipicamente é uma sequência de dígitos com um possível separador (um ponto ou uma vírgula). Contudo, também é permissível ter um texto arbitrário que representa um número, por exemplo "onze".
20 <mo> MathML, MathML:Element, MathML:Token Elements, Referência de MathML
O elemento de MathML <mo> representa um operador. Para além de operadores de matemática, este elemento inclui "operadores" como parênteses, separadores como a vírgula ou ponto e virgula, ou barras do "valor absoluto".
21 <mover> MathML, MathML:Element, MathML:Script and Limit Schemata, Referência de MathML
O elemento de MathML <mover> é usado para adicionar um acento ou limite sob uma expressão. Use antes a sintaxe seguinte: <mover> base overscript </mover>
22 <mpadded> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento MathML <mpadded> é usado para adicionar extra padding e para ajustar a posição e o tamanho de elementos enroupados.
23 <mphantom> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento de MathML <mphantom> é feito invisível, mas as suas dimensões (como a altura, largura e posição base) são mantidas.
24 <mroot> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento de MathML <mroot> apresenta raízes com um índice explicito. Dois argumentos são aceites, com a sintaxe seguinte: <mroot> base index </mroot>.
25 <mrow> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento de MathML <mrow> é utilizado para agrupar sub-expressões, que normalmente contêm um ou mais operadores com os seus respetivos operandos (tais como <mi> e <mn>). Este elemento é apresentado como uma linha horizontal contendo os seus argumentos.
26 <ms> MathML, MathML:Element, MathML:Token Elements, Referência de MathML
O elemento de MathML <ms> representa uma string literal que deve ser intrepetada por linguagens de programação e sistemas de álgebra informática. Por norma, os string literals são exibidos como delimitados por aspas duplas (&quot;); utilizando os atributos lquote e rquote, é possível definir caracteres personalizados para exibir. Note que as aspas não devem ser especificadas a menos que façam parte da cadeia de caracteres literal. O conteúdo de um elemento <ms> não é uma string de ASCII, mas sim uma sequência de caracteres e os elementos <mglyph> e <malignmark>.
27 <mspace> MathML, MathML:Element, MathML:Token Element, Referência de MathML
O elemento de MathML <mspace> é usado para visualizar um espaço em branco, cujo tamanho é decidido pelos seus atributos.
28 <msqrt> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento de MathML <msqrt> é usado para apresentar raízes quadradas (sem índice à vista). A raiz quadrada aceita só um argumento, com a sintaxe seguinte: <msqrt> base </msqrt>.
29 <mstyle> MathML, MathML:Element, MathML:General Layout Schemata, Referência de MathML
O elemento MathML <mstyle> é utilizado para mudar o estilo dos seus elementos filhos. Aceita todos os atributos de todos os elementos de apresentação MathML com algumas excepções e atributos adicionais listados abaixo.
30 <msub> MathML, MathML:Element, MathML:Script and Limit Schemata, Referência de MathML
O elemento de MathML <msub> para adicionar um subscript a uma expressão.
31 <msubsup> MathML, MathML:Element, MathML:Script and Limit Schemata, Referência de MathML
O elemento de MathML <msubsup> é usado para adicionar um subscript e um superscript, juntos, a uma expressão.
32 <msup> MathML, MathML:Element, MathML:Script and Limit Schemata, Referência de MathML
O elemento de MathML <msup> é usado para adicionar um superscript a uma expressão.
33 <mtable> MathML, MathML:Element, MathML:Tabular Math, Referência de MathML
O elemento MathML <mtable> permite criar tabelas e matrizes. Dentro de um <mtable> só podem aparecer os elementos <mtr> e <mtd>. Estes elementos são semelhantes aos elementos de HTML <table> , <tr> e <td>.
34 <mtd> MathML, MathML:Element, MathML:Tabular Math, Referência de MathML
O elemento <mtd> de MathML representa uma célula numa tabela ou matriz. Só pode aparecer dentro do elemento <mtr>. Este elemento é semelhante ao elemento <td> de HTML.
35 <mtext> MathML, MathML:Element, MathML:Token Elements, Referência de MathML
O elemento de MathML <mtext> é usado para apresentar texto sem significado notacional, como comentários e anotações.
36 <mtr> Math:Element, MathML, MathML:Tabular Math, Referência de MathML
O elemento de MathML, <mtr>, representa uma linha numa tabela ou matriz. Este elemento só pode aparecer no elemento <mtable>. Este elemento é semelhante ao elemento <tr> de HTML.
37 <munder> MathML, MathML:Element, MathML:Script and Limit Schemata, Referência de MathML
O elemento de MathML <munder> é usado para adicionar um acento ou limite abaixo duma expressão. Usa a seguinte sintaxe: <munder> base underscript </munder>
38 <munderover> MathML, MathML:Element, MathML:Script and Limit Schemata, Referência de MathML
O elemento de MathML <munderover> é usado para adicionar acentos ou limites acima e abaixo duma expressão.
39 <semantics> MathML, MathML:Element, Referência de MathML
O elemento <semantics> atua como um elemento contentor que associa anotações e então deve ter elementos (senão levanta um erro de marcação inválido). O elemento <annotation> é o elemento contentor que contem informação semântica num formato não XML, enquanto o elemento <annotation-xml> contém conteúdo num formato XML, por exemplo, Content MathML ou OpenMath.
