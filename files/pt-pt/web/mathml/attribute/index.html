Isto é uma lista alfabética de atributos de MathML. Mais detalhes para cada atributo estão disponiveis em certas páginas de elementos.

Notes:

  • Os elementos de MathML <mstyle> e <math> aceitam todos os atributos de elementos de apresentação de MathML.
  • Veja valores para notas sobre valores e unidades de MathML.
  • Os atributos background, color, fontfamily, fontsize, fontstyle, fontweight e xlink:href estão obsoletos.
  • Os atributos de XLink em elementos de MathML: xlink:actuate, xlink:href, xlink:show e xlink:type estão obsoletos também.
