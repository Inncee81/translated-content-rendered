The length read-only property of the DOMTokenList interface is an integer representing the number of objects stored in the object.

بنية الجملة

tokenList.length;

القيمة العائدة

تُعيد رقم صحيح.

أمثلة

في المثال التالي نقوم بإسترداد قيمة الـ classes الموضوعة داخل <span> element as a DOMTokenList using Element.classList, then write the length of the list to the <span>'s Node.textContent.

أولاً، الـ HTML:

<span class="a b c"></span>

الآن الـ JavaScript:

var span = document.querySelector("span");
var classes = span.classList;
var length = classes.length;

span.textContent = 'classList length = ' + length;
