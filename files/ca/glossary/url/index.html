Uniform Resource Locator (URL) és una cadena de text que especifica on es pot trobar un recurs en Internet.

En el context de HTTP, les URLs es denominen "adreça web" o "enllaç". El vostre navegador mostra les URLs a la barra d'adreces, per exemple https://developer.mozilla.org

Les URLs també es poden utilitzar per a la transferència d'arxius (FTP), correus electrònics (SMTP) i altres aplicacions.
