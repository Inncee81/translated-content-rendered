内存工具可以帮助你获取当前标签页内存堆的快照，它可以提供每个对象在内存中的使用以及在你的代码中哪里分配了内存的堆视图。

基础

  • 打开内存工具
  • 创建快照
  • 比较快照
  • 删除快照
  • 保存和加载快照
  • 记录调用堆栈

分析快照

树状图是在Firefox 48中新增的，关联性是在Firefox 46中新增的。

创建一张快照后内存工具会提供以下三种视图：

  • 树状图视图 以树形结构展示内存使用情况。
  • 聚合视图 以已分配类型的表格的形式展示内存使用情况。
  • 关联性视图 展示对象分配的大小，即对象的大小加上通过引用的其他对象的大小。

如果你选择记录快照的分配栈，聚合视图和关联性视图会准确列出在你代码中哪里进行了内存分配。
