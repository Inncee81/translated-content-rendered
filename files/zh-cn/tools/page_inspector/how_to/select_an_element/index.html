如果你通过点击“检查元素”查看元素”打开了查看器，然后一个元素就已经被选中了。

否则，当你在页面上移动鼠标时候，鼠标下的元素就会高亮显示，注释显示HTML标记、任何类或ID属性和元素的大小。同时，在上下文中，在检查员的左窗格中显示HTML定义：

从Firefox 39起，您可以使用方向键来改变当前选中的元素：

  • 左：高亮当前元素的父级
  • 右：高亮显示当前元素的子级，如果没有子级则就轮到同级， 如果没有同级就轮到下一个节点

当一个元素及其父级占据了屏幕同一个空间的情况下，只使用鼠标很难选中其中一个元素，这个功能就特别有用了。

点击高亮显示的元素选择它。一旦一个元素被选中，就会在查看器左侧HTML面板上标记突出显示，右侧的CSS面板中显示样式信息：

有两种方法可以选中一个新的元素：要么在HTML面板点击标记，或单击“选择元素”按钮后点击网页中的元素。
