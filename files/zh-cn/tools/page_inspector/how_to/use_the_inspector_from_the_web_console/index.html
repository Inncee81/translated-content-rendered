在页面查看器最后选择的一个元素，可以在控制台输入$0取得该元素。

在控制台输出的DOM元素右侧有一个链接图标，如果把鼠标悬停在链接图标的上方，该链接将被高亮，并且如果点击该链接，就会跳到页面选择器中并且选中该DOM元素。
