valueOf() 方法返回一个Boolean对象的原始值。

语法

bool.valueOf()

返回值

给定Boolean对象的原始值

描述

Boolean的 valueOf 方法返回一个Boolean对象或Boolean字面量的原始值作为布尔数据类型。

该方法通常在 JavaScript 内部调用，而不是在代码中显式调用。

示例

使用 valueOf

x = new Boolean();
myVar = x.valueOf()      // assigns false to myVar
