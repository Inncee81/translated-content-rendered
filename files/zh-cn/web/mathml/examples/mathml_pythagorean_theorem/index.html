现在，我们来证明毕达哥拉斯定理（勾股定理）：

命题：在一个直角三角形中，斜边的平方等于其他两边的平方之和。

例如，设 a 和 b 为两直角边，c 为斜边，那么  a 2 + b 2 = c 2 。

证明： 我们可以通过代数证明来展示大正方形面积等于内正方形（斜边的平方）加上四个三角形的面积： ( a + b ) 2 = c 2 + 4 ⋅ ( 1 2 a b ) a 2 + 2 a b + b 2 = c 2 + 2 a ba 2 + b 2 = c 2
