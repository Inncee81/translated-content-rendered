属性scale定义了用在<feDisplacementMap>滤镜上的置换缩放因子。它的总值表达在<filter>元素的属性primitiveUnits确定的坐标系统中。

如果这个属性的值是0，则它的操作对源图像没有影响。

如果没有指定这个属性，则效果等同于它被指定了0值。

用法

类别 无
值 <number>
可变性 Yes
规范文档 SVG 1.1 (2nd Edition)
