属性filter定义了由<filter>元素定义滤镜效果，该滤镜效果将应用到该元素上。

作为一个外观属性，它可以直接用作CSS样式表内部的属性。请阅读css filter以了解更多信息。

用法

类别 外观属性
值 <funciri> | none | inherit
可变性 Yes
规范文档 SVG 1.1 (2nd Edition)
<funciri>
元素的引用，它定义了将要应用到该元素的滤镜效果。
none
不对该元素应用任何滤镜效果。

元素

下列元素可以使用filter属性：

  • 图形元素 »
  • <a>
  • <defs>
  • <glyph>
  • <g>
  • <marker>
  • <missing-glyph>
  • <pattern>
  • <svg>
  • <switch>
  • <symbol>
