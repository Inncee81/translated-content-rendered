用于order属性定义一列<number>为<feConvolveMatrix> 元素构成核矩阵. 值之间用空格或逗号分开. <number>的数量必须等于在order属性中定义的<orderX>和<orderY>的乘积.

Usage context

Categories None
Value <list of number>
Animatable Yes
Normative document SVG 1.1 (2nd Edition)
