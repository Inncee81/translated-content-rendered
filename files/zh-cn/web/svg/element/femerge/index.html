feMerge滤镜允许同时应用滤镜效果而不是按顺序应用滤镜效果。利用result存储别的滤镜的输出可以实现这一点，然后在一个 <feMergeNode>子元素中访问它。

用法

类别滤镜元素
允许的内容物任意数量、任意排序的下列元素：
<feMergeNode>
