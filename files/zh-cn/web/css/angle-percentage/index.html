CSS数据类型 <angle-percentage> 代表了一种既可以是 angle ，也可以是percentage的数据类型

 

语法

有关此类型的详细文档，请参阅 angle 和 percentage 

在calc()中使用这个类型

当 <angle-percentage> 被正确定义后，它就代表了一个可以被 calc() 表达式解析的角度
