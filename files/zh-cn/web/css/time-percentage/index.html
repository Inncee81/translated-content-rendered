CSS数据类型<time-percentage> 的值可以表示为时间或百分比。

Syntax

有关此类型允许的各个语法的详细信息，请参阅time和percentage的文档。

Use in calc()

如果将<time percentage>指定为允许的类型，这意味着该百分比解析为时间，因此可以在calc()表达式中使用。
