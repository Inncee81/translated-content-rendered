CSS Fragmentation是CSS的模块，它定义了内容在跨多个页面，区域或列中被分割（分段）时如何显示。

当一个内联框包装成多行时会发生碎片。当一个块跨越一个列布局容器内的多个列，或者在打印时跨越一个分页符时，也会发生这种情况。元素的每个渲染片段称为一个片段。
