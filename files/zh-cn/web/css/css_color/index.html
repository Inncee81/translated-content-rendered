CSS Color 是一个处理颜色的 CSS 模块，包括颜色类型、颜色混合、不透明度、以及你如何将这些颜色和效果应用于 HTML 内容。

不是所有把 <color>  作为值的 CSS 属性都是这个模块的一部分，但是它们确实依赖这个模块。
