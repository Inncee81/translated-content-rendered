<flex> CSS 数据类型 表示了网格（grid）容器中的一段可变长度。于grid-template-columns、grid-template-rows及相关属性中使用。

语法

<flex>数据类型为一<number>后加单位fr。与所有CSS尺寸相同，单位与数字间无空格。

Examples

1fr    /* 使用整型 */
2.5fr  /* 使用浮点 */
