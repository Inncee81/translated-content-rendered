摘要

:optional CSS 伪类 表示任意没有required属性的 <input>，<select> 或  <textarea> 元素使用它。

/* Selects any optional <input> */
input:optional {
  border: 1px dashed black;
}

它允许表单容易的展示可选字段并且渲染其外观。

注：为必填字段设置外观，请使用 :required伪类。
