column-count CSS属性，描述元素的列数。

column-count: 3;
column-count: auto;

column-count: inherit;
column-count: initial;
column-count: unset;

语法

初始值auto
适用元素Block containers except table wrapper boxes
是否是继承属性否
适用媒体visual
计算值as specified
Animation typean integer
正规顺序per grammar

取值

auto
用来表示列的数量由其他CSS属性指定, 例如 column-width.
<number>
是个严格的正数 <integer> ，用来描述元素内容被划分的理想列数. 假如 column-width也被设置为非零值, 此参数仅表示所允许的最大列数. 

例子

.content-box {
  border: 10px solid #000000;
  column-count:3;
}
