RenderingContext 是一个辅助类型，描述下面任何一个渲染上下文：  CanvasRenderingContext2D， WebGLRenderingContext 或者 WebGL2RenderingContext （继承自 WebGLRenderingContext）。

这是简化规范的辅助类型，它不是一个接口，也没有对象实现它。
