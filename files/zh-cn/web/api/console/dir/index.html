非标准
该特性是非标准的，请尽量不要在生产环境中使用它！

在控制台中显示指定JavaScript对象的属性，并通过类似文件树样式的交互列表显示。

Note: 此特性在 Web Worker 中可用。

语法

console.dir(object);

参数

object
打印出该对象的所有属性和属性值.
