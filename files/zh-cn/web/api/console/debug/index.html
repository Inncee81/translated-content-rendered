Console.debug() 输出“调试”级别的消息且仅仅控制台配置为显示调试输出时才显示该消息。

Note: 此特性在 Web Worker 中可用。

语法

console.debug(对象1 [, 对象2, ..., 对象N]);
console.debug(消息[, 字符串1, ..., 字符串N]);

参数

对象1 ... 对象N
要输出的JavaScript 对象列表。 按传参的顺序把对象输出到控制台。
消息
一个JavaScript字符串，其中包含零个或多个替代字符串。
字符串1 ... 字符串N
JavaScript 对象，用来依次替换msg中的替代字符串。你可以在替代字符串中指定对象的输出格式。查看使用字符串替换 in Console了解替换字符串如何工作。

有关详细信息，请参阅console 对象文档中的将文本输出到控制台。
