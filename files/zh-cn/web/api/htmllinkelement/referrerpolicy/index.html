HTMLLinkElement.referrerPolicy 反映了 <link> 元素的HTML属性 referrerpolicy 的定义，这个属性定义了<link> 元素在获取资源时的引用方式。

详情请参考HTTP header中的  Referrer-Policy 。

 

语法

DOMString HTMLLinkElement.referrerPolicy

例子

var links = document.getElementsByTagName("link");
links[0].referrerPolicy; // "no-referrer"
