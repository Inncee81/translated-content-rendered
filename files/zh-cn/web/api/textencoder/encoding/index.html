这是一个实验中的功能
此功能某些浏览器尚在开发中，请参考浏览器兼容性表格以得到在不同浏览器中适合使用的前缀。由于该功能对应的标准文档可能被重新修订，所以在未来版本的浏览器中该功能的语法和行为可能随之改变。

TextEncoder.encoding 只读属性返回一个DOMString ，其中包含特定编码器使用的编码算法的名称。

现在只会返回“utf-8”。

语法

b = encoder.encoding;
