The CSSValueList 接口继承自 CSSValue 接口，提供一个经过排序的 CSS 值的抽象集合。

一些属性允许在它们的语法中使用空的列表。这时，these properties take the none identifier. So, an empty list means that the property has the value none.

The items in the CSSValueList are accessible via an integral index, starting from 0.

属性

从它的父接口 CSSValue 继承属性。

CSSValueList.length 只读
一个 unsigned long 数值，表示此列表中 CSSValues 的个数。

方法

CSSValueList.item()
This method is used to retrieve a CSSValue by ordinal index. The order in this collection represents the order of the values in the CSS style property. If index is greater than or equal to the number of values in the list, this returns null.
