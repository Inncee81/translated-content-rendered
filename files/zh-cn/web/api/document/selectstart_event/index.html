Selection API 的 selectstart 事件在用户开始一个新的选择时候触发。

如果事件被取消，选择将不被触发。

Bubbles Yes
Cancelable Yes
Target objects Document
Interface Event

例子

document.addEventListener("selectstart", function() {
  console.log('Selection started');
}, false);

继承

selectstart 事件实现Event 接口. 你可以使用此界面上定义的属性和方法。
