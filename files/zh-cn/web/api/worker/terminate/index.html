Worker 接口中的 terminate()  方法用于立即终止 Worker 的行为. 本方法并不会等待 worker 去完成它剩余的操作；worker 将会被立刻停止

Syntax

myWorker.terminate();

参数

没有。

返回值

Void.

Example

以下代码示例表明，通过使用 Worker() 构造器创建出的Worker 对象，在下一步操作之后会被立即终止。

var myWorker = new Worker('worker.js');

myWorker.terminate();
