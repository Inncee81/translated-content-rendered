Range.startOffset 是一个只读属性，用于返回一个表示 Range 在 startContainer 中的起始位置的数字。

如果 startContainer 是一个文本（Text）、注释（Comment）或者CDATA区块（CDATASection）节点，那么返回的偏移量是从 startContainer 开始到 Range 的边界点的字符数量。对于其他的节点类型， startOffset 返回 startContainer 到边界点的子节点数量。

可使用Range.setStart 方法改变 Range 的 startOffset 位置。

语法

startRangeOffset = range.startOffset;

示例

range = document.createRange();
range.setStart(startNode,startOffset);
range.setEnd(endNode,endOffset);
startRangeOffset = range.startOffset;
