CanvasPattern 接口表示描述一个模板（基于image, canvas或video）的不透明对象，通过 CanvasRenderingContext2D.createPattern() 方法创建.

属性

非透明对象，没有暴露出属性

方法

没有继承来的方法

CanvasPattern.setTransform()
应用 SVGMatrix 对模板做线性变换
