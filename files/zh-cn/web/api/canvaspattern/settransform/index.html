CanvasPattern.setTransform() 方法使用 SVGMatrix 对象作为图案的变换矩阵，并在此图案上调用它。

语法

void pattern.setTransform(matrix);

参数

matrix
SVGMatrix ，被用作图案的变换矩阵。

示例

使用 setTransform 方法

这是一段简单的代码片段，使用 setTransform 方法创建一个来自 SVGMatrix 具有指定图案变化的CanvasPattern 。如例子所示，如果你把图案赋值给当前的 fillStyle ，当你使用 fillRect() 方法时，图案会被应用到 canvas 上绘制出效果。

HTML

<canvas id="canvas"></canvas>
<svg id="svg1"></svg>

JavaScript

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");

var svg1 = document.getElementById("svg1");
var matrix = svg1.createSVGMatrix();

var img = new Image();
img.src = 'https://mdn.mozillademos.org/files/222/Canvas_createpattern.png';

img.onload = function() {
  var pattern = ctx.createPattern(img, 'repeat');
  pattern.setTransform(matrix.rotate(-45).scale(1.5));
  ctx.fillStyle = pattern;
  ctx.fillRect(0,0,400,400);
};
