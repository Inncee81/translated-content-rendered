概述

返回浏览器窗口可占用的水平宽度（单位：像素）。

语法

window.screen.availWidth

示例

var screenAvailWidth = window.screen.availWidth;

console.log(screenAvailWidth);

规范

DOM Level 0 不属于任何标准.
