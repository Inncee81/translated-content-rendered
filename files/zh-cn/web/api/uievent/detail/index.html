UIEvent.detail是只读属性, 当值为非空的时候, 提供当前点击数(和环境有关) 。

对 click 或者 dblclick 事件, UIEvent.detail 是当前点击数量。

对 mousedown 或者 mouseup 事件, UIEvent.detail是1加上当前点击数。

对所有的其它UIEvent 对象, UIEvent.detail 总是零.
