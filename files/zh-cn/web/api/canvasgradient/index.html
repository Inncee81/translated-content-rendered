CanvasGradient 接口表示描述渐变的不透明对象。通过 CanvasRenderingContext2D.createLinearGradient() 或 CanvasRenderingContext2D.createRadialGradient() 的返回值得到.

属性

不透明对象，没有暴露的属性.

方法

没有继承的方法

CanvasGradient.addColorStop()
添加一个由偏移(offset)和颜色(color)定义的断点到渐变中。如果偏移值不在0到1之间，将抛出INDEX_SIZE_ERR错误，如果颜色值不能被解析为有效的CSS颜色值 <color>，将抛出SYNTAX_ERR错误。
