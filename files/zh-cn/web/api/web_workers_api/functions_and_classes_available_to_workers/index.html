除了标准的 JavaScript 函数集 (例如 String, Array, Object, JSON 等), DOM有多种功能可供 workers使用。

本文提供了一些列表。

Workers 在另一个全局上下文内运行, DedicatedWorkerGlobalScope 与当前 window不同。

默认情况下，Window的方法和属性不可用，但DedicatedWorkerGlobalScope像Window，实现WindowTimers 和 WindowBase64。
