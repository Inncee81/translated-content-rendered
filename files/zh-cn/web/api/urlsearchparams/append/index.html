append() 是 URLSearchParams 接口的一个方法。可以插入一个新搜索参数。

语法

URLSearchParams.append(name, value)

参数

name
需要插入搜索参数的键名。
value 
需要插入搜索参数的值。

返回

无

例子

let url = new URL('https://example.com?foo=1&bar=2');
let params = new URLSearchParams(url.search.slice(1));

//添加第二个foo搜索参数。
params.append('foo', 4);
//查询字符串变成: 'foo=1&bar=2&foo=4'
