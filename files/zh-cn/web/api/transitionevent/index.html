TransitonEvent 接口指那些提供了与 transition 有关信息的事件。

属性

同时也继承了父类 Event 的属性.

TransitionEvent.propertyName 只读
是一个DOMString，代表与这个 transition 有关的 CSS 属性名.
TransitionEvent.elapsedTime 只读
是一个 float 值，代表从这个事件触发开始，这个 transition 运行的时间（以秒为单位）。这个值不受 transition-delay 属性的影响.
TransitionEvent.pseudoElement 只读
是一个以 '::' 开头的 DOMString, 代表这个过渡运行于其上的伪元素的名字。如果这个过渡不是在一个伪元素上而是在一个元素上运行的，这个值就是一个空值''。

构造函数

TransitionEvent()
用给定的参数创建一个 TransitionEvent 事件。

方法

同时也继承了父类 Event的方法.

TransitionEvent.initTransitionEvent()
使用已经废弃的Document.createEvent("TransitionEvent") 方法初始化已经创建的 TransitonEvent 事件。
