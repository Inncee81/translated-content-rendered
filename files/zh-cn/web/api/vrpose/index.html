The VRPose interface of the WebVR API represents the state of a VR sensor at a given timestamp (which includes orientation, position, velocity, and acceleration information.)

WebVR API  中的 VRPose 接口表示在一个给定的时间戳中，一个VR传感器的状态（包括了方向、位置、速度和加速度信息）。

This interface is accessible through the VRDisplay.getPose() and VRDisplay.getImmediatePose() methods.

这个接口能通过 VRDisplay.getPose() 和 VRDisplay.getImmediatePose() 方法访问

Properties

VRPose.timeStamp 只读
Returns the current time stamp of the system — a monotonically increasing value useful for determining if position data has been updated, and what order updates have occured in.
返回当前系统的时间戳。此单调递增的值可以有助于确定位置数据是否有更新，以及更新的顺序。
VRPose.position 只读
Returns the position of the VRDisplay at the current VRPose.timestamp as a 3D vector
以三维向量的形式返回当前时间戳 VRPose.timestamp  时 VRDisplay  的位置信息。
VRPose.linearVelocity 只读
Returns the linear velocity of the VRDisplay at the current VRPose.timestamp, in meters per second.
返回当前时间戳 VRPose.timestamp  时 VRDisplay  的线速度，单位为米/秒。
VRPose.linearAcceleration 只读
Returns the linear acceleration of the VRDisplay at the current VRPose.timestamp, in meters per second per second.
返回当前时间戳 VRPose.timestamp  时 VRDisplay  的线加速度，单位为米/平方秒。
VRPose.orientation 只读
Returns the orientation of the sensor at the current VRPose.timestamp, as a quarternion value.
以四元数的形式返回当前时间戳 VRPose.timestamp  时传感器的方向，
VRPose.angularVelocity 只读
Returns the angular velocity of the VRDisplay at the current VRPose.timestamp, in radians per second.
返回当前时间戳 VRPose.timestamp  时 VRDisplay  的角速度，单位为弧度/秒。
VRPose.angularAcceleration 只读
Returns the angular acceleration of the VRDisplay at the current VRPose.timestamp, in meters per second per second.
返回当前时间戳 VRPose.timestamp  时 VRDisplay  的角加速度，单位为弧度/平方秒。
