属性

属性继承自父类，HTMLElement。

HTMLDivElement.align
这是一个 DOMString 表示一个指示元素内容相对于周围内容的对齐方式的可枚举属性。可取的值有 "left"，"right"，"justify"，和 "center"。

方法

没有特有的方法；方法继承自父类，HTMLElement。
