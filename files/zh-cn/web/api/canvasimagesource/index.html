CanvasImageSource 是一个辅助类型，描述下面类型的任何一个对象：HTMLImageElement, HTMLVideoElement, HTMLCanvasElement, CanvasRenderingContext2D, 或 ImageBitmap.

这是简化规范的辅助类型， 它不是一个接口，也没有对象实现它。
