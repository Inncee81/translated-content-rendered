当调用requestFullscreen() 元素以将该元素置于全屏模式时，使用 FullscreenOptions 字典提供配置选项。

属性

navigationUI可选
在元素处于全屏模式时，控制是否保持浏览器用户界面元素可见的字符串。默认的“auto”让浏览器做出这个决定。
