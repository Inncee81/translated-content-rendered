Position  接口表示在给定的时间的相关设备的位置。这个位置，用一个Coordinates对象表示，包括设备在地球上的二维位置， 但也可以包括设备的海拔和速度。      

属性

Position接口没有继承任何属性。

Position.coords 只读
返回一个定义了当前位置的Coordinates 对象.
Position.timestamp 只读
返回一个时间戳DOMTimeStamp， 这个时间戳表示获取到的位置的时间。

方法

Position 接口没有实现也没有继承任何方法。
