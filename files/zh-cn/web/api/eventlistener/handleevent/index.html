当事件被发送到 EventListener 时，user agent 将调用 EventListener 方法 handleEvent()，以便处理在观察到的 EventTarget 上发生的事件。

语法

eventListener.handleEvent(event);

参数

event
一个 Event 对象，描述已被触发并需要处理的事件。

返回值

undefined。如果返回了值，浏览器将忽略它。
