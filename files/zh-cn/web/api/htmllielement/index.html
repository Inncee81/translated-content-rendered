属性

从其父级 HTMLElement 继承属性。

HTMLLIElement.type
是DOMString表示项目符号的类型 "disc"，"square" 或 "circle"。由于定义列表类型的标准方法是通过 CSS list-style-type属性，应当使用 CSSOM 方法通过脚本设置它。
HTMLLIElement.value
是long指示给定<ol>中列表元素的序号位置。它反映了HTML <li>元素的value属性，并且可以小于0。如果<li>元素不是<ol>元素的子元素，则该属性没有意义。

方法

没有具体方法; 从其父级 HTMLElement 继承属性。
