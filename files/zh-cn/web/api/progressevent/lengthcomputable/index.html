ProgressEvent.lengthComputable 只读属性是一个布尔Boolean 标志，表示ProgressEvent 所关联的资源是否具有可以计算的长度。否则 ，ProgressEvent.total 属性将是一个无意义的值。

语法

flag = ProgressEvent.lengthComputable
