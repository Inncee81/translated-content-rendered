当浏览器失去网络连接时，offline事件被触发。并且navigator.onLine的值变为 false。

规范

接口

否

无

属性

属性 类型 描述
target 只读 EventTarget 产生该事件的对象(DOM树中最顶级的那个对象).
type 只读 DOMString 事件类型.
bubbles 只读 Boolean 该事件是否冒泡.
cancelable 只读 Boolean 该事件是否可取消默认行为.
