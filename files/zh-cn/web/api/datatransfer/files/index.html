DataTransfer.files属性在拖动操作中表示文件列表。如果操作不包含文件，则此列表为空。

此功能可用于将文件从用户桌面拖动到浏览器。

语法

dataTransfer.files;

返回值

拖动操作中的文件列表，操作中每个文件的一个列表项。 如果拖动操作没有文件，此列表为空。
