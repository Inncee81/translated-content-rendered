downlink 是 NetworkInformation 接口的一个只读属性，返回以Mb/s为单位的有效带宽，并保留该值为25kb/s的最接近的整数倍。该值基于最近监测的保持活跃连接的应用层吞吐量，排除了到私有地址空间的连接。当缺少最近的带宽测量数据时，该属性由底层连接技术属性决定。

Syntax

var downLink = NetworkInformation.downlink

Value

double 双精度浮点数。
