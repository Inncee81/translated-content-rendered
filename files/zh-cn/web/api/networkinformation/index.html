这是一个实验中的功能
此功能某些浏览器尚在开发中，请参考浏览器兼容性表格以得到在不同浏览器中适合使用的前缀。由于该功能对应的标准文档可能被重新修订，所以在未来版本的浏览器中该功能的语法和行为可能随之改变。

NetworkInformation 提供有关设备正在使用的连接与网络进行通信的信息，并提供了在连接类型更改时通知脚本的事件。NetworkInformation 接口不能被是实例化， 而是通过 Navigator 的 connection 属性进行访问。

Note: 此特性在 Web Worker 中可用。

Properties

这些属性接口继承自 EventTarget.

NetworkInformation.type 只读
返回设备正在与网络进行通信的连接类型。 它将是以下值之一：
  • bluetooth
  • cellular
  • ethernet
  • none
  • wifi
  • wimax
  • other
  • unknown
NetworkInformation.downlinkMax 只读
返回基础连接技术的最大下载速度（Mbps）。

Event handlers

NetworkInformation.onchange
当连接信息更改并在此对象上触发更改时触发的 change  。

Methods

这些属性接口同样继承自 EventTarget.
