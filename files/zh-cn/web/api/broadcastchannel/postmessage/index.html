可以使用 BroadcastChannel.postMessage() 发送一条任意 Object 类型的消息，给所有同源下监听了该频道的所有浏览器上下文。消息以 message 事件的形式发送给每一个绑定到该频道的广播频道。

Note: 此特性在 Web Worker 中可用。

语法

var str = channel.postMessage(object);
