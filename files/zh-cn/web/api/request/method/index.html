Request的只读属性method包含请求的方法(GET, POST, etc.)

语法

var myMethod = request.method;

Value

A ByteString indicating the method of the request.

举例

In the following snippet, we create a new request using the Request.Request() constructor (for an image file in the same directory as the script), then save the method of the request in a variable:

var myRequest = new Request('flowers.jpg');
var myMethod = myRequest.method; // GET
