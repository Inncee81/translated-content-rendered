这是一个实验中的功能
此功能某些浏览器尚在开发中，请参考浏览器兼容性表格以得到在不同浏览器中适合使用的前缀。由于该功能对应的标准文档可能被重新修订，所以在未来版本的浏览器中该功能的语法和行为可能随之改变。

DocumentFragment() 构造函数返回一个新创建的 DocumentFragment 对象。

语法

fragment = new DocumentFragment()

示例

fragment = new DocumentFragment();
