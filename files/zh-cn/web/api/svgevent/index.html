SVGEvent 是一种与SVG相关事件所对应的事件对象接口。

Properties

Property Type Description
target 只读 EventTarget 事件目标（DOM树的最顶端）
type 只读 DOMString 事件类型
bubbles 只读 Boolean 事件是否冒泡
cancelable 只读 Boolean 事件是否可取消
