GamepadEvent interface 的 GamepadEvent.gamepad 属性返回一个 Gamepad 对象，为触发 gamepadconnected 和gamepaddisconnected 事件提供相关联控制器数据的访问。

语法

只读 属性 Gamepad gamepad;

示例

在触发的 Window.gamepadconnected 事件上调用 gamepad 属性。

window.addEventListener("gamepadconnected", function(e) {
  console.log("控制器已连接于 %d 位：%s. %d 个按钮, %d 个坐标方向。",
  e.gamepad.index, e.gamepad.id,
  e.gamepad.buttons.length, e.gamepad.axes.length);
});

值

一个 Gamepad 对象。
