GamepadEvent_ 构造函数用于创建一个新的 GamepadEvent 对象。

语法

var gamepadEvent = new GamepadEvent(typeArg, options)

参数

typeArg
一个 DOMString ，必须为 gamepadconnected 或 gamepaddisconnected。
options 可选
选项如下所示：
  • gamepad: 一个 Gamepad 实例，描述了与事件相关的控制器（对象）。
