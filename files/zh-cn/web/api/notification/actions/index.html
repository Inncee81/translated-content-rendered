Notification接口的只读属性actions返回使用Notification()构造函数创建通知时使用actions选项设置的NotificationAction对象列表。这是用户可以在通知上下文中选择立即执行的应用定义的操作列表。

Note: 此特性在 Web Worker 中可用。

语法

var actions[] = Notification.actions;

值

NotificationAction对象的只读数组。用户在通知中选择每项的单一的功能。
