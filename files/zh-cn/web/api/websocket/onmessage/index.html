WebSocket.onmessage 属性是一个当收到来自服务器的消息时被调用的 EventHandler。它由一个MessageEvent调用。

语法

aWebSocket.onmessage = function(event) {
  console.debug("WebSocket message received:", event);
};

值

一个 EventListener.
