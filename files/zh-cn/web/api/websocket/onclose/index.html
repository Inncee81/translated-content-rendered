WebSocket.onclose 属性返回一个事件监听器，这个事件监听器将在 WebSocket 连接的readyState 变为 CLOSED时被调用，它接收一个名字为“close”的 CloseEvent 事件。

语法

WebSocket.onclose = function(event) {
  console.log("WebSocket is closed now.");
};

值

一个EventListener.
