The WebSocket.url read-only property returns the absolute URL of the WebSocket as resolved by the constructor.

WebSocket.url是一个只读属性，返回值为当构造函数创建WebSocket实例对象时URL的绝对路径。

语法

var url = aWebSocket.url;

 

返回值

A DOMString.
