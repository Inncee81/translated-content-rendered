CSS Object Model 是一组允许用JavaScript操纵CSS的API。 它是继DOM和HTML API之后，又一个操纵CSS的接口，从而能够动态地读取和修改CSS样式。

浏览器兼容性

近几年所有的这些新特性正在一点一点的向不同的浏览器内添加，不过这确实是一个很复杂无法用简单表格总结的过程。如果具体项目中需要用到，请详细的检查接口的可用性。
