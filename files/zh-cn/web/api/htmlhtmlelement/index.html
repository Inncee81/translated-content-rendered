您可以通过读取document.documentElement属性的值来获取文档的 HTMLHtmlElement对象。

属性

继承 HTMLElement的属性。

HTMLHtmlElement.version 
这是一个用于表示管理此文档的HTML文档类型定义（DTD）版本的 DOMString。 但此属性不应再使用，因为它不符合要求。请仅忽略它。

方法

没有明确的方法； 可从父项继承—— HTMLElement。
