 HTMLFormElement.submit()用来提交表单。

这个方法和触发提交表单按钮很类似，但有所不同：

  • No submit event is raised. In particular, the form's onsubmit event handler is not run.
  • Constraint validation is not triggered.

如果一个表单控件（比如一个提交按钮）的 name 或 id 的值为 submit，则它将覆盖表单的 submit 方法.

语法

HTMLFormElement.submit()

例子

document.forms["myform"].submit();

规范

HTML Living standard: The form element
