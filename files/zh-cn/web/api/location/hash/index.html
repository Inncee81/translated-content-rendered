Location 接口的 hash 属性返回一个 USVString，其中会包含URL标识中的 '#' 和 后面URL片段标识符。

这里 fragment 不会经过百分比编码（URL编码）。如果 URL 中没有 fragment，该属性会包含一个空字符串，""

Syntax

string = object.hash;
object.hash = string;

Examples

  console.log(anchor.hash); // 返回'#Examples'
