插件接口提供有关浏览器插件的信息 .

注：在最新版本的浏览器中插件对象本身的属性不可枚举。

属性

名称 描述 返回值类型 兼容性
Plugin.description 一个人类可读的插件描述。  只读. DOMString DOM 0
Plugin.filename 插件文件名. 只读. DOMString DOM 0
Plugin.name 插件名. 只读. DOMString DOM 0
Plugin.version 插件版本号字符串. 只读. DOMString Gecko browsers only (Firefox 4+)

方法

Plugin.item
返回支持的MIME类型的索引号.
Plugin.namedItem
返回支持的MIME类型.
