概要

PerformanceTiming.responseEnd 是一个返回代表一个时刻的 unsigned long long 型只读属性，为浏览器从服务器、缓存或者本地资源接收响应的最后一个字节或者连接被关闭之时的 Unix毫秒时间戳。

语法

time = performanceTiming.responseEnd;
