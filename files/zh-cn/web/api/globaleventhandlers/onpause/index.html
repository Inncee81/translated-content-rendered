概述

onpause可以用来获取或设置当前元素的onpause事件的事件处理函数。当媒体播放被暂停时，将触发pause事件。

语法

element.onpause = handlerFunction;
var handlerFunction = element.onpause;

handlerFunction应该是null或指定事件的JavaScript函数。
