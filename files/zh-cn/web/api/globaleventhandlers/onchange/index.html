概述

onchange可以用来获取或设置当前元素的change事件的事件处理函数.

语法

element.onchange = 事件处理函数

备注

下面的伪代码演示了,在Mozilla中,onchange事件是如何触发的:

  control.onfocus = focus;
  control.onblur = blur;
  function focus () {
      original_value = control.value;
  }

  function blur () {
      if (control.value != original_value)
        control.onchange();
  }

因此,如果你在你的focus或blur事件的处理函数里修改一个控件的值的话,你可能遇到一些关于change事件意想不到的现象.另外, change事件是在blur事件发生之后才触发的. 这一点和IE不同.
