ByteString 是一个可以对应所有可能的字节序列的 UTF-8 字符串。在 JavaScript 中，当返回时 ByteString 时，它会映射到一个 String 上去； 通常，只有在和一些协议交互，进行字节和字符串的相互转换时，它才会被用到，例如 HTTP 协议。
