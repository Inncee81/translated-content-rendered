delete() 方法可以从Headers对象中删除指定header.

下列原因将会导致该方法抛出一个TypeError:

  • header名在HTTP header中是不存在的.
  • header被锁定了.​

Note:出于安全原因, 部分头信息只能被用户代理控制. 这些头信息包括 forbidden header names  和 forbidden response header names.

Syntax

myHeaders.delete(name);

Parameters

name
需删除的HTTP header名称.

Returns

Void.

Example

创建一个空的Headers对象:

var myHeaders = new Headers(); // Currently empty

可以通过append()方法添加header:

myHeaders.append('Content-Type', 'image/jpeg');
myHeaders.get('Content-Type'); // Returns 'image/jpeg'

可以通过delete()方法删除已有header:

myHeaders.delete('Content-Type');
myHeaders.get('Content-Type'); // Returns null, as it has been deleted
