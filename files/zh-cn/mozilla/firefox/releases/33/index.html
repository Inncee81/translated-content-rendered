普通 Web 开发者应该注意的变化

Developer Tools

No change.

CSS

  • 实现了 @counter-style 规则 (bug 966166)。
  • -moz-ethiopic-numeric 去除前缀(bug 985825)。
  • 支持了disclosure-open/disclosure-closed 计数器样式(bug 982355)。

HTML

No change.

JavaScript

  • 移除了非标准的 Number.toInteger() 方法 (bug 1022396).

Interfaces/APIs/DOM

No change.

No change.

No change.

No change.

No change.
