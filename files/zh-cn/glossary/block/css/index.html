网页上的区块指的是出现在新行上的 HTML 元素，即在该元素的前一个元素的下方，在后一个元素之上（通常称为块级元素）。 例如，HTML 中的 <p> 元素默认是块级元素，而 <a> 元素是一个内联元素 - 你可以在 HTML 源码中将多个标记紧挨着写在一起，渲染结果是，它们将在同一行显示。

通过使用 CSS 的 display 属性，你可以更改元素是内联还是块（在许多其他选项中）; 块也受到定位方案（positioning schemes）和使用 CSS 中 position 属性的影响。
