WebGL (Web图形库) 是一个用于绘制交互式2D/3D图形的 JavaScript API ，基于 OpenGL ES 2.0 。

Khronos Group 负责WebGL的维护。

你可以在 HTML <canvas> 元素中调用WebGL。

目前所有主流 浏览器 都支持WebGL，即便如此，WebGL是否可用还是取决于硬件设备等其他因素（比如GPU是否支持WebGL）。
