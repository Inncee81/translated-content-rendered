हाइपरटेक्स्ट ट्रांसफर प्रोटोकॉल (HTTP) अंतर्निहित नेटवर्क है protocol इस पर हाइपरमीडिया दस्तावेज़ों को स्थानांतरित करने में सक्षम बनाता है Web, आमतौर पर एक ब्राउज़र और सर्वर के बीच ताकि इंसान उन्हें पढ़ सकें। HTTP विनिर्देशन के वर्तमान संस्करण को HTTP / 2 कहा जाता है।

के हिस्से के रूप में URI, "http: //" को "स्कीमा" कहा जाता है और आमतौर पर एक पते की शुरुआत में खड़ा होता है, उदाहरण के लिए "https://developer.mozilla.org" में ब्राउज़र को इंगित करने के लिए HTTP प्रोटोकॉल का उपयोग करके दस्तावेज़ का अनुरोध करना। इस मामले में https HTTP प्रोटोकॉल के सुरक्षित संस्करण को संदर्भित करता है, SSL (जिसे टीएलएस भी कहा जाता है)।

HTTP शाब्दिक है (सभी संचार सादे पाठ में किया जाता है) और स्टेटलेस (कोई भी संचार पिछले संचार से अवगत नहीं है)। यह संपत्ति मनुष्यों के लिए वर्ल्ड वाइड वेब पर दस्तावेजों (वेब ​​साइटों) को पढ़ने के लिए आदर्श बनाती है। हालाँकि, HTTP को आधार के रूप में भी इस्तेमाल किया जा सकता है REST सर्वर से सर्वर या के लिए वेब सेवाओं AJAX उन्हें और अधिक गतिशील बनाने के लिए वेब साइटों के भीतर अनुरोध।
