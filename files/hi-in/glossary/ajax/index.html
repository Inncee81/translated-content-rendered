अतुल्यकालिक JavaScript तथा XML (AJAX) अधिक जटिल, गतिशील वेबपृष्ठों के निर्माण की एक प्रोग्रामिंग प्रथा है, जिसे एक तकनीक के रूप में जाना जाता है XMLHttpRequest.

AJAX आपको क्या करने की अनुमति देता है, इसके कुछ हिस्सों को अपडेट करें DOM के HTML पूरे पृष्ठ को पुनः लोड करने के बजाय वेबपृष्ठ। AJAX आपको असिंक्रोनस रूप से काम करने देता है, जिसका अर्थ है कि आपका कोड चलना जारी है, जबकि आपके वेबपेज का वह भाग पुनः लोड करने की कोशिश कर रहा है (तुल्यकालिक रूप से जो आपके कोड को तब तक चलने से रोक देगा, जब तक कि आपके वेबपेज का हिस्सा पुनः लोड नहीं हो जाता)।

इंटरैक्टिव वेबसाइटों और आधुनिक वेब मानकों के साथ, AJAX को धीरे-धीरे जावास्क्रिप्ट चौखटे और अधिकारी के भीतर कार्यों द्वारा प्रतिस्थापित किया जा रहा है Fetch API मानक.
