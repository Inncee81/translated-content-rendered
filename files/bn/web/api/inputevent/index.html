InputEvent ইন্টারফেসটি সম্পাদনযোগ্য কন্টেন্টে কোন পরিবর্তন হলে, সে পরিবর্তন সম্পর্কে জানায়।

প্রোপার্টি

InputEvent ইন্টারফেসটির কোন প্রোপার্টি নেই।

মেথড

InputEvent ইন্টারফেসটির কোন মেথড নেই।

ইভেন্ট

InputEvent.data Read only
লেখা বর্ণগুলি নিয়ে একটু DOMString রিটার্ন করে। এই স্ট্রিংটি খালি হতে পারে, যদি পরিবর্তনের ফলে নতুন বর্ণ লেখা না হয়। (যেমন বর্ণ মুছার সময়)
InputEvent.isComposing Read only
compositionstart এর আগে এবং compositionend এর পরে একটু বুলিয়ান ভ্যালু নিয়ে ইভেন্ট ফায়ার করে।
