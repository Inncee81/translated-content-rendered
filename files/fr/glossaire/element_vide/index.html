Un élément vide (empty element en anglais) est un élément HTML, SVG, ou MathML qui ne peut pas avoir de nœud enfant (que ce soit un autre élément ou du texte).

Les spécifications HTML, SVG et MathML (en anglais) définissent de manière précise quels éléments peuvent contenir quels autres. De nombreuses combinaisons n'ont aucun sens sémantique ; c'est le cas par exemple d'un élément <audio> imbriqué dans un élément <hr>.

Les éléments vides en HTML sont les suivants :

  • <area>
  • <base>
  • <br>
  • <col>
  • <embed>
  • <hr>
  • <img>
  • <input>
  • <keygen> (HTML 5.2 brouillon supprimé)
  • <link>
  • <meta>
  • <param>
  • <source>
  • <track>
  • <wbr>
