La propriété en lecture seule name de l'interface DedicatedWorkerGlobalScope renvoie le nom qui a été (facultativement) donné au  Worker lors de sa création. C'est le nom que le constructeur Worker() peut transmettre pour obtenir une référence à DedicatedWorkerGlobalScope.

Syntaxe

var nameObj = self.name;

Valeur

Une DOMString (chaîne de caractères).

Exemple

Un worker est créé en utilisant un constructeur avec une option de name :

var myWorker = new Worker("worker.js", { name : "myWorker" });

le DedicatedWorkerGlobalScope aura désormais le nom de "myWorker", qui peut être renvoyé pendant l'exécution

self.name

à l'intérieur du worker.
