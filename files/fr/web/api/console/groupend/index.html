Quitte le groupe en ligne actuel dans la console Web. Voir  Using groups in the console  dans la documentation console.group() pour plus de détails et des exemples.

Note : Cette fonctionnalité est disponible via les Web Workers.

Syntaxe

console.groupEnd();

Paramètres

Aucun.
