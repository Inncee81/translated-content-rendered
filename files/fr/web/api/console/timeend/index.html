Arrête un chronomètre (timer) précédemment démarré par console.time().

Voir Timers dans la documentation de console pour des détails et des exemples.

Syntaxe

console.timeEnd(label);

Paramètres

label
Le nom du timer à stopper. Une fois arrêté, le temps écoulé est automatiquement affiché dans la Console Web.

Spécification

Spécification Statut Commentaire
Console API
