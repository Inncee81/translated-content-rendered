Interface d'élément SVG

Toutes les interfaces SVG DOM qui correspondent directement à des éléments du langage SVG dérivent de l'interface SVGElement.

Vue d'ensemble de l'interface

Implémente aussi Voir l'interface générique DOM Element.
Méthodes Aucune
Propriétés
  • readonly DOMString id
  • readonly DOMString xmlbase
  • readonly SVGSVGElement ownerSVGElement
  • readonly SVGElement viewportElement
Normative document SVG 1.1 (2nd Edition)

Properties

Nom Type Description
id DOMString La valeur de l'attribut id de l'élément, ou une chaîne vide si id n'est pas présent.
xmlbase DOMString Correspond à l'attribut xml:base de l'élément.
ownerSVGElement SVGSVGElement

L'élément <svg> parent le plus près. Null si l'élément est l'élément svg le plus à l'extérieur.
viewportElement SVGElement

L'élément qui a établi la fenêtre actuelle. Souvent l'élément parent <svg> le plus prêt. Null si l'élément est l'élément svg le plus à l'extérieur.

Méthodes

L'interface SVGElement ne possède aucune méthode spécifique.
