Interface SVG stylisable

L'interface SVGStylable est implémentée sur tout les objets SVG pouvant avoir des attributs style, class ou d'autre attributs modifiants leur présentation, leur affichage.

Aperçu de l'interface

Implémente aussi /
Métodes
  • CSSValue getPresentationAttribute(in DOMString name)
Propriétés
  • readonly SVGAnimatedString className
  • readonly CSSStyleDeclaration style
Norme SVG 1.1 (2e Édition)

Propriétés

Nom Type Description
className SVGAnimatedString Correspond à l'attribut class de l'élément donné.
style CSSStyleDeclaration Correspond à l'attribut style de l'élément donné.

Métodes

Nom & Arguments Retourne Description
getPresentationAttribute(in DOMString name) CSSValue Retourne la valeur par défaut (ex: static) d'un attribut de présentation donné comme un objet de type CSSValue.
The returned object is live; changes to the objects represent immediate changes to the objects to which the CSSValue is attached.
