Obsolète depuis Gecko 25 (Firefox 25 / Thunderbird 25 / SeaMonkey 2.22)
Cette fonctionnalité est obsolète. Bien qu'encore supportée par des navigateurs, son utilisation est découragée pour tout nouveau projet. Évitez de l'utiliser.

L'interface DocumentTouch était utilisée pour fournir des méthodes pratiques pour la création des objets Touch et TouchList, mais elle a été supprimée. Les méthodes sont maintenant présentes dans l'interface Document.

Méthodes

DocumentTouch.createTouch()
Crée un nouvel objet Touch.
DocumentTouch.createTouchList()
Crée un nouvel objet TouchList.
