Une interface MessageEvent permet de recevoir un message (une donnée texte), grâce soit au WebSocket soit par un WebRTC RTCDataChannel. 
 

L'interface dispose d'un gestionnaire d'événements déclenché par l'événement WebSocket.onmessage ou RTCDataChannel.onmessage.

Note : Cette fonctionnalité est disponible via les Web Workers.

MessageEvent()
