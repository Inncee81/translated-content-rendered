Important: Application Cache est déprécié depuis Firefox 44, et ne sera plus disponible dans des contextes non sécurisés avant Firefox 60 (bug 1354175, currently uniquement sur les versions Nightly/Beta). Ne pas utiliser cette fonctionnalité sur des sites web en production, utilisez plutôt les services workers.

La propriété en lecture seule applicationCache de l'interface SharedWorkerGlobalScope retourne l'objet ApplicationCache pour le worker (voir Utiliser le cache d'application).

Syntaxe

var nameObj = self.applicationCache;

valeur

Un ApplicationCache.

Exemple

Si un worker partagé a un AppCache, vous pouvez récupérer une référence en utilisant à l'intérieur d'un worker partagé:

self.applicationCache
