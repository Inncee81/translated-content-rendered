L'interface HTMLFormControlsCollection représente une collection d'éléments de contrôle de formulaires. Il représente les listes retournées par le HTMLFormElement et le HTMLFieldSetElement de la propriété d'interface elements.

Cette interface remplace une méthode de HTMLCollection, sur laquelle elle est basée.

Propriétés

Cette interface hérite des propriétés de son parent HTMLCollection.

Méthodes

Cette interface hérite des méthodes de son parent HTMLCollection.

HTMLFormControlsCollection.namedItem()
Renvoie la RadioNodeList (liste de noeuds radio) ou l'Element dans la collection dont le name (nom) ou l'id (identifiant) correspond au nom spécifié, ou null si aucun noeud ne correspond. Noter que cette version de namedItem() cache celui hérité de HTMLCollection. Comme pour le premier, en JavaScript, en utilisant le crochet de la syntaxe de tableau (array) avec une String (chaîne de caractères), collection["value"] est équivalent à collection.namedItem("value").
