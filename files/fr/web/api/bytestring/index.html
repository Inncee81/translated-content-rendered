ByteString est une chaîne de caractères en UTF-8 qui contient l'ensemble des séquences d'octets possibles. ByteString correspond à une chaîne de caractères (type String) lorsqu'il est manipulé en JavaScript. En général, cet objet est uniquement utilisé dans l'interfaçage de protocoles qui utilisent les octets et les chaînes de façon interchangeable (HTTP par exemple).
