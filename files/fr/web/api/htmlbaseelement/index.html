L'interface HTMLBaseElement contient l'URI de base pour un document. Cet objet hérite de toutes les propriétés et des méthodes telles quelles sont décrites dans l'interface HTMLElement.

Propriétés

Hérite des propriétés de son parent, HTMLElement.

HTMLBaseElement.href
est un DOMString qui reflète l'attribut HTML href, contenant l'URL de base pour les URLs relatives dans le document.
HTMLBaseElement.target
est un DOMString qui reflète l'attribut HTML target, contenant un contexte de navigation cible par défaut ou une frame pour les éléments qui n'ont pas de cibe spécifiée.

Methodes

Pas de méthodes spécifiques; Hérite des propriétés de son parent, HTMLElement.
