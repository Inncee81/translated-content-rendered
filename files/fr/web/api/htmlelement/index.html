Interface des éléments HTML

HTMLElement est une interface abstraite pour le DOM, héritée par tous les éléments HTML.

Propriétés

Nom Type Description
accessKey HTML5 DOMString La touche de navigation clavier assignée à l'élément.
accessKeyLabel HTML5 DOMString Une chaîne de caractères définissant la touche de navigation clavier assignée.
className DOMString Le nom de la classe CSS définie pour l'élément.
dataset HTML5 DOMStringMap Un tableau associatif des attributs  data-* de l'élément.
dir DOMString L'attribut dir de l'élément.
id DOMString L'ID de l'élément.
lang DOMString L'attribut  lang de l'élément.
title DOMString L'attribut title de l'élément.

Méthodes

Nom et paramètres Renvoie Description
blur() HTML5 void Retire le focus clavier de l'élément actuellement sélectionné.
click() HTML5 void Envoie un évènement click à l'élément.
focus() HTML5 void Place le focus clavier sur l'élément.
