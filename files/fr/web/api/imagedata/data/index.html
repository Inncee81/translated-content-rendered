La propriété en lecteur seul ImageData.data retourne un Uint8ClampedArray. Il représente un tableau unidirectionnel contenant les données sous forme d'entiers inclus entre 0 et 255 dans l'ordre RGBA comme ceci : [r1,g1,b1,a1,…,rn;gn;bn;an].

Syntax

imagedata.data

Exemple

var imagedata = new ImageData(100, 100);
imagedata.data; // Uint8ClampedArray[40000]
