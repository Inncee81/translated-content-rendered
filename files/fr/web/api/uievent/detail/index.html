UIEvent.detail est une propriété en lecture seule, qui, lorsqu'elle ne vaut pas zéro, donne le nombre de clics courant (ou suivant en fonction de l'événement).

Pour les événements click ou dblclick, UIEvent.detail donne le nombre de clics courant.

Pour les événements mousedown ou mouseup, UIEvent.detail donne le nombre de clics courant plus un.

Pour tous les autres objets UIEvent, UIEvent.detail vaut toujours zéro.
