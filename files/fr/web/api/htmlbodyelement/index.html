L'interface HTMLBodyElement des propriétés particulières (au-delà de celles de l'interface HTMLElement dont-elle hérite également) pour manipuler les éléments.

propriétés

Propriétés hérite de son parent, HTMLElement et de WindowEventHandlers.

HTMLBodyElement.aLink
Est un DOMString qui représente la couleur des liens hypertextes actifs.
HTMLBodyElement.background
Est un DOMString qui représente la description de l'emplacement de la ressource d'image d'arrière-plan. Notez que ce n'est pas un URI, même si certaines anciennes versions de certains navigateurs s'y attendent.
HTMLBodyElement.bgColor
Est un DOMString qui représente la couleur de fond du document.
HTMLBodyElement.link
Est un DOMString qui représente la couleur des liens non visités.
HTMLBodyElement.text
Est un DOMString qui représente la couleur de premier plan du texte.
HTMLBodyElement.vLink
Est un DOMString qui représente la couleur des liens visités.

méthodes

Aucune méthode spécifique; méthodes hérite de ses parents, HTMLElement et de WindowEventHandlers.

Les gestionnaires d'événements

Pas de gestionnaire d'événement spécifique; gestionnaires d'événements hérite de ses parents, HTMLElement et de WindowEventHandlers.

WindowEventHandlers.onafterprint
Est un EventHandler représentant le code d'appel lorsque l'événement afterprint est déclenché.
WindowEventHandlers.onbeforeprint
Est un EventHandler représentant le code d'appel lorsque l'événement beforeprint est déclenché.
WindowEventHandlers.onbeforeunload
Est un EventHandler représentant le code d'appel lorsque l'événement beforeunload est déclenché.
WindowEventHandlers.onhashchange
Est un EventHandler représentant le code d'appel lorsque l'événement hashchange est déclenché.
WindowEventHandlers.onlanguagechange
Est un EventHandler représentant le code d'appel lorsque l'événement languagechange est déclenché.
WindowEventHandlers.onmessage
Est un EventHandler représentant le code d'appel lorsque l'événement message est déclenché.
WindowEventHandlers.onoffline
Est un EventHandler représentant le code d'appel lorsque l'événement offline est déclenché.
WindowEventHandlers.ononline
Est un EventHandler représentant le code d'appel lorsque l'événement online est déclenché.
WindowEventHandlers.onpagehide
Est un EventHandler représentant le code d'appel lorsque l'événement pagehide est déclenché.
WindowEventHandlers.onpageshow
Est un EventHandler représentant le code d'appel lorsque l'événement pageshow est déclenché.
WindowEventHandlers.onpopstate
Est un EventHandler représentant le code d'appel lorsque l'événement popstate est déclenché.
WindowEventHandlers.onresize
Est un EventHandler représentant le code d'appel lorsque l'événement resize est déclenché.
WindowEventHandlers.onstorage
Est un EventHandler représentant le code d'appel lorsque l'événement storage est déclenché.
WindowEventHandlers.onunload
Est un EventHandler représentant le code d'appel lorsque l'événement unload est déclenché.
