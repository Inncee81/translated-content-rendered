La propriété search de l'interface URL est une chaine d'interrogation, aussi appellée une chaine de recherche, c'est un USVString contenant le caractère '?' suivi des paramètres de l'URL.

Les navigateurs récents fournissent la propriété URL.searchParams qui permet d'analyser plus facilement les paramètres passés à la chaîne de recherche.

Note : Cette fonctionnalité est disponible via les Web Workers.

Syntaxe

chaineDeCaractere = objet.search;
object.search = chaineDeCaractere;

Valeur

Un USVString.

Exemples

var url = new URL('https://developer.mozilla.org/en-US/docs/Web/API/URL/search?q=123');
var queryString = url.search; // Retourne:"?q=123"
