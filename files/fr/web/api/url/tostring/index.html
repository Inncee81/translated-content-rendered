La méthode URL.toString() retourne un USVString contenant tout l'URL. C'est en effet, une version de URL.href en lecture seule.

Note : Cette fonctionnalité est disponible via les Web Workers.

Syntaxe

string = url.toString();

Paramètres

Aucun.

Valeur de retour

Un USVString.

Examples

const url = new URL("https://developer.mozilla.org/en-US/docs/Web/API/URL/toString");
url.toString() // doit retourner l'URL en tant que chaîne
