La méthode statique URL.revokeObjectURL() libère une URL d’objet existante précédemment créée par un appel à URL.createObjectURL(). Appelez cette méthode quand vous n’utilisez plus une URL d’objet pour indiquer au navigateur qu’il ne doit plus garder de référence sur l’objet.

Note : Cette fonctionnalité est disponible via les Web Workers.

Syntaxe

window.URL.revokeObjectURL(objectURL);

Paramètres

objectURL
Une DOMString représentant une URL d’objet qui a été précédemment créée par un appel à createObjectURL().

Valeur de retour

undefined.
