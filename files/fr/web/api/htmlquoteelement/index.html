Interface des éléments HTML de citation

Les objets DOM quote expose l'interface HTMLQuoteElement (ou HTML 4 HTMLQuoteElement), qui fournissent plusieurs propriétés pour manipuler les éléments de citation (et viennent s'ajouter aux propriétés habituelles pour un objet de l'interface element, disponibles par héritage).

Propriétés

Nom Type Description
cite DOMString Reflète l'attribut HTML cite contenant l'URL de la source de la citation.

Voir également

Cette interface est supportée par les deux éléments HTML <blockquote> et <q>.
