La propriété onopen de l'interface EventSource est un EventHandler qui est appelé lorsqu'un est évènement open est reçu, indiquant que la connexion vient d'être établie.

Syntaxe

eventSource.onopen = function

Exemples

evtSource.onopen = function() {
  console.log("Connexion au serveur établie.");
};

Note: Vous pouvez trouver un exemple complet sur GitHub — voir Simple SSE demo using PHP.
