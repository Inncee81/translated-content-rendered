Un objet qui implémente l'interface StyleSheet représente une seule feuille de style. Les feuilles de style CSS implémenteront également l'interface plus spécialisée CSSStyleSheet.

Propriétés

StyleSheet.disabled
Un booléen (Boolean) indiquant si la feuille de style est appliquée ou non.
StyleSheet.href Lecture seule
Une chaîne de caractères (DOMString) représentant l'emplacement de la feuille de style.
StyleSheet.media Lecture seule
Une MediaList représentant le média de destination pour les informations de style.
StyleSheet.ownerNode Lecture seule
Un objet Node qui représente le nœud associant cette feuille de style avec le document courant.
StyleSheet.parentStyleSheet Lecture seule
Un objet StyleSheet qui inclut l'objet courant si ce dernier possède un parent ou null s'il n'y en a pas.
StyleSheet.title Lecture seule
Une chaîne de caractères (DOMString) indiquant le titre indicatif de la feuille de style courante.
StyleSheet.type Lecture seule
Une chaîne de caractères DOMString indiquant le langage de représentation pour cette feuille de style.
