Contexte sécurisé
Cette fonctionnalité est uniquement disponible dans des contextes sécurisés (HTTPS), pour certains navigateurs qui la prennent en charge.

L'interface AuthenticatorResponse, rattachée à l'API Web Authentication, constitue l'interface de base pour les interfaces qui fournissent une racine cryptographique de confiance pour une paire de clés. Les interfaces filles contiennent des informations relatives au navigateur telles que l'origine du challenge et qui peuvent être renvoyée par PublicKeyCredential.response.

Propriétés

AuthenticatorResponse.clientDataJSON
Une chaîne de caractères JSON stockée dans un ArrayBuffer et qui représente les données du client passées à CredentialsContainer.create() ou CredentialsContainer.get().
