La propriété GlobalEventHandlers.onreset contient un EventHandler (gestionnaire d'évènements) déclenché quand un évènement reset est reçu.

Syntaxe

window.onreset = funcRef;

Paramètres

  • funcRef est une référence à une fonction.

Exemple

<html>
<script>
function reg() {
  window.captureEvents(Event.RESET);
  window.onreset = hit;
}

function hit() {
 alert('hit');
}
</script>

<body onload="reg();">
 <form>
   <input type="reset" value="reset" />
 </form>
 <div id="d"> </div>
</body>
</html>

Notes

L'évènement reset est déclenché quand l'utilisateur clique sur le bouton de réinitialisation dans un formulaire (<input type="reset"/>).
