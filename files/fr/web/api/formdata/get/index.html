La méthode get() de l'interface FormData renvoie la première valeur associée à une clé donnée dans un objet FormData. Si vous attendez plusieurs valeurs et que vous les voulez toutes, utilisez plutôt la méthode getAll().

Note : Cette méthode est disponible dans les Web Workers.

Syntaxe

formData.get(name);

Paramètres

name
Un USVString représentant le nom de la clé que vous souhaitez retrouver.

Valeur retournée

Un FormDataEntryValue contenant la valeur. Si la clé n'existe pas, la méthode renvoie zéro.

Exemple

La ligne suivante crée un objet FormData vide :

var formData = new FormData();

Si nous ajoutons deux valeurs username en utilisant FormData.append:

formData.append('username', 'Chris');
formData.append('username', 'Bob');

La fonction suivante get() peut uniquement retrouner la première valeur username annexée :

formData.get('username'); // Returns "Chris"
