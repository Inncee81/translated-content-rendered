La méthode getAll() de l'interface FormData renvoie toutes les valeurs associées à une clé donnée à partir d'un objet FormData.

Note : Cette méthode est disponible dans les Web Workers.

Syntaxe

formData.getAll(name);

Paramètres

name
Un USVString représentant le nom de la clé que vous voulez récupérer.

Retours

Un tableau de FormDataEntryValue dont la clé correspond à la valeur passée dans le paramètre name. Si la clé n'existe pas, la méthode renvoie une liste vide.

Exemple

La ligne suivante crée un objet FormData vide :

var formData = new FormData();

Si nous ajoutons deux valeurs username en utilisant FormData.append :

formData.append('username', 'Chris');
formData.append('username', 'Bob');

La fonction getAll() suivante va retourner chaque valeurs de username dans un tableau :

formData.getAll('username'); // Returns ["Chris", "Bob"]
