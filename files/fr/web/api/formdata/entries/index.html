La methode FormData.entries() retourne un iterateur permettant d'accéder aux paires clefs/valeurs contenues dans cet objet.
La clef de chaque paire est une USVString. De la même manière, la valeur peut être une USVString ou un Blob.

Note: Cette methode est disponible dans les Web Workers.

Syntaxe

formData.entries();

Valeur retournée

Retourne un iterateur.

Exemple

// Creation d'un objet FormData
var formData = new FormData();
formData.append('key1', 'value1');
formData.append('key2', 'value2');

// Affichage des paires clefs/valeurs
for(var pair of formData.entries()) {
   console.log(pair[0]+ ', '+ pair[1]);
}

Le resultat est:

key1, value1
key2, value2
