La méthode FormData.values() renvoie une itération permettant de passer en revue toutes les valeurs contenues dans cet objet. Les valeurs sont des objets USVString ou Blob.

Note : Cette méthode est disponible dans les Web Workers.

Syntaxe

formData.values();

Valeur de retour

Retourne une itération .

Exemple

// Créer un objet FormData test
var formData = new FormData();
formData.append('cle1', 'valeur1');
formData.append('cle2', 'valeur2');

// Affiche les valeurs
for (var value of formData.values()) {
   console.log(value);
}

Le résultat est :

valeur1
valeur2
