L'interface CanvasGradient  représente un objet opaque décrivant un dégradé. Il est retourné par les méthodes CanvasRenderingContext2D.createLinearGradient() ou CanvasRenderingContext2D.createRadialGradient().

Propriétés

Représentant un objet opaque, il n'y a aucune propriété exposée.

Méthodes

Il n'y a pas de méthode héritée.

CanvasGradient.addColorStop()
Ajoute un nouveau point d'arrêt, défini par un décalage et une couleur. Si le décalage n'est pas compris entre 0 et 1, une INDEX_SIZE_ERR est générée ; si la couleur ne peut être analysée comme une <color> CSS,  une SYNTAX_ERR est générée.
