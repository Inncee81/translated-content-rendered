La propriété en lecture seule NavigatorStorage.storage renvoie le singleton StorageManager utilisé pour accéder aux capacités globales de stockage du navigateur pour le site actuel ou l’application actuelle. L’objet renvoyé permet d’examiner et de configurer la persistance des stockages de données, et savoir approximativement combien d’espace est encore disponible dans le navigateur pour utiliser le stockage local.

Syntaxe

var storageManager = navigator.storage;

Valeur

Un objet StorageManager permettant de maintenir la persistance des données stockées, et de déterminer grossièrement combien d’espace est disponible pour stocker des données.
