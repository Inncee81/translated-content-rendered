Non standard
Cette fonctionnalité n'est ni standard, ni en voie de standardisation. Ne l'utilisez pas pour des sites accessibles sur le Web : elle ne fonctionnera pas pour tout utilisateur. Il peut également y avoir d'importantes incompatibilités entre les implémentations et son comportement peut être modifié dans le futur.

Le constructeur FederatedCredential permet de créer un nouvel objet FederatedCredential.

Pour les navigateurs qui le prennent en charge, une instance de cette classe pourra être passée comme propriété credential pour l'objet init utilisé comme argument pour fetch.

Syntaxe

var mesCredentials = new FederatedCredential(init)

Paramètres

init
Les propriétés de cet objet sont :
  • provider : une chaîne de caractères USVString identifiant le fournisseur d'information d'authentification.
