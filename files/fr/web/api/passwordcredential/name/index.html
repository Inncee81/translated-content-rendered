La propriété name est accessible en lecture seule et est rattachée à l'interface PasswordCredential. C'est une chaîne de caractères USVSTring qui contient un nom public, compréhensible pour un humain et qui est destiné à être affiché dans un sélecteur de fournisseur d'authentification.

Syntaxe

name = passwordCredential.name

Valeur

A USVString containing a name.
