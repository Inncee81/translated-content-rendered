La propriété iconURL, rattachée à l'interface PasswordCredential, est accessible en lecture seule et fournit une chaîne de caractères USVString représentant une URL pointant vers l'image d'une icône. Cette image est destinée à être affichée dans un sélecteur d'authentification. L'URL fournie doit être accessible sans authentification.

Syntaxe

url = passwordCredential.iconURL

Valeur

Une chaîne de caractères USVString contenant une URL.
