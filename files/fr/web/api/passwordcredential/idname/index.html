Obsolète
Cette fonctionnalité a été supprimée des standards du Web. Bien que quelques navigateurs puissent encore la supporter, elle est en cours d'éradication. Ne l'utilisez ni dans d'anciens projets, ni dans de nouveaux. Les pages et applications Web l'utilisant peuvent cesser de fonctionner à tout moment.

La propriété idName de l'interface PasswordCredential fournit une chaîne de caractères USVString représentant le nom qui sera utilisé pour le champ d'identifiant, lorsqu'on envoie l'objet courant à un point d'accès distant when submitting the current object to a remote endpoint via fetch. Par défaut, cette propriété vaut 'username', mais on peut la définir avec n'importe quelle valeur qui serait attendue de la part d'un service distant.

Syntaxe

var idName = PasswordCredential.idName
PasswordCredential.idName = "userID"

Valeur

Une chaîne de caractères USVString qui représente le nom utilisé pour le champ d'identifiant lorsqu'on envoie l'objet courant à un point distant avec fetch.
