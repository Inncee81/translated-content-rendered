Note : Cette fonctionnalité est disponible via les Web Workers.

Contexte sécurisé
Cette fonctionnalité est uniquement disponible dans des contextes sécurisés (HTTPS), pour certains navigateurs qui la prennent en charge.

La propriété onerror de l'interface Notification spécifie un écouteur d'événements pour recevoir les événements error. Ces événements se produisent lorsque quelque chose ce déroule mal avec une Notification (dans de nombreux cas, une erreur qui empêche l'affichage de la notification.)

Syntaxe

Notification.onerror = function(event) { ... }
Notification.onerror = (even) => { ... }
