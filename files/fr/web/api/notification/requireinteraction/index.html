Note : Cette fonctionnalité est disponible via les Web Workers.

Contexte sécurisé
Cette fonctionnalité est uniquement disponible dans des contextes sécurisés (HTTPS), pour certains navigateurs qui la prennent en charge.

La propriété en lecture seule requireInteraction de l'interface Notification renvoie un Boolean indiquant qu'une notification doit rester active jusqu'à ce que l'utilisateur clique dessus ou la rejette, plutôt que de se fermer automatiquement. Ceci peut être défini à l'aide de l'option requireInteraction lors de la création de la notification à l'aide du constructeur Notification.Notification().

Syntaxe

Notification.requireInteraction

Valeur

Un Boolean.
