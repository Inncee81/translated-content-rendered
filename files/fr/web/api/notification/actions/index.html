Note : Cette fonctionnalité est disponible via les Web Workers.

Contexte sécurisé
Cette fonctionnalité est uniquement disponible dans des contextes sécurisés (HTTPS), pour certains navigateurs qui la prennent en charge.

La propriété actions en lecture seule de l'interface Notification renvoie la liste des NotificationAction objets définis à l'aide de l'option actions lors de la création de la notification à l'aide du constructeur Notification(). Il s'agit d'une liste des actions définies par l'application que l'utilisateur peut choisir de prendre directement depuis le contexte d'une notification.

Note: Le périphérique et l'agent utilisateur peuvent ne pouvoir afficher qu'un nombre limité d'actions (en raison, par exemple, d'un espace d'écran limité). Cette limite peut être déduite de Notification.maxActions.

Syntaxe

Notification.actions

Valeur

Un tableau en lecture seule d'objets NotificationAction, chacun décrivant une action unique que l'utilisateur peut choisir dans une notification.
