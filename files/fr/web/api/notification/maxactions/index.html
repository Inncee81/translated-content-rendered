Note : Cette fonctionnalité est disponible via les Web Workers.

Contexte sécurisé
Cette fonctionnalité est uniquement disponible dans des contextes sécurisés (HTTPS), pour certains navigateurs qui la prennent en charge.

L'attribut maxActions de l'interface Notification renvoie le nombre maximal d'actions prises en charge par l'appareil et l'agent utilisateur. En effet, c'est le nombre maximum d'éléments dans le tableau Notification.actions qui seront respectés par l'agent utilisateur.

Syntaxe

Notification.maxActions

Valeur

Un entier Number qui indique le plus grand nombre d'actions de notification pouvant être présentées à l'utilisateur par l'agent utilisateur et l'appareil.

Exemples

L'extrait de code suivant enregistre le nombre maximal d'actions prises en charge.

const { maxActions } = Notification
console.log(`This device can display at most ${maxActions} actions on each notification.`);
