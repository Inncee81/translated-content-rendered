Note : Cette fonctionnalité est disponible via les Web Workers.

Contexte sécurisé
Cette fonctionnalité est uniquement disponible dans des contextes sécurisés (HTTPS), pour certains navigateurs qui la prennent en charge.

La propriété en lecture seule silent de l'interface Notification spécifie si la notification doit être silencieuse, c'est-à-dire qu'aucun son ou vibration ne doit être émis, quels que soient les paramètres de l'appareil. Ceci est comme définis à l'aide de l'option silent lors de la création de la notification à l'aide du constructeur Notification().

Syntaxe

Notification.silent

Valeur

Un Boolean. false est la valeur par défaut; true rend la notification silencieuse.
