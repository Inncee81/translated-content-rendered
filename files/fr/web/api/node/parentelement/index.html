La propriété en lecture seule Node.parentElement renvoie le parent du noeud DOM (Element) ou null si ce dernier n'a pas de parent ou si le parent n'est pas un Element du DOM.

Syntaxe

parentElement = node.parentElement

parentElement référence l'élément parent d'un nœud (node). C'est toujours un objet Element du DOM ou null.

Exemple

if (node.parentElement) {
    node.parentElement.style.color = "red";
}

Compatibilité des navigateurs

Sur quelques navigateurs, la propriété parentElement est seulement définie sur les noeuds qui sont eux-mêmes des Element. En particulier, elle n'est pas définie sur les noeuds texte.

The compatibility table on this page is generated from structured data. If you'd like to contribute to the data, please check out https://github.com/mdn/browser-compat-data and send us a pull request.
