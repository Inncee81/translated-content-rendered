Obsolète
Cette fonctionnalité est obsolète. Bien qu'encore supportée par des navigateurs, son utilisation est découragée pour tout nouveau projet. Évitez de l'utiliser.

La méthode Node.getUserData()renvoie tout DOMUserData d'utilisateur défini précédemment sur le noeud donné par Node.setUserData().

Les méthodes Node.setUserData et Node.getUserData ne sont plus disponibles pour le contenu à partir du web. Element.dataset ou WeakMap peuvent être utilisés à la place.

Syntaxe

userData = someNode.getUserData(userKey);

Paramètres

  • userKey est la clé pour choisir les données spécifiques recherchées pour le nœud donné. Plus d'une clé peuvent avoir été assignées sur un nœud donné, contenant leurs propres valeurs.

Exemple

var d = document.setUserData('key', 15, null);
console.log(document.getUserData('key')); // 15
