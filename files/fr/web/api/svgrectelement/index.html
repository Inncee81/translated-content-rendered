Propriétés

Cette interface hérite aussi des propriétés de SVGGeometryElement.

SVGRectElement.x Lecture seule
Retourne un SVGAnimatedLength correspondant à l'attribut x de l'élément <rect> donné.
SVGRectElement.x Lecture seule
Retourne un SVGAnimatedLength  correspondant à l'attribut x de l'élément <rect> donné.
SVGRectElement.y Lecture seule
Retourne un SVGAnimatedLength correspondant à l'attribut y de l'élément <rect> donné.
SVGRectElement.width Lecture seule
Retourne un SVGAnimatedLength correspondant à l'attribut width de l'élément <rect> donné.
SVGRectElement.height Lecture seule
Retourne un SVGAnimatedLength correspondant à l'attribut height de l'élément <rect> donné.
SVGRectElement.rx Lecture seule
Retourne un SVGAnimatedLength correspondant à l'attribut rx de l'élément <rect> donné.
SVGRectElement.ry Lecture seule
Retourne un SVGAnimatedLength correspondant à l'attribut ry de l'élément <rect> donné.

Méthode

Cette interface n'implémente pas de méthodes spécifiques mais elle hérite des méthodes parentes de SVGGeometryElement.
