La méthode navigate() de l'interface  WindowClient charge une URL spécifiée dans une page de client contrôlée, puis retourne une  Promise  qui devra être analysée par  WindowClient (le demandeur).

Syntaxe

WindowClient.navigate(url).then(function(WindowClient) {
  // do something with your WindowClient after navigation
});

Paramètres

url
L'emplacement pour naviguer vers (url est une string) .

Retour

Une Promise  qui sera analysée par le demandeur WindowClient (fonctionnement asynchrone : je te promet de faire, mais je suis pas sûr, à toi de vérifier) .
