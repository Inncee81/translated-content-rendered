L'interface GeolocationPosition représente la position de l'appareil concerné à un instant donné. La position, représentée par un objet GeolocationCoordinates, comprend la position 2D de l'appareil, sur un sphéroïde représentant la Terre, mais aussi son altitude et sa vitesse.

Propriétés

L'interface GeolocationPosition n'hérite d'aucune propriété.

GeolocationPosition.coords Lecture seule Contexte sécurisé
Renvoie un objet GeolocationCoordinates définissant l'emplacement actuel.
GeolocationPosition.timestamp Lecture seule Contexte sécurisé
Renvoie un DOMTimeStamp représentant l'heure à laquelle l'emplacement a été récupéré.

Méthodes

L'interface GeolocationPosition n'implémente ni n'hérite d'aucune méthode.
