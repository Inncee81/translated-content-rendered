La propriété en lecture seule scope de l'interface ServiceWorkerRegistration renvoie un identifiant unique pour un enregistrement de service worker. Le service worker doit être sur la même origine que le document qui enregistre le ServiceWorker.

Note: Cette fonctionnalité est disponible dans Web Workers.

Syntaxe

serviceWorkerRegistration.scope

Spécification État Commentaire
