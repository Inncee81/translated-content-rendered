La propriété en lecture seule WheelEvent.deltaZ est un double représentant la quantité de défilement le long de l'axe z, dans l'unité WheelEvent.deltaMode.

Syntaxe

var dZ = event.deltaZ;

Exemple

var syntheticEvent = new WheelEvent("syntheticWheel", {"deltaZ": 4, "deltaMode": 0});

console.log(syntheticEvent.deltaZ);
