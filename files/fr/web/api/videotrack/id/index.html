La propriété id contient une chaîne qui identifie de manière unique une piste représentée par VideoTrack. Cet ID peut être utilisé avec la méthode VideoTrackList.getTrackById() pour localiser une piste spécifique dans le media associé à l'élément media.

Cet ID peut aussi être utilisé comme un fragement d'URL pour charger une piste (si le media supporte les fragments de media).

Syntaxe

var trackID = VideoTrack.id;

Valeur

Une DOMString qui identifie la piste, utilisable par la suite pour l'appel de getTrackById() sur une VideoTrackList tel que spécifié par la propriété videoTracks d'un élément media.
