Le Navigator.serviceWorker est une propriété en lecture seule qui renvoie un objet ServiceWorkerContainer, et qui permet d'accéder à l'enregistrement, la suppression, la mise à jour, et la communication avec les objets ServiceWorker pour le document  associé.

syntaxe

var workerContainerInstance = navigator.serviceWorker;

Valeur

L'objet ServiceWorkerContainer.

 

Exemples

Le code ci-dessous vérifie si le navigateur supporte les service workers.

if ('serviceWorker' in navigator) {
  // Supported!
}
