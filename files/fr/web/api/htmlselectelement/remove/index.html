La méthode HTMLSelectElement.remove() enlève l'option à l'index choisi dans la liste des options de cet objet select.

Syntaxe

collection.remove(index);

Paramètre

  • index est un entier long définissant la position du HTMLOptionElement à enlever de la collection. Si rien n'est trouvé à cette position dans la liste, la méthode n'a aucun effet.

Exemple

var sel = document.getElementById("maListe");
sel.remove(1);

/*
  Prend l'objet select existant suivant :

  <select id="maListe" name="maListe">
    <option value="1">Option: Value 1</option>
    <option value="2">Option: Value 2</option>
    <option value="3">Option: Value 3</option>
  </select>

  Et le transforme en:

  <select id="maListe" name="maListe">
    <option value="1">Option: Value 1</option>
    <option value="3">Option: Value 3</option>
  </select>
*/
