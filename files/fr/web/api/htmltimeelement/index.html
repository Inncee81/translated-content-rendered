Propriétés

Hérite des propriétés de HTMLElement.

HTMLTimeElement.dateTime
C'est une DOMString qui reflète l'attribut datetime de l'élément HTML correspondant, dans une forme interprétable par un ordinateur de la date et de l'heure.

Métodes

Pas de méthode spécifique, mais cette interface hérite des méthodes de HTMLElement.
