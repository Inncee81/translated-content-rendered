Obsolète
Cette fonctionnalité est obsolète. Bien qu'encore supportée par des navigateurs, son utilisation est découragée pour tout nouveau projet. Évitez de l'utiliser.

L'interface ElementTraversal définissait des méthodes permettant d'accèder depuis un Node (noeud) à un autre dans l'arbre du document.

Elle s'est avérée inutile, car très peu de types de Node ont pu implémenter toutes ses méthodes et propriétés. Elle a été scindée en deux interfaces, ChildNode et ParentNode contenant les méthodes et propriétés utiles pour chacun des types de noeud.

Comme c'était une pure interface, sans aucun objet de ce type, ce changement n'a eu aucun effet sur le web.
