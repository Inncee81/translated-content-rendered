La propriété, en lecture seule, Window.crypto retourne l'objet Crypto  associé à l'objet global. Cet objet permet aux pages web d'avoir un accès à certains services liés à cryptographie.

Syntaxe

var cryptoObj = window.crypto || window.msCrypto; // pour IE 11
