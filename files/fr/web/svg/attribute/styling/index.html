Les attributs SVG de style sont tous les attributs qui peuvent être spécifiés sur tout élément SVG pour appliquer des styles CSS.

  • class
  • style

Attributs

class
Assigne un nom de classe ou un ensemble de noms de classe à un élément. Il fonctionne de manière identique à l'attribut class en HTML.
Valeur: Tout ID valide; Animation: Oui
style
Spécifie les informations de style d'un élément. Il fonctionne de manière identique à l'attribut style en HTML.
Valeur: Toute chaîne de caractères valide; Animation: Non
