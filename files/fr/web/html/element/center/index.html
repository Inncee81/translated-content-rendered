Obsolète
Cette fonctionnalité est obsolète. Bien qu'encore supportée par des navigateurs, son utilisation est découragée pour tout nouveau projet. Évitez de l'utiliser.

L’élément HTML <center> est un élément de type bloc qui contient des paragraphes et d'autres éléments de type bloc ou en ligne. Le contenu entier de cet élément est centré horizontalement au sein de son conteneur parent (généralement l'élément <body>).

Cet élément a été déprécié en HTML4 (et XHTML 1) au profit de la propriété CSS text-align qui peut être appliqué à l'élément <div> ou à un élément <p>. Pour centrer des blocs, on utilisera d'autres propriétés (margin-left et margin-right avec la valeur auto par exemple ou les boîtes flexibles).

Interface DOM

Cet élément implémente l'interface HTMLElement.

Note d'implémentation : Jusqu'à Gecko 1.9.2 inclus, Firefox implémentait l'interface HTMLSpanElement pour cet élément.

Exemples

Premier exemple (HTML)

HTML

<center>
  Ce texte sera centré.
  <p>Ainsi que ce paragraphe.</p>
</center>

Résultat

Deuxième exemple (CSS)

HTML

<div class="centrer">
  Ce texte sera centré.
  <p>Ainsi que ce paragraphe.</p>
</div>

CSS

.centrer {
  text-align: center;
}

Résultat

Troisième exemple (CSS)

HTML

<p class="centrer">
  Cette ligne sera centrée.<br>
  Ainsi que cette ligne.
</p>

CSS

.centrer {
  text-align: center;
}

Résultat

Notes

Appliquer text-align: center sur un élément <p> ou <div> centre le contenu de ces éléments, tout en laissant leurs dimensions générales inchangées.
