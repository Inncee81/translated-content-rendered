Le type de donnée <dimension> représente une valeur de type <number> directement suivie d'une unité : par exemple 10px.

CSS utilise des dimensions pour définir des distances (type <length>), des durées (type <time>), des fréquences (type <frequency>), des résolutions (type <resolution>) ainsi que d'autres quantités.

Syntaxe

La syntaxe d'une valeur de type <dimension> est un nombre (valeur de type <number>) immédiatement suivi d'une unité représentée par son identifiant. Les identifiants utilisés pour les unités sont insensibles à la casse.

Exemples

Dimensions valides

12px      12 pixels
1rem      1 rem
1.2pt     1.2 points
2200ms    2200 millisecondes
200hz     200 Hertz
200Hz     200 Hertz (les unités sont insensibles à la casse)

Dimensions invalides

12 px       L'unité doit immédiatement être indiquée après le nombre
12"px"      Les unités sont des identifiants et ne doivent pas être entourées de quotes
