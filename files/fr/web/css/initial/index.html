Le mot-clé initial applique la valeur initiale d'une propriété à un élément. La valeur initiale est fournie par le navigateur et peut être utilisée pour chaque propriété CSS. Cette propriété prendra alors la valeur initiale spécifiée pour cette la propriété.

La propriété all peut être utilisée avec cette valeur afin de réinitialiser l'ensemble des propriétés CSS.

Note : Pour les propriétés héritées, la valeur initiale peut donner des résultats étranges et mieux vaudra utiliser les mots-clés inherit, unset ou revert. Attention également à ne pas confondre la valeur initiale avec la valeur décrite dans la feuille de style portée par le navigateur.

Exemples

CSS

/* On écrit le texte en rouge */
.exemple {
  color: red;
}

/* Et on veut que le texte mis en avant
   prenne la valeur initiale de
   color */
.exemple em {
  color: initial;
}

HTML

 <p class="exemple">
    Texte rouge
       <em>
          ce texte est avec la couleur initiale (noire)
       </em>
    ce texte est rouge aussi
 </p>
