Le descripteur font-family permet aux auteurs d'un document de définir la police de caractères à utiliser au sein d'une règle @ @font-face.

Syntaxe

/* Valeurs de chaînes de caractères */
/* Type <string>                    */
font-family: "police de caractères a";
font-family: 'une autre police';

/* Valeur de type <custom-ident> */
font-family: exemplepolice;

Valeurs

<family-name>
Définit le nom de la police de caractères.

Syntaxe formelle

<family-name>
où 
<family-name> = <string> | <custom-ident>+

Exemples

@font-face {
  font-family: exemplepolice;
  src: url('exemplepolice.ttf');
}
