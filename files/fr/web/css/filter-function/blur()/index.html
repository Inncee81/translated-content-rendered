La fonction CSS blur() permet d'appliquer une flou gaussien sur l'image d'entrée. Le résultat de cette fonction est une valeur <filter-function>.

Le code source de cet exemple interactif est disponible dans un dépôt GitHub. Si vous souhaitez contribuez à ces exemples, n'hésitez pas à cloner https://github.com/mdn/interactive-examples et à envoyer une pull request !

Syntaxe

blur(rayon)

Paramètres

rayon
Le rayon d'application du flou, défini par une longueur CSS (<length>). Cela définit l'écart-type de la fonction gaussienne appliquée (c'est-à-dire le nombre de pixels qui se mélangeront). Ainsi, plus la valeur est grande, plus le flou sera prononcé. Si on utilise la longueur nulle 0, l'image ne sera pas modifiée. La valeur utilisée par défaut lors d'une interpolation est 0.

Exemples

blur(0);        /* Aucun effet */
blur(8px);      /* Un flou avec un rayon de 8px */
blur(1.17rem);  /* Un flou avec un rayon de 1.17rem */
