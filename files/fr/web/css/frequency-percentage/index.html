Le type de donnée <frequency-percentage> représente une valeur qui peut être une valeur de type frequency ou une valeur de type percentage.

Syntaxe

Se référer à la documentation des types frequency et percentage pour plus de détails sur les syntaxes possibles avec chacun de ces types.

Utilisation avec calc()

Lorsqu'une valeur de type <frequency-percentage> peut être utilisée dans une déclaration, cela signifie que le pourcentage sera résolu comme une fréquence et qu'il peut alors être utilisé dans une expression calc().
