Résumé

normal est une valeur commune pour certaines propriétés CSS et souvent c'est la valeur par défaut. On peut la comparer à la valeur none qui peut être utilisée de manière similaire pour d'autres propriétés.

Utilisation

Cette liste n'est pas exhaustive et les propriétés qui ne sont pas encore supportées par les différents navigateurs ne sont pas incluses.
  • animation-direction
  • content
  • counter-increment
  • counter-reset
  • font-style
  • font-variant
  • font-weight
  • letter-spacing
  • white-space
  • word-spacing
  • word-wrap
