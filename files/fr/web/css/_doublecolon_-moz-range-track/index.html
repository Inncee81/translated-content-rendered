Non standard
Cette fonctionnalité n'est ni standard, ni en voie de standardisation. Ne l'utilisez pas pour des sites accessibles sur le Web : elle ne fonctionnera pas pour tout utilisateur. Il peut également y avoir d'importantes incompatibilités entre les implémentations et son comportement peut être modifié dans le futur.

Le pseudo-élément ::-moz-range-track est un pseudo-élément spsécifique à Mozilla et représente la piste d'un élément <input> de type range.

Note : ::-moz-range-track n'aura aucun effet s'il est utilisé sur autre chose qu'un élément <input type="range">.

Syntaxe

::-moz-range-track

Exemples

CSS

input[type=range]::-moz-range-track {
  background-color: green;
}

HTML

<input type="range" min="0" max="100" step="5" value="50"/>

Spécifications

Ce pseudo-élément est un pseudo-élément propriétaire lié à Gecko/Mozilla et ne fait partie d'aucune spécification.

Voir aussi

  • Les pseudo-éléments de Gecko qui permettent de mettre en forme les autres parties des éléments <input> de type range :
    • ::-moz-range-thumb qui représente le curseur qui se déplace le long de la piste.
    • ::-moz-range-progress qui représente la partie inférieure de la piste (ce qui est déjà « couvert »).
  • Les pseudo-éléments utilisés par les autres navigateurs :
    • ::-webkit-slider-runnable-track pour WebKit/Blink (Safari, Chrome et Opera).
    • ::-ms-track pour Internet Explorer/Edge.
  • CSS-Tricks : Gérer des champs input de type range de façon compatible entre les navigateurs (en anglais)
