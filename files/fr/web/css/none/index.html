Résumé

none est une valeur commune pour la plupart des propriétés CSS et souvent c'est la valeur par défaut. On peut la comparer à la valeur normal qui peut être utilisée de manière similaire pour d'autres propriétés.

Utilisation

Cette liste n'est pas exhaustive et les propriétés qui ne sont pas encore supportées par les différents navigateurs ne sont pas incluses.
  • animation-name
  • background-image
  • border-style (border-bottom-style, border-left-style, border-right-style et border-top-style
  • clear
  • content
  • display
  • float
  • font-size-adjust
  • list-style-image
  • list-style-type
  • max-height
  • max-width
  • outline-style
  • quotes
  • resize
  • text-decoration
  • text-transform
