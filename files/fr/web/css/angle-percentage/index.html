Le type de donnée <angle-percentage> représente une valeur qui peut être une valeur de type angle ou une valeur de type percentage.

Syntaxe

Voir la documentation des types angle et percentage pour plus de détails sur les syntaxes possibles avec chacun de ces types.

Utilisation avec calc()

Lorsqu'il est possible d'utiliser une valeur de type <angle-percentage> dans une déclaration, cela signifie que le pourcentage sera résolu avec une valeur angulaire et qu'il peut être utilisé dans une expression calc().
