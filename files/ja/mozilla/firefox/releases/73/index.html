Firefox 73 は、米国時間 2020 年 2 月 11 日にリリースされました。このページでは、開発者に影響する Firefox 73 の変更点をまとめています。

ウェブ開発者向けの変更点一覧

開発者ツール

  • CORS のエラー をコンソールでエラーとして示す (警告として表示しない) ことで、適切な表示になりました (バグ 1602093)。
  • ウェブコンソールの文字列検索や正規表現検索で、'-' を前につけて否定することが可能になりました (バグ 1291192)。

HTML

変更なし。

CSS

  • overscroll-behavior-x および overscroll-behavior-y を論理的にしたものである overscroll-behavior-block および overscroll-behavior-inline を追加しました (バグ 950799)。

廃止

  • 独自仕様である -moz-touch-enabled メディアクエリーを削除しました (バグ 1486964)。代わりに pointer: coarse を使用してください。

SVG

  • letter-spacing および word-spacing プロパティが SVG で動作するようになりました (バグ 371787)。

MathML

廃止

  • 非推奨の <mfenced> 要素を削除しました (バグ 1603773)。代わりに <mrow> および <mo> 要素を使用してください。

JavaScript

  • The yearName および relatedYear フィールドが DateTimeFormat.prototype.formatToParts() メソッドで使用可能になりました。CJK カレンダー向けに役立つフォーマットです (バグ 1591664)。

API

DOM

  • Window オブジェクトの innerWidth および innerHeight プロパティが、ときどき視覚的なビューポートに基づいて幅や高を返すことなく、常にレイアウトビューポートの幅や高さを返すようになりました。特に、以前は レスポンシブデザインモード を使用するときに視覚的なビューポートの寸法を返しており、想定とは異なる動作でした (バグ 1514429)。

WebVR

  • 非推奨の WebVR API (拡張現実 および バーチャル・リアリティ アプリケーションの両方をサポートする WebXR に取って代わられました) を動作させるために、HTTPS プロトコルを使用する 安全なコンテキストが必要になりました。これは、個人情報を含む可能性がある機微な入力ソースを使用できるためです (バグ 1381645)。

廃止

  • VideoPlaybackQuality の corruptedVideoFrames プロパティは仕様書で非推奨とされており、Firefox から削除しました (バグ 1602163)。

セキュリティ

変更なし。

WebDriver conformance (Marionette)

  • カレントページを PDF 文書として印刷する WebDriver:Print を追加しました (バグ 1604506)。
  • Webdriver:TakeScreenshot が、キャプチャーする要素が指定されていない場合に現在選択されているブラウジングコンテキストではなく、常にトップレベルのブラウジングコンテキストをキャプチャーするようになりました (バグ 1398087, バグ 1606794)。
  • Webdriver:TakeScreenshot の full 引数を使用すると、ページ全体をキャプチャーします (バグ 1571424)。

アドオン開発者向けの変更点

API の変更点

  • sidebarAction.toggle() 関数を実装しました (bug 1453355)。

マニフェストの変更点

変更なし。
