このページでは、開発者に影響する Firefox 83 の変更点をまとめています。Firefox 83 は、2020 年 11 月 17 日にリリースされました。

注記: Mozilla hacks の Firefox 83 is upon us もご覧ください。

ウェブ開発者向けの変更点一覧

開発者ツール

  • ウェブコンソール で :screenshot ヘルパーコマンド を使用するとき、--fullpage を使用して全画面のスクリーンショットを撮影する場合も --dpr 引数を無視しないようになりました (バグ 1645284)。
  • 開発者が スクロール可能なオーバーフローをデバッグする ために、インスペクター で スクロール バッジを使用できるようになりました。バッジを選択すると、オーバーフローを発生させている要素を強調して、オーバーフロー バッジをつけます (バグ 1669129)。

HTML/XML

  • <link rel=icon> で crossorigin 属性をサポートしました (バグ 1661075)。
  • displaystyle 属性 をすべての MathML 要素に実装しました (バグ 1666075)。

CSS

  • ベンダー接頭辞がつく :-moz-any() を、標準の :is() CSS 疑似クラス関数の別名にしました (バグ 1666086)。
  • CSS Conic Gradients をサポートしました (バグ 1632351) および (バグ 1175958)。

JavaScript

  • Intl[@@toStringTag] を追加しました。既定値 Intl を返します (最近 ECMA 仕様書に追加されました)。(バグ 1670053)

HTTP

変更なし。

API

変更なし。

WebDriver conformance (Marionette)

  • WebDriver:SwitchToShadowRoot の予備的なサポートを削除しました。これは WebDriver の仕様書にまだ追加されていません (バグ 1662468)。
  • WebDriver:Back および WebDriver:Forward で、削除された <iframe> によってナビゲーションが発生したときに Marionette がハングアップする不具合を修正しました (バグ 1672758)。

アドオン開発者向けの変更点

変更なし。
