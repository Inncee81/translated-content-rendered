Gecko 24 を搭載した Firefox 24 は米国時間 2013 年 9 月 17 日にリリースされました。このページでは、開発者に影響する Firefox 24 の変更点をまとめています。

Web 開発者向けの変更点一覧

CSS

  • cursor プロパティの値のうち -moz-zoom-in と -moz-zoom-out の 2 つは、接頭辞を外して zoom-in および zoom-out になりました (バグ 772153)。

HTML

  • media.webvtt.enabled 設定のもとに、<track> 要素を実装しました (バグ 833385)。

JavaScript

  • "use strict" により明示的に要求される場合を除き、Arrow functions は自動的に strict モードにならないようにしました。(バグ 852762)
  • String.prototype.repeat JS メソッドを実装しました (バグ 815431)。

DOM

  • Range() コンストラクタをサポートしました (バグ 868999)。
  • Text() コンストラクタをサポートしました (バグ 869000)。
  • Comment() コンストラクタをサポートしました (バグ 869006)。
  • DocumentFragment() コンストラクタをサポートしました (バグ 869002)。
  • FocusEvent インターフェイスを実装しました (バグ 855741)。
  • ChildNode.remove() メソッドをサポートしました (バグ 856629)。
  • <track> 要素に関するインターフェイスである HTMLTrackElement、TextTrack、TextTrackCue、TextTrackList、TextTrackCueList を、既定値が false である設定 media.webvtt.enabled のもとに実装しました (バグ 833385)。
  • Gamepad インターフェイスと Navigator.getGamepads を、既定値が false である設定 dom.gamepad.enabled のもとに実装しました (バグ 690935)。
  • デスクトップ版 Firefox に限り、HTMLCanvasElement.getContext() の値として experimental-webgl に加えて webgl をとれるようになりました (バグ 870232)。
  • HTMLMediaElement の非標準メソッド mozLoadFrom() を削除しました (バグ 877135)。

開発者ツール

  • ネットワークインスペクタで、コンテンツタイプ (CSS/画像/フォント など) による絞り込みと、絞り込み結果についてサイズや読み込み時間を確認できるようになりました。
  • 左側にある開発ツールのオプションパネルで、JavaScript を一時的に無効化/有効化できるようになりました。
  • 拡張機能の開発者は chrome レベルのスクリプトに対して、新たにブラウザコンソールを使用できるようになりました (エラーコンソールを置き換えます)。

MathML

  • 例えば <math> 要素や <mrow> 要素で、式の書字方向を制御する dir 属性が、direction CSS プロパティを使用することと同等になりました。
  • 等号 ("=") を引き延ばすことが可能になりました。
  • <menclose> 要素で、notation 属性に値 "updiagonalarrow" を追加しました。
