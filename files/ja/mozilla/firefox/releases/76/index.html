このページでは、開発者に影響する Firefox 76 の変更点をまとめています。Firefox 76 は、2020 年 5 月 5 日 にリリースされました。

付随する hacks ブログの記事「Firefox 76: Audio worklets and other tricks」もご覧ください。

ウェブ開発者向けの変更点一覧

開発者ツール

デバッガー

  • ソースリストペイン のコンテキストメニューで、ソースグループやフォルダーのブラックボックス化や解除が可能になりました (バグ 1118152)。
  • コールスタックペイン のコンテキストメニュー項目「スタックトレースをコピー」で、ファイル名だけでなく URL 全体をコピーするようになりました (バグ 1619039)。

ネットワークモニター

  • ネットワーク要求の一覧で列の境目をダブルクリックすると、境目の左側の列を内容に合わせてリサイズするようになりました (バグ 1615102)。
  • ネットワーク要求のコンテキストメニュー項目「コピー > cURL としてコピー で新たなオプション --globoff が利用可能になりました。これはコピーした URL に各括弧文字が含まれている場合に、cURL の globbing (ワイルドカードのマッチング) を抑制します (バグ 1549773)。
  • WebSocket 要求 の詳細ペインにある メッセージ タブに、制御フレームを表示する新たなフィルターである Control を追加しました。また、フィルターが選択リストにグループ分けされました (バグ 1566780)。

ウェブコンソール

  • マルチラインモード で、5 行を超えるコードスニペットを 5 行に省略して先頭に展開用の三角印 (あるいは "twistie")、末尾に省略記号 (…) をつけるようになりました。これらの領域をクリックするとコードを表示できます。もう一度クリックすると折りたたむことができます (バグ 1578212)。
  • コンソールに出力した DOM 要素の参照のコンテキストメニュー項目に「インスペクターで確認」を追加しました。これは、インスペクター の HTML ペインで要素を表示します (バグ 1612276)。

リモートデバッグ

  • 開発ツールのバージョンの違いのため、デスクトップ版の Firefox 69 以降からバージョン 68 ベースの Android 版 Firefox をデバッグすることができません。デバッグしようとするとデスクトップ版 Firefox は、この問題の説明と利用可能な次のステップをユーザーに知らせるメッセージを表示します (バグ 1625906)。詳しくは Connection to Firefox for Android 68 をご覧ください。

HTML

  • <input> 要素の min および max 属性が、値が周期的な (すなわち、ある時点で値が最小値に戻る) コントロールで min の値が max の値より大きい場合に、正しく動作するようになりました。これは、例えば日付や時刻の入力で午後 11 時から午前 2 時の範囲を指定するようなときに、特に役に立ちます (バグ 1608010)。

CSS

  • CSS4 system colors を Firefox でサポートしました (バグ 1590894)。

SVG

変更なし。

JavaScript

  • Intl.NumberFormat、Intl.DateTimeFormat、Intl.RelativeTimeFormat コンストラクターの numberingSystem、calendar オプションを、デフォルトで有効にしました (バグ 1625975)。

API

新規 API

  • Firefox で、audio worklets と AudioContext.audioWorklet をデフォルトでサポートしました。これは、メインスレッドの外部で音声をリアルタイム処理するために AudioWorkletProcessor および AudioWorkletNode インターフェイスを使用可能にします (バグ 1616725)。

DOM

  • window.open() の引数 windowFeatures の UI 部品に関する項目で、UI 部品ごとに表示・非表示を制御できなくなり、ポップアップウィンドウを開くか否かの条件になりました (バグ 1507375)。
  • location.href や <meta http-equiv="refresh"> のような方法を使用して未知のプロトコルへ移動しようとするとブロックされるようになりました (バグ 1528305、詳しくは 未知のプロトコルへのページ遷移はブロックされます をご覧ください)。
  • IntersectionObserver() コンストラクターが、root として Element オブジェクトと同様に Document オブジェクトも受け入れるようになりました (バグ 1623623)。これにより、要素間交差の境界としてウィンドウのコンテンツ領域全体を明示的に使用できます。
  • Fetch API で、要求の audioworklet destination をサポートしました。これにより、受信したデータが AudioWorklet へ送られます (バグ 1402784)。

廃止

  • Window の appinstalled イベント (および関連するハンドラープロパティの Window.onappinstalled) を完全に削除しました。これらは公開されたことがなく、また Web Manifest 仕様 から削除されました (バグ 1625384)。

HTTP

変更なし。

セキュリティ

変更なし。

WebDriver conformance (Marionette)

  • geckodriver でオートメーションやテストを行っているとき、navigator.webdriver が false を返さないようになりました (バグ 1632556)。

アドオン開発者向けの変更点

変更なし。
