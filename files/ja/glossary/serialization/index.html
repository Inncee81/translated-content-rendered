--- title: Serialization (シリアライズ) slug: Glossary/Serialization tags: - CodingScripting - Glossary - JavaScript - Serialization translation_of: Glossary/Serialization ---

オブジェクトまたはデータ構造が、ネットワークまたはストレージ（例えば、アレイバッファまたはファイルフォーマット）上の転送に適したフォーマットに変換されるプロセス。
 

たとえば JavaScript では、JSON.stringify() 関数を呼び出して、オブジェクトを JSON 文字列にシリアライズできます。

CSS 値は、CSSStyleDeclaration.getPropertyValue() 関数を呼び出すことによってシリアライズされます。
