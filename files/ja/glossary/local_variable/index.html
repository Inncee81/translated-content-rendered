--- title: Local variable (ローカル変数) slug: Glossary/Local_variable tags: - CodingScripting - Glossary - NeedsContent - 用語集 translation_of: Glossary/Local_variable ---

変数のうち、その値を指す名前がローカルスコープ内にのみあるものです。

例

var global = 5; //is a global variable

function fun(){
    var local = 10; //is a local variable
}
