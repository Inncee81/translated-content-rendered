Microsoft Edge は、グラフィカルな無料のウェブブラウザーで、 Microsoft Windows に付属しており、マイクロソフトが 2014年から開発しています。 Edge は、最初は Spartan として知られ、長い間使われた Internet Explorer を置き換えました。
