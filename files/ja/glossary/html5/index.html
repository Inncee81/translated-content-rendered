HTML5 は、HTML の最新の安定バージョンであり、ドキュメントを構築するだけのマークアップであった HTML をアプリケーション開発基盤へと昇華させました。また、その他にも HTML5 は新しい Element (要素) と JavaScript API を含んでおり、ストレージやマルチメディア、ハードウェアアクセスが強化されています。
