jQuery は JavaScript ライブラリー であり、DOM 操作を簡単にすること、AJAX 呼び出し、イベント ハンドリングに注力しています。

jQuery は $(selector).action() というフォーマットを使って要素をイベントに割り当てます。詳しく説明すると、$(selector) が jQuery を呼び出して selector 要素を選択し、  .action()というイベント API に割り当てます。

$(document).ready(function(){
  alert("Hello World!");
  $("#blackBox").hide();
});

上のコードは下のコードと同じ関数を実行します:

window.onload = function() {
  alert("Hello World!");
  document.getElementById("blackBox").style.display = "none";
};

または

window.addEventListener("load", () => {
  alert("Hello World!");
  document.getElementById("blackBox").style.display = "none";
});
