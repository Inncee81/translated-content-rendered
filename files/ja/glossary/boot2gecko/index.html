Boot2Gecko (B2G) は Firefox OS の開発コード名であり、公式の Firefox OS ブランドに追加されていないビルドを指す。（Firefox OS はプロジェクトの正式名称が決まる以前はよく Boot2Gecko と呼ばれていた）
