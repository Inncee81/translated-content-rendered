Ecma International（正式名：欧州電子計算機工業会; European Computer Manufacturers Association）は、コンピュータ・ハードウェアや通信、プログラム言語の標準規格を策定する非営利組織です。

ウェブの世界では、 JavaScript 言語の中核的規格である ECMA-262 規格（別名： ECMAScript ）を維持・管理する組織として有名です。
