URI (Uniform Resource Identifier)は、リソースを示す文字列です。もっとも一般的なものは URL であり、ウェブ上の場所を指定することで、リソースを識別します。対照的に URN は、本の ISBN のような与えられた名前空間の中で名前によってリソースを参照します。
