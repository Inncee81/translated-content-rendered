オブジェクトはデータとそのデータをやり取りするための命令が入ったデータ構造です。オブジェクトはしばしば現実世界の事物を参照する事があります。例えば、レーシングゲームでは car や map オブジェクトと言った具合です。オブジェクト指向プログラミング言語の例としては、JavaScript、Java、C++、Python、そして Ruby などが挙げられます。
