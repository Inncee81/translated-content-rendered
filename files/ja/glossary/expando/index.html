--- title: Expando slug: Glossary/Expando tags: - CodingScripting - JavaScript - Reference - expando translation_of: Glossary/Expando ---

expando プロパティは、 JavaScript によって DOM ノードに追加されたプロパティであり、 DOM オブジェクトの仕様書に含まれていないものです。

window.document.foo = 5; // foo は expando

この用語はオブジェクトの本来の目的以外に追加されたプロパティにも適用されます。例えば、配列に追加された非数値型の名前つきプロパティなどにも用います。
