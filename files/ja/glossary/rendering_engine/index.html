レンダリングエンジンは画面にテキストと画像を描画するソフトウェアです。このエンジンは構造化文書 (たいてい HTML) を描画して、所与のスタイル定義 (たいてい CSS にて与えられる) に基づいて整形します。レイアウトエンジンの例としては、 Blink, Gecko, EdgeHTML, WebKit があります。
