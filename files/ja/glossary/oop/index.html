OOP (Object-Oriented Programming、オブジェクト指向プログラミングの略) はデータを オブジェクト 内にカプセル化して、その構成部品ではなくてオブジェクト自体が操作される、プログラミングアプローチです。

JavaScript はとてもオブジェクト指向です。これは プロトタイプベースのモデル (クラスベースの反対) に従っています。
