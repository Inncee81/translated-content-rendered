IndexedDB は大きなデータ構造をブラウザーに保管して高パフォーマンスな検索用にインデックスする Web API です。SQL-ベースの RDBMS と同様に、IndexedDB はトランザクションできるデータベースシステムです。しかし、データを保管するのに固定長のカラムではなく JavaScript オブジェクトを使います。
