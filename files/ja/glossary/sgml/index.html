Standard Generalized Markup Language (SGML) は、宣言型のマークアップ言語を定義するための ISO 規格です。

ウェブにおいて、HTML 4、 XHTML、 XML は広く普及した SGML ベースの言語です。HTML は、第5版からもはや SGML を基にしておらず、独自の構文解析規則をもつ点は注意が必要です。
