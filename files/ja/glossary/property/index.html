プロパティという用語はコンテキストに応じて幾つかの意味を持ちます。以下を参照してください：

Property (プロパティ) (CSS)
CSS のプロパティは、関連付けられた値によりブラウザーがどのように要素を表示するかを定義する特性です。
Property (プロパティ) (JavaScript)
JavaScript  プロパティは、多くの場合データ構造と関連付けられた属性を記述するオブジェクトの特性です。
