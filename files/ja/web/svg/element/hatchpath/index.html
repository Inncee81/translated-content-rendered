これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

<hatchpath> は SVG の要素で、 <hatch> 要素で使用されるハッチパスを定義します。

使用場所

カテゴリー無し
許可された内容任意の数、任意の順で以下の要素を配置可能：
アニメーション要素
説明的要素
<script> 、 <style>

属性

グローバル属性

  • コア属性
  • グローバルイベント属性
  • プレゼンテーション属性
  • スタイル属性

固有の属性

  • d
  • offset

DOM インターフェイス

この要素は SVGHatchpathElement インターフェイスを実装しています。

例

SVG

<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <hatch id="hatch" hatchUnits="userSpaceOnUse" pitch="5"
        rotate="135">
      <hatchpath stroke="#a080ff" stroke-width="2"/>
    </hatch>
  </defs>

  <rect fill="url(#hatch)" stroke="black" stroke-width="2"
     x="10%" y="10%" width="80%" height="80%" />
</svg>
