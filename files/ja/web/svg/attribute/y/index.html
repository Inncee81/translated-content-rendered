この属性はユーザー座標系のy軸座標を指定します。この属性の正確な影響は各要素の座標に依存します。ほとんどの場合、この属性参照している要素の左上を原点とした長方形のx軸座標を示します（例外については個々の要素のドキュメントを参照してください）。

値の明示がなければ、0として記入されたものとして、 <filter> と <mask> 要素に関しては -10% と記入されたものと解釈します。
