CSS プロパティの解析値 は、getComputedStyle()によって返される値です。

ほとんどのプロパティでは計算値がそのまま解析値となりますが、古いプロパティ(width と height を含む)では、代わりに使用値が解析値となります。プロパティごとの詳細については、以下の仕様のリンクを参照してください。
