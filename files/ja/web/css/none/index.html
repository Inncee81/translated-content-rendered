概要

none キーワードは、多くの CSS プロパティで使用される共通の値です。デフォルト値であることが多いですが、そのプロパティの「標準値」を意味し、多くの場合は初期値となります。よく似た値として normal が有ります。

none 値が使用可能な CSS プロパティ

  • float
    要素はフロートしません。
  • text-decoration
    要素の装飾を行いません。
  • max-height / max-width
    高さ・幅を特定値で制限しません。
  • background-image
  • border-style
  • clear
  • content
  • counter-increment
  • counter-reset
  • display
  • list-style-type
  • quotes
  • text-transform
