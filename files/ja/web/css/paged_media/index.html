ページ化メディアのプロパティは、印刷物などの分離したページでコンテンツが分割されるメディアのコンテンツの表現を制御します。ページ区切り、印刷可能領域の制御、左と右のページの違い、要素内の改ページの制御などを設定することができます。広く対応されているプロパティは以下の通りです。

  • page-break-before
  • page-break-after
  • page-break-inside
  • orphans
  • widows
  • @page
