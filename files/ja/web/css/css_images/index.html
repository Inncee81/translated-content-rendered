CSS 画像 (CSS Images) は CSS のモジュールの一つで、どのような種類の画像が利用できるか (<image> 型、その中の URL、グラデーション、その他の種類の画像)、どのように大きさを変更するか、様々なレイアウトモデルで画像やその他の置換コンテンツが応答するかを定義します。
