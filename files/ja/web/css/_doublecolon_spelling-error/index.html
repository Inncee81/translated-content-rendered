これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

CSS の ::spelling-error 疑似要素は、綴りが正しくないとユーザーエージェントが判断したテキスト区間を示します。

::spelling-error {
  color: red;
}

利用できるプロパティ

::spelling-error 擬似要素では、一部の CSS プロパティのみが利用できます。

  • color
  • background-color
  • cursor
  • caret-color
  • outline およびその個別指定プロパティ
  • text-decoration および関連プロパティ
  • text-emphasis-color
  • text-shadow
