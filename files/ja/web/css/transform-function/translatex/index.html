translateX() は CSS のfunctionで、要素を二次元平面上の水平方向で再配置します。結果は <transform-function> データ型になります。

注: translateX(tx) は translate(tx, 0) または translate3d(tx, 0, 0) と等価です。

構文

/* <length-percentage> values */
transform: translateX(200px);
transform: translateX(50%);

値

<length-percentage>
変換ベクトルの横座標を表す <length> または <percentage> です。パーセント値は transform-box プロパティで定義される参照ボックスの幅からの相対値です。
ℝ2 のデカルト座標 ℝℙ2 の同次座標 ℝ3 のデカルト座標 ℝℙ3 の同次座標

ℝ2 では線形変換ではないので、デカルト座標の行列で表すことはできない。

 10t010001 10t010001 100t010000100001
[1 0 0 1 t 0]

形式文法

translateX(<length-percentage>)

例

HTML

<div>Static</div>
<div class="moved">Moved</div>
<div>Static</div>

CSS

div {
  width: 60px;
  height: 60px;
  background-color: skyblue;
}

.moved {
  transform: translateX(10px); /* translate(10px) と等価 */
  background-color: pink;
}
