この記事では、Web 開発者が適切な仕事に適切なツールをよりよく選択できるように、 CSS セレクタと XPath の違いを文書化して Web 開発者に提供しています。

XPath 機能 相当する CSS
ancestor, parent, preceding-sibling 軸 :has() セレクター
attribute 軸 属性セレクター
child 軸 子結合子
descendant 軸 子孫結合子
following-sibling 軸 一般兄弟結合子または隣接兄弟結合子
self 軸 :scope または :host セレクター
