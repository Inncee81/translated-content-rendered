XSLT/XPath リファレンス: XSLT 要素, EXSLT 関数, XPath 関数, XPath 軸

starts-with 関数は第 1 引数の文字列が第 2 引数の文字列で始まるかどうかを調べ、true または false を返します。

構文

starts-with(haystack ,needle)

引数

haystack
検索される文字列。
needle
検索する文字列。

戻り値

haystackがneedle で始まれば true。 そうでなければ false。

定義

XPath 1.0 4.2

Gecko でのサポート

サポート済み。
