round 関数は、与えられた数値に最も近い整数を返します。

構文

round(decimal )

引数

decimal
丸められる小数。

戻り値

decimalよりも小さいか、大きいか、等しい、最も近い整数。

注記

  • -0.5 は負のゼロに丸められます。 0.4 は正のゼロに丸められます。

定義

XPath 1.0 4.4
