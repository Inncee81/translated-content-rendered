XSLT/XPath リファレンス: XSLT 要素, EXSLT 関数, XPath 関数, XPath 軸

false 関数は真偽値 false を返します。

構文

false()

戻り値

真偽値 false。

注記

この関数は比較の一部として役立ちます。

<xsl:if test="boolean((1 &gt; 2) = false())">
 The expression evaluates as true
</xsl:if>

定義

XPath 1.0 4.3
