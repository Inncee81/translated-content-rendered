XSLT/XPath リファレンス: XSLT 要素, EXSLT 関数, XPath 関数, XPath 軸

function-available関数は、指定された関数が使用可能かどうかを判断し、真偽値 true または false を返します。

構文

function-available(name )

引数

name
テストする関数の名前

戻り値

真偽値の true もしくは false

定義

XPath 1.0 15

Gecko のサポート

サポート済み
