XSLT/XPath リファレンス: XSLT 要素, EXSLT 関数, XPath 関数, XPath 軸

last 関数は、式評価コンテキストのコンテキストサイズに等しい数値を返します。

構文

last()

戻り値

式評価コンテキストのコンテキストサイズに等しい整数。

注記

  • この関数は、特定のノードがノード集合内で最後のものであるかどうかを判定するために、 position() 関数をともなってよく用いられます。

定義

XPath 1.0 4.1
