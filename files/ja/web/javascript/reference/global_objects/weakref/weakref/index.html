WeakRef コンストラクターは、指定された対象オブジェクトを参照する WeakRef オブジェクトを生成します。

構文

new WeakRef(targetObject);

引数

targetObject
WeakRef が参照する対象オブジェクト (referent とも呼ばれます)。

例

新しい WeakRef オブジェクトの生成

完全な例は中心となる WeakRef ページを参照してください。

class Counter {
  constructor(element) {
    // DOM 要素への弱い参照を覚える
    this.ref = new WeakRef(element);
    this.start();
  }
}
