BigInt() コンストラクターは、 BigInt オブジェクトを生成するために使用します。

構文

BigInt(value);

引数

value
作成しようとしているオブジェクトの数値。文字列または整数にすることができます。

メモ: BigInt() は new 演算子と共には使用されません。

例

新しい BigInt の生成

BigInt(123);
// 123n
