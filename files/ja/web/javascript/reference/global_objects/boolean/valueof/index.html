valueOf() メソッドは、Boolean オブジェクトのプリミティブ値を返します。

構文

bool.valueOf()

戻り値

与えられた Boolean オブジェクトのプリミティブ値です。

説明

valueOf メソッドは、Boolean オブジェクトのプリミティブ値か真偽値データ型としての Boolean リテラルを返します。

このメソッドはたいてい JavaScript によって内部的に呼び出され、コードで明示的に呼び出されることはありません。

例

valueOf を使う

x = new Boolean();
myVar = x.valueOf()      // myVar に false が代入される
