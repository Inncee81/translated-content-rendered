XSLT/XPath リファレンス: XSLT 要素, EXSLT 関数, XPath 関数, XPath 軸

<xsl:copy-of> 要素は、select 属性が出力文書に指定するものすべての深いコピー (子孫ノードを含む) を作成します。

構文

<xsl:copy-of select=EXPRESSION />

必須属性

select
何をコピーするかを指定する XPath 式を使用します。

任意属性

なし

タイプ

命令は、テンプレート内に表示されます。

定義

XSLT, section 11.3.

Gecko のサポート

サポート済み
