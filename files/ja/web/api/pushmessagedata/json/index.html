PushMessageData インターフェースの json()メソッドは、プッシュメッセージデータを JSON オブジェクトとして抜き出します。

構文

var myJson = PushEvent.data.json();

パラメーター

なし。

戻り値

JSON オブジェクト。

例

self.addEventListener('push', function(event) {
  var jsonObj = event.data.json();

  // JSON を使って何かを実行する
});
