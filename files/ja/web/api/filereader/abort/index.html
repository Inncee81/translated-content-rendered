abort メソッドは読み取り操作を中止します。戻る時に、 readyState が DONE になります。

構文

instanceOfFileReader.abort();

例外

DOM_FILE_ABORT_ERR
読み込み操作が実行されていない間に abort が呼び出されたとき (つまり、状態が LOADING でないとき) にスローされます。
