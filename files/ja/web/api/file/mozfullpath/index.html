非標準
この機能は標準ではなく、標準化の予定もありません。公開されているウェブサイトには使用しないでください。ユーザーによっては使用できないことがあります。実装ごとに大きな差があることもあり、将来は振る舞いが変わるかもしれません。

File インターフェイスに対する特権的な拡張として、mozFullPath プロパティには、表現されたファイルの絶対パス名が含まれます。 このプロパティはブラウザコード、または古いスタイルの XPCOM ベースの Firefox 拡張機能でのみ使用できます。Web コンテンツでは使用できません。
