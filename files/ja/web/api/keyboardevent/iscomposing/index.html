KeyboardEvent.isComposing は読み取り専用プロパティで、 Boolean 値でイベントが変換セッションの途中、すなわち compositionstart の後かつ compositionend の前に発行されたことを示します。

構文

var bool = event.isComposing;

例

var kbdEvent = new KeyboardEvent("syntheticKey", false);
console.log(kbdEvent.isComposing); // false を返す
