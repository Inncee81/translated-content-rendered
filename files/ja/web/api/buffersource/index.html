BufferSource は、 ArrayBuffer または、 ArrayBufferView を提供する TypedArray のどちらかのオブジェクトを表現するために用いられる typedef です。

これは仕様を簡潔にするためのヘルパー型です。インターフェースではなく、それを実装したオブジェクトもありません。
