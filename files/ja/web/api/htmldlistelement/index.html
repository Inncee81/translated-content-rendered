HTMLDListElement インタフェースは <dl> を操作する為の特別なプロパティを持つインタフェースです。 <dl> 要素はこのインタフェースの継承元である HTMLElement からも操作可能です。

プロパティ

HTMLElement インタフェースから継承するプロパティの他に、以下の固有プロパティを持ちます。

名称 型 説明
compact Boolean リスト項目間の（視覚上の）間隔を詰める事を示す

メソッド

HTMLElement インタフェースから継承するもの以外のメソッドは持ちません。
