WebVR API の VRFieldOfView インターフェイスは，中心点からの視野を示す4つの角度で定義される視界を表現します．

プロパティ

VRFieldOfView.upDegrees 読取専用
視界の上方向への拡張量を示す角度．
VRFieldOfView.rightDegrees 読取専用
視界の右方向への拡張量を示す角度．
VRFieldOfView.downDegrees 読取専用
視界の下方向への拡張量を示す角度．
VRFieldOfView.leftDegrees 読取専用
視界の左方向への拡張量を示す角度．
