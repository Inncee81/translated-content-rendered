Console の dir() メソッドは、指定された JavaScript オブジェクトのプロパティの対話的なリストを表示します。出力は折り畳み式の階層的なリストで表示され、子オブジェクトの中身を見ることができます。

言い換えれば console.dir() は指定された JavaScript オブジェクトのプロパティをすべてコンソール上で見る方法であり、開発者は簡単にオブジェクトのプロパティを得ることができます。

註: この機能は Web Workers 内で利用可能です。

構文

console.dir(object);

引数

object
プロパティを出力する JavaScript オブジェクトです。
