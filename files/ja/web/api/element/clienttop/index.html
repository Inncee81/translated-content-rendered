概要

要素の上ボーダーの幅のピクセル数です。上マージン、及び、上パディングは含みません。clientTop は読込専用です。

Gecko ベースアプリケーションは、 Gecko 1.9 (Firefox 3、バグ 111207 で実装) から clientTop をサポートします。このプロパティは、Firefox 2 以前ではサポートされません。

構文

top = element.clientTop;
