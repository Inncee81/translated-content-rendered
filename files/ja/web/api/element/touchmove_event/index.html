touchmove イベントは、タッチ面上で1つ以上のタッチ点が移動したときに発生します。

バブリング あり
キャンセル 可
インターフェイス TouchEvent
イベントハンドラープロパティ ontouchmove

例

これらのイベントのコード例は、専用のページ「タッチイベント」にあります。
