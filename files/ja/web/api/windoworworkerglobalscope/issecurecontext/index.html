WindowOrWorkerGlobalScope インターフェイスの読み取り専用 isSecureContext プロパティは、現在のコンテキストが安全（true）かそうではない（false）を示すブール値を返します。

構文

var isItSecure = self.isSecureContext; // or just isSecureContext

値

Boolean.
