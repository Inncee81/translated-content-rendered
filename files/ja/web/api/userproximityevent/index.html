これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

UserProximityEvent は、端末の近接センサーによって、端末の近くに物体が存在するかどうかを示します。

プロパティ

UserProximityEvent.near
端末が近くの物体を検知したかどうかを示す真偽値を返す
