WebSocket.onclose プロパティは、WebSocket コネクションの readyState が CLOSED に変わったときに呼ばれる EventHandler です。CloseEvent と一緒に呼び出されます。

構文

aWebSocket.onclose = function(event) {
  console.log("WebSocket is closed now.");
};

値

EventListener.
