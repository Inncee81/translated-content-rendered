WebSocket.protocol は、サーバーが選択したサブプロトコル名を返す読み取り専用のプロパティです。これは WebSocket オブジェクトが作成されるときに、引数の protocols で指定された文字列のいづれかになりますが、もし接続が確立されていない場合は、空の文字列となります。

構文

var protocol = aWebSocket.protocol;

値

DOMString
