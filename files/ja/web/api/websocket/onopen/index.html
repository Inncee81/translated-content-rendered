WebSocket.onopen プロパティはイベントハンドラー (EventHandler) で、 WebSocket のコネクションの readyState が 1 に変化したときに呼び出されます。これは、コネクションでデータを送信したり受信したりする準備ができたことを示します。これは Event を伴って呼び出されます。

構文

aWebSocket.onopen = function(event) {
  console.log("WebSocket is open now.");
};

値

EventListener です。
