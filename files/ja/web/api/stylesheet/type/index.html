概要

そのスタイルシートの type 属性の値を取得 / 設定します。

構文

string = stylesheet.type;

例

var ss = document.styleSheets[0]; // 文書中の最初のスタイルシートを参照

ss.type = "text/css"; // 設定
alert( ss.type ); // 取得
