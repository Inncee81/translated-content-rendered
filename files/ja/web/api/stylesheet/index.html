StyleSheetインターフェースを実装するオブジェクトは、単一のスタイルシートを表します。CCSスタイルシートは、より特殊化されたCSSStyleSheetインターフェースを実装します。

プロパティ

disabled
このプロパティは、現在のスタイルシートが適用されているかどうかを示します。
href 読取専用
スタイルシートの位置を返します。
media 読取専用
Specifies the intended destination medium for style information.
ownerNode 読取専用
このスタイルシートをドキュメントに関連付けるノードを返します。
parentStyleSheet 読取専用
存在する場合、これを含む StyleSheet を返します; ない場合は null を返します。
title 読取専用
現在のスタイルシートへのアドバイザリタイトルを表す DOMString を返します。
type 読取専用
このスタイルシートのスタイルシート言語を表す DOMString を返します。
