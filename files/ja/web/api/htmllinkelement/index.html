プロパティ

親、HTMLElement、および LinkStyle からプロパティを継承します。

HTMLLinkElement.as
HTML リンクによって読み込まれるコンテンツのタイプを表す DOMString。
HTMLLinkElement.crossOrigin
このリンク要素のCORS設定に対応する DOMString。詳細については、CORS設定属性を参照してください。
HTMLLinkElement.disabled
リンクが無効であるかどうかを表す Boolean。現在、スタイルシートリンクでのみ使用されています。
HTMLLinkElement.href
対象リソースの URI を表す DOMString。
HTMLLinkElement.hreflang
リンク先リソースの言語コードを表す DOMString。
HTMLLinkElement.media
リソースに適用される（ 1 つ以上の）メディアフォーマットのリストを表す DOMString。
HTMLLinkElement.referrerPolicy
使用するリファラーを示す referrerpolicy HTML 属性を反映する DOMString。
HTMLLinkElement.rel
リンク元文書からリンク先リソースに対する関連性を表す DOMString。
HTMLLinkElement.relList 読取専用
rel HTML属性をトークンのリストとして反映する DOMTokenList。
HTMLLinkElement.sizes 読取専用
トークンのリストとして sizes HTML属性を反映する DOMSettableTokenList。
LinkStyle.sheet 読取専用
指定された要素に関連付けられた StyleSheet オブジェクトを返します。存在しない場合は null を返します。
HTMLLinkElement.type
リンク先リソースの MIME タイプを表す DOMString。

廃止されたプロパティ

HTMLLinkElement.charset
対象リソースの文字エンコーディングの取得 / 設定
HTMLLinkElement.rev
リンク先リソースからリンク元文書に対する 逆相の 関係性の取得 / 設定
Note: Currently the W3C HTML 5.2 spec states that rev is no longer obsolete, whereas the WHATWG living standard still has it labeled obsolete. Until this discrepancy is resolved, you should still assume it is obsolete.
HTMLLinkElement.target
リソースを適用するターゲットフレームの名前の取得 / 設定

メソッド

固有のメソッドは持ちません。 HTMLElement と LinkStyle のメソッドを継承します。
