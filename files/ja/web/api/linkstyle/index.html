LinkStyle インターフェイスで、ノードに関連付けられた CSS スタイルシートにアクセスできます。

LinkStyle は生のインターフェイスであり、この型のオブジェクトは生成できません。これは HTMLLinkElement および HTMLStyleElement オブジェクトによって実装されています。

プロパティ

継承しているプロパティはありません。

LinkStyle.sheet 読取専用
指定した要素に関連付けられている StyleSheet オブジェクトを返します。何も関連付けられていない場合は null を返します。

メソッド

このインターフェイスはメソッドを実装していません。
