プロパティ

また、親である Event からプロパティを継承します。

ErrorEvent.message 読取専用
関数を説明するヒューマンリーダブルなエラーメッセージを含む DOMString です。
ErrorEvent.filename 読取専用
エラーが発生したスクリプトファイルの名前を含む DOMString です。
ErrorEvent.lineno 読取専用
エラーが発生したスクリプトファイルの行番号を含む integer です。
ErrorEvent.colno 読取専用
エラーが発生したスクリプトファイルの列番号を含む integer です。
ErrorEvent.error 読取専用
イベントに関係する JavaScript Object です。

コンストラクタ

ErrorEvent()
指定されたパラメータで ErrorEvent イベントを作成します。

関数

親である Event から関数を継承します。
