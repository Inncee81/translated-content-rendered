Performance インターフェイスの timeOrigin 読み取り専用プロパティは、パフォーマンス測定の開始時刻のハイレゾリューションタイムスタンプを返します。

註: この機能は Web Workers 内で利用可能です。

構文

var timeOrigin = performance.timeOrigin

値

ハイレゾリューションのタイムスタンプ

仕様

仕様書 ステータス
High Resolution Time Level 2
