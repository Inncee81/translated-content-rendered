概要

hasChildNodes は指定したノードが子ノードを持つか否かを示す真偽値を返します。

構文

node.hasChildNodes()

例

"foo" を id に持つ要素の子ノードの存在を確認し、存在する場合は最初の子ノードを削除する例を以下に示します。

var foo = document.getElementById("foo");

if ( foo.hasChildNodes() ) {
  foo.removeChild( foo.childNodes[0] );
}
