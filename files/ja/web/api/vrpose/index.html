これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

WebVR API の VRPose インターフェイスは，特定のタイムスタンプにおけるVRセンサーの状態（向き，位置，速度，加速度の情報を含みます）を示します．

このインターフェイスには，VRDisplay.getPose() と VRDisplay.getImmediatePose() メソッドを通じてアクセスします．

プロパティ

VRPose.timeStamp 読取専用
システムの現在のタイムスタンプを返します — これは単調増加する値で，位置データが更新されたか，どんな順番で更新が発生しているかを決めるのに使えます．
VRPose.position 読取専用
現在の VRPose.timestamp における VRDisplay の位置を3Dベクトルとして返します．
VRPose.linearVelocity 読取専用
現在の VRPose.timestamp における VRDisplay の（線形の）速度を返します．単位はメートル/秒です．
VRPose.linearAcceleration 読取専用
現在の VRPose.timestamp における VRDisplay の（線形の）加速度を返します．単位はメートル/秒/秒です．
VRPose.orientation 読取専用
現在の VRPose.timestamp におけるセンサーの向きをクォータニオンとして返します．
VRPose.angularVelocity 読取専用
現在の VRPose.timestamp における VRDisplay の角速度を返します．単位はラジアン/秒です．
VRPose.angularAcceleration 読取専用
現在の VRPose.timestamp における VRDisplay の角加速度を返します．単位はラジアン/秒/秒です．
