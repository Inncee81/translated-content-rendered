標準的な JavaScript の関数（String や Array、Object、 JSON など）に加えて、DOMから worker に利用できる様々な関数があります。この記事ではそれらの機能のリストを提供します。

Worker は、現在の window とは異なるグローバルコンテキスト、 DedicatedWorkerGlobalScope で実行されます。既定では Window のメソッドとプロパティは使用できませんが、Window に似ている DedicatedWorkerGlobalScope は WindowTimers と WindowBase64 を実装しています。
