TextTrackCues は、TextTrack に一定期間表示されるテキストの文字列を表します。 これには、キューが表示される開始時間と終了時間が含まれます。 TextTrackCue を直接使用することはできません。 その代わりに、派生型の1つ（例えば VTTCue）を使用する必要があります。

プロパティ

このインターフェイスは EventTarget からもプロパティを継承しています。

TextTrackCue.track 読取専用
このキューが属する TextTrack。 いずれにも属さない場合は null。
TextTrackCue.id
キューを識別する DOMString。
TextTrackCue.startTime
キューが表示され始める動画時間を表す double 型。
TextTrackCue.endTime
キューが表示されなくなるまでの動画時間を表す double 型。
TextTrackCue.pauseOnExit
このキューが表示されなくなったときに動画を一時停止するかどうか boolean 型。

イベントハンドラ

TextTrackCue.onenter
enter イベントのイベントハンドラ。
TextTrackCue.onexit
exit イベントのイベントハンドラ。
