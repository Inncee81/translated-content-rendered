ServiceWorkerRegistration インターフェイスの sync プロパティは、バックグラウンド同期プロセスを管理する SyncManager インターフェイスへの参照を返します。

構文

var syncManager = serviceWorkerRegistration.sync;

値

SyncManager オブジェクト。
