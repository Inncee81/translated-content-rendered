InputEvent インターフェイスは、編集可能なコンテンツが変更されたことを通知するイベントを表します。

コンストラクタ

InputEvent()
InputEvent オブジェクトを作成します。

プロパティ

このインターフェイスは、親インターフェイスである UIEvent、Event のプロパティを継承します。

InputEvent.data 読取専用
挿入された文字を DOMString で返します。変更により挿入されたテキストがない (例えば文字を削除する) 場合は、空文字列になるでしょう。
InputEvent.isComposing読取専用
イベントが compositionstart の後かつ compositionend の前に発生したかを示す Boolean 値を返します。

メソッド

このインターフェイスではメソッドが定義されていませんが、親インターフェイスである UIEvent、Event のメソッドを継承します。
