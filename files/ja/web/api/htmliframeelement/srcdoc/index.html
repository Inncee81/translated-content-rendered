HTMLIFrameElement の srcdoc プロパティは、ページの内容を指定します。

例

var iframe = document.createElement("iframe");
iframe.srcdoc = `<!DOCTYPE html><p>Hello World!</p>`;
document.body.appendChild(iframe);
