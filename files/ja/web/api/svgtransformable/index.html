SVG transformable interface

インターフェイス SVGTransformable には、属性 transform. を持つ全ての要素に適用されるプロパティとメソッドが含まれています。

Methods 無し
Properties
  • 読み取り専用 SVGAnimatedTransformList transform
Normative document SVG 1.1 (2nd Edition)

プロパティ

名前 型 説明

SVGTransformable インターフェイスは、特定のメソッドを提供しません。
