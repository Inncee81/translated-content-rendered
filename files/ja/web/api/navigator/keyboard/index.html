keyboard は Navigator インターフェイスの読み取り専用プロパティで、物理キーボードのキーボードレイアウトマップを受け取ったり、キー押下のキャプチャの切り替えたりするための Keyboard オブジェクトを返します。

構文

var keyboard = navigator.keyboard

値

Keyboard オブジェクト。
