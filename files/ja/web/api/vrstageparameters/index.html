WebVR API の VRStageParameters インターフェイスは，ルームスケール体験に対応しているデバイス用にステージエリアを記述する値を表現しています．

このインターフェイスには VRDisplay.stageParameters プロパティを通じてアクセスできます．

プロパティ

VRStageParameters.sittingToStandingTransform 読取専用
VRDisplay.getPose()/VRDisplay.getImmediatePose() で返される着席(sitting-space)位置を立位(standing-space)位置にするための変換行列を含んでいます．
VRStageParameters.sizeX 読取専用
プレイエリア境界の幅を返します．単位はメートルです．
VRStageParameters.sizeY 読取専用
プレイエリア境界の奥行きを返します．単位はメートルです．
