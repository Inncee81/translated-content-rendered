属性

親であるHTMLElementより属性を継承しています。

HTMLBRElement.clear
は{domxref("DOMString")}}であり，回り込みオブジェクトの周囲の文章を回り込ませるよう指示します。

メソッド

固有のメソッドなし。親であるHTMLElementよりメソッドを継承しています。
