これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

ClipboardEvent インターフェイスは、 クリップボードの変更に関連する情報を提供するイベントを表します。すなわち、cut と copy、paste イベントです。

プロパティ

親 Event からのプロパティも継承します。

ClipboardEvent.clipboardData 読取専用
ユーザーが開始した cut または copy、paste 操作によって影響を受けたデータとその MIME タイプを含む DataTransfer オブジェクト。

コンストラクタ

ClipboardEvent()
指定したパラメータで ClipboardEvent イベントを生成する。

メソッド

固有のメソッドはありません。親 Event からのメソッドを継承します。
