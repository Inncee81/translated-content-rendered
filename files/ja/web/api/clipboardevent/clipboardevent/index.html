これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

ClipboardEvent() コンストラクタは、新たに生成された ClipboardEvent を返します。これは、cut や copy、paste イベントで変更されたクリップボード関連の情報を提供するイベントを表します。

構文

clipboardEvent = new ClipboardEvent(type, {dataType: aMimeType, data: someData});

引数

The ClipboardEvent() コンストラクタは、Event() からの引数も継承します。

type
ClipboardEvent の種類の名前を表す DOMString。大文字小文字を区別し、次のいずれかの値を指定できます: 'copy' または 'cut'、'paste'。
dataType
data 引数に含まれるデータの MIME タイプを含む DOMString。
data
clipboard イベントに関連したデータを含む DOMString。
