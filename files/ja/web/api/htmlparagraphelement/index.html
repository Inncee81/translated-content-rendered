HTMLParagraphElement インタフェースは <p> 要素を操作する為のインタフェースです。p 要素の操作は element オブジェクトインタフェースからも可能です。HTML5 に於いては、このインタフェースは HTMLElement を継承元とするインタフェースとなっています。

プロパティ

HTMLElement より継承するプロパティの他に以下のプロパティが定義されていましたが、HTML5 で廃止されています。

名称 型 説明
align DOMString 前後のコンテンツからの相対的な配置位置 ("left", "right", "justify", "center") を定義する為の列挙属性

メソッド

固有のメソッドは持っていません。HTMLElement から継承するメソッドを利用可能です。
