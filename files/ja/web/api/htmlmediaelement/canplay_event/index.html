canplay イベントは、ユーザーエージェントがメディアを再生できるようになったものの、追加のバッファリングのために停止することなくメディアの最後まで再生するには充分なデータが読み込まれていないとみられる場合に発生します。

バブリング なし
キャンセル可能 なし
インターフェイス Event
対象 Element
既定のアクション なし
イベントハンドラーのプロパティ GlobalEventHandlers.oncanplay
仕様書 HTML5 media

プロパティ

プロパティ 種類 説明
target 読取専用 EventTarget イベントのターゲット (DOM ツリー内での最上位のターゲット)
type 読取専用 DOMString イベントの型
bubbles 読取専用 Boolean イベントがバブリングするかどうか
cancelable 読取専用 Boolean イベントがキャンセル可能であるかどうか
