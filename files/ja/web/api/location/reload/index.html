Location.reload() メソッドは、再読み込みボタンのように現在の URL を再読み込みします。

再読み込みがブロックされ、 SECURITY_ERROR DOMException が発生する可能性があります。これは、Location.reload() を呼び出すスクリプトのオリジンが、 Location オブジェクトを保持するページのオリジンと異なる場合に発生します。詳細については、同一オリジンポリシーを参照してください。

構文

location.reload();
