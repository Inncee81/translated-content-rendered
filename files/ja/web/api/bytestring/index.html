ByteString は UTF-8 の文字列であり、一連のすべての可能性があるバイトの並びです。 ByteString は一般に JavaScript に返されるときに String に対応します。一般に、 HTTP などのバイト列と文字列を交換できるようなプロトコルにのみ使用されます。
