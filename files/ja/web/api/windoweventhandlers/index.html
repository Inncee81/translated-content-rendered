WindowEventHandlers ミックスインは、Window、HTMLBodyElement、HTMLFrameSetElement などのいくつかのインターフェイスに共通のイベントハンドラを記述します。 これらの各インターフェイスは、追加の固有のイベントハンドラを実装できます。

注: WindowEventHandlers はインターフェースではなく、ミックスインです。 実際には、WindowEventHandlers 型のオブジェクトを作成することはできません。

プロパティ

onXYZ という形式のイベントプロパティは、WindowEventHandlers で定義され、Window および Web Workers の WorkerGlobalScope によって実装されます。

unhandledrejection イベントが発生したときに呼び出されるコードを表す EventHandler です。 Promise は拒否されたが拒否が処理されなかったことを示します。
