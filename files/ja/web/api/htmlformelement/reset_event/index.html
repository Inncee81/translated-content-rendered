reset イベントは <form> がリセットされたときに発行されます。

バブリング あり (ただし、バブリングしない単純なイベントとして指定されている)
キャンセル 可
インターフェイス Event
イベントハンドラープロパティ GlobalEventHandlers.onreset

例

この例は EventTarget.addEventListener() を使用してフォームがリセットされるのを、実行されたときに現在の Event.timeStamp をログ出力します。

HTML

<form id="form">
  <label>Test field: <input type="text"></label>
  <br><br>
  <button type="reset">Reset form</button>
</form>
<p id="log"></p>

JavaScript

function logReset(event) {
  log.textContent = `Form reset! Time stamp: ${event.timeStamp}`;
}

const form = document.getElementById('form');
const log = document.getElementById('log');
form.addEventListener('reset', logReset);
