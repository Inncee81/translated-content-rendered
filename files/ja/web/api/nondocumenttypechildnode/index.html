NonDocumentTypeChildNode インターフェイスは親を持つ Node オブジェクトに固有な、DocumentType に適していないメソッドを含みます。

NonDocumentTypeChildNode は純粋なインターフェースで、この型のオブジェクトを作ることはできません。 Element、CharacterData オブジェクトで実装されています。

プロパティ

継承されたプロパティはありません。

NonDocumentTypeChildNode.previousElementSibling 読取専用
このノードの親ノードの子リスト内にある、このノードの直前にある Element を返します。リスト内でこのノードの前に Element がない場合は null を返します。
NonDocumentTypeChildNode.nextElementSibling 読取専用
このノードの親ノードの子リスト内にある、このノードの直後にある Element を返します。リスト内でこのノードの後に Element がない場合は null を返します。

メソッド

継承、および実装されたメソッドはありません。
