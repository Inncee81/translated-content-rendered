概要

ウィンドウで利用可能な水平空間の大きさをピクセル単位で返します。

構文

iAvail = window.screen.availWidth

例

var screenAvileWidth = window.screen.availWidth;

console.log(screenAvileWidth);

仕様

DOM Level 0。どの標準にも属しません。
