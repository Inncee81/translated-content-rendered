SourceBuffer インターフェイスの appendStream() メソッドは、ReadableStream オブジェクトから SourceBuffer にメディアセグメントデータを追加します。

構文

sourceBuffer.appendStream(stream, maxSize);

パラメーター

stream
SourceBuffer に追加するメディアセグメントデータのソースである ReadableStream。
maxSize
この操作で追加できる最大バイト数を示す unsigned long 値。

戻り値

undefined。

例外

なし。

例

未定
