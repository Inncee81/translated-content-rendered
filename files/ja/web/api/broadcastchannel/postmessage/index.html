BroadcastChannel.postMessage() は、任意の種類の Object のメッセージを、同じオリジンを持つ任意の閲覧コンテキスト内の各リスナーに送信します。 メッセージは、チャネルに結び付けられた各 BroadcastChannel を対象とした message イベントとして送信します。

註: この機能は Web Workers 内で利用可能です。

構文

var str = channel.postMessage(object);
