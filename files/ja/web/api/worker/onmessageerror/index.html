Worker インターフェイスの onmessageerror イベントハンドラは EventListener であり、 messageerror 型の MessageEvent が Worker インスタンスで発生する度に呼び出されます。 — つまり  deserialized できないメッセージを受け取ったときです。

註: この機能は Web Workers 内で利用可能です。

構文

Worker.onmessageerror = function() { ... };
