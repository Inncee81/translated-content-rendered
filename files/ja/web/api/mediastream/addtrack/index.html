MediaStream.addTrack()メソッドはストリームへ新しいトラックを追加するメソッドです。パラメータとして渡されるこのトラックの型はMediaStreamTrackです。

If the specified track is already in the stream's track set, this method has no effect.

文法

stream.addTrack(track);

パラメータ

track
このストリームへ追加するMediaStreamTrackです。

戻り値

undefined
