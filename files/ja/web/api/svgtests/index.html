SVGTests インターフェイスは条件的な属性処理の反映に使われて、その属性をサポートする要素のための他のインターフェイスと混ざります。

プロパティ

SVGTests.requiredFeatures 読取専用
SVGStringList で、所与の要素の requiredFeatures 属性に対応する。
SVGTests.requiredExtensions 読取専用
SVGStringList で、所与の要素の requiredExtensions 属性に対応する。
SVGTests.systemLanguage 読取専用
SVGStringList で、所与の属性の systemLanguage 属性に対応する。

メソッド

SVGTests.hasExtension() 読取専用
ブラウザーが、URIで指定される拡張子をサポートする場合 true。
