MediaRecorder.stream 読み取り専用プロパティは、MediaRecorder を作成したときに MediaRecorder() コンストラクタに渡したストリームを返します。

構文

var stream = MediaRecorder.stream

値

最初に MediaRecorder を作成したとき MediaRecorde() コンストラクタに渡した MediaStream。
