AudioContext() コンストラクタは新しい AudioContext オブジェクトを作成します。このオブジェクトはオーディオモジュールが相互に接続された音声処理のグラフを表現しています。このグラフ中で用いられるオーディオモジュールは AudioNode として表現されます。

記法

var audioContext = new AudioContext(options)

引数

なし。
