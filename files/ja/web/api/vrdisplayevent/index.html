WebVR API の VRDisplayEvent インターフェイスは、 WebVR 関連イベントのイベントオブジェクトを表します (WebVR ウインドウ拡張機能のリストを参照)。

コンストラクタ

VRDisplayEvent.VRDisplayEvent()
VRDisplayEvent オブジェクトのインスタンスを生成します。

プロパティ

VRDisplayEvent は、親オブジェクト Event からプロパティも継承します。

VRDisplayEvent.display 読取専用
このイベントに関連付けられた VRDisplay 。
VRDisplayEvent.reason 読取専用
イベントが発生したヒューマンリーダブルな理由。

例

window.addEventListener('vrdisplaypresentchange', function(e) {
  console.log('Display ' + e.display.displayId + ' presentation has changed. Reason given: ' + e.reason + '.');
})
