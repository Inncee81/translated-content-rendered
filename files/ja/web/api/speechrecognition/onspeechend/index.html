SpeechRecognition インターフェイスの onspeechend プロパティは、認識できる音声が検出し終えた時に (speechend イベントが発生した時)、実行されるイベントハンドラーになります。

構文

mySpeechRecognition.onspeechend = function() { ... };

例

recognition.onspeechend = function() {
  console.log('音声の検出が終わりました。');
}
