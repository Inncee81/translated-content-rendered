PointerEvent インターフェイスの getCoalescedEvents() メソッドは、送出された pointermove イベントに合体したすべての PointerEvent インスタンスのシーケンスを返します。

構文

var pointerEvents[] = PointerEvent.getCoalescedEvents()

パラメーター

なし。

Returns

PointerEvent インスタンスのシーケンス。
