これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

Window インターフェースの ongamepaddisconnected プロパティはゲームパッドが切断した時 (gamepaddisconnected イベントが発生したとき) に実行されるイベントハンドラーを表します。

イベントのオブジェクトの型は GamepadEvent です。

構文

window.ongamepaddisconnected = function() { ... };

例

window.ongamepaddisconnected = function() {
  // A gamepad has been disconnected
};
