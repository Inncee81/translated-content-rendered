Window  インターフェイスの ongamepadconnected プロパティは、ゲームパッドが接続されたとき (gamepadconnectedイベントが発生したとき) に実行されるイベントハンドラを表します。
 

イベントのオブジェクトの型は GamepadEvent です。

構文

window.ongamepadconnected = function() { ... };

例

window.ongamepadconnected = function(event) {
  // All buttons and axes values can be accessed through
  event.gamepad;
};
