Request インターフェイスの integrity 読み取り専用プロパティには、リクエストの サブリソース完全性 値が含まれています。

構文

var myIntegrity = request.integrity;

値

リクエストの サブリソース完全性 値 (例, sha256-BpfBw7ivV8q2jLiT13fxDYAe2tJllusRSZ273h2nFSE=)。

完全性が指定されていない場合、 '' を返します。

例

次のスニペットでは Request.Request() コンストラクタを使用して新しい request を作成し ( script と同じディレクトリにある画像ファイルの場合)、 request の integrity 値を変数に保存します:

var myRequest = new Request('flowers.jpg');
var myIntegrity = myRequest.integrity;
