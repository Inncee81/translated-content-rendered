MediaStreamTrack.onunmuteイベントハンドラはunmuteイベントを受け取った時に呼び出されるプロパティです。このイベントは、トラックが再びデータを送信できるようになった時に発生します。

文法

track.onunmute = function;

値

  • functionはユーザーが定義した関数の名前で末尾の()やパラメータを省略したもの、または、function(event) {...}の形式で表現された匿名関です。イベントハンドラは常に 、Event型のイベントを含む1つのパラメータを持ちます。

例

dc.onunmute = function(ev) { alert("unmute event detected!"); };
