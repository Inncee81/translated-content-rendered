locale は CompositionEvent インターフェイスの読み取り専用のプロパティで、現在の入力メソッドのロケール (例えば、結合操作が IME に関連付けられているならばキーボードレイアウトのロケール) を返します。

構文

myLocale = CompositionEvent.locale

値

DOMString で、現在の入力メソッドのロケールを表します。

仕様書 状態 備考
Document Object Model (DOM) Level 3 Events Specification 廃止された すでに仕様書にはありませんが、まだ対応されています。
