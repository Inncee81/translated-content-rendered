HTMLShadowElement インターフェースは、Shadow DOM で使用される <shadow> HTML 要素を表します。

プロパティ

このインターフェースは、HTMLElement のプロパティを継承しています。

メソッド

このインターフェースは、HTMLElement のメソッドを継承しています。

HTMLShadowElement.getDistributedNodes()
<shadow> 要素に関連した distributed nodes の静的な NodeList を返す。
