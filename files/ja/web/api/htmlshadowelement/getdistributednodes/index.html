HTMLShadowElement.getDistributedNodes() メソッドは、<shadow> 要素に関連した distributed nodes の静的な NodeList を返します。

構文

var nodeList = object.getDistributedNodes()

例

// 分散ノードを取得
var nodes = myShadowObject.getDistributedNodes();
