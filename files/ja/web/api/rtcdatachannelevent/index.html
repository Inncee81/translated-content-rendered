これは実験的な機能です。本番で使用する前にブラウザー実装状況をチェックしてください。

RTCDataChannelEvent インタフェースはRTCPeerConnectionへRTCDataChannelをアタッチしている時に発生するイベントを表現します。イベントタイプは、datachannelです。

プロパティ

RTCDataChannelEvent は Eventであり、このイベントは Eventのプロパティも実装します。

RTCDataChannelEvent.channel 読取専用
このイベントに関連付けられたデータチャンネルを含むRTCDataChannelを含みます。

コンストラクタ

RTCDataChannelEvent()
新しいRTCDataChannelEventを返します。このコンストラクタは2つの引数を持ち、最初の引数は、イベントタイプを表す DOMString で、2つ目の引数はRTCDataChannelを含むディクショナリです。

メソッド

RTCDataChannelEvent は Eventであり、このイベントは Eventのプロパティも実装します。 具体的な RTCDataChannelEvent メソッドはありません。

例

pc.ondatachannel = function( ev ) {
                      alert("The data channel " +
                            ev.channel.label +
                            " has been added to this connection.");
                   }
