以下のいずれかの方法で、ブレークポイントを設置できます:

  • ソースペイン で、停止したい行の行番号をクリックします。
  • ソースペイン で、停止したい行でコンテキストメニューを開いて [ブレークポイントを追加] を選択します。
  • ソースペイン で、ブレークポイントを設置したい行をハイライトして Ctrl+B (Windows/Linux) または Command+B (Mac OS X) を押下します。

それぞれのブレークポイントは、デバッガー内の 2 か所に表示します:

  • ブレークポイント一覧 で、ブレークポイントのファイル名や行番号を表示
  • ソースペイン で行に青色の矢印を表示、または 条件付き ブレークポイントであれば橙色の矢印を表示
