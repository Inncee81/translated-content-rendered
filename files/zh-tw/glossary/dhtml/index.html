DHTML 是動態 (dynamic) HTML 的縮短形式，且通常用於指那些未使用 flash 或 java 等外掛的交互式網頁後面的程式碼。The term aggregates the combined functionality 允許網頁開法者使用 HTML ， CSS ， Document Object Model ，以及 JavaScript。

DHTML 與 Ajax 的不同之處在於，在載入階段它仍然基於 HTML, CSS 和 JS 等程式碼來定義。相反的，基於 Ajax 的頁面則使用 XMLHttpRequest 介面來動態載入更多的資料。
