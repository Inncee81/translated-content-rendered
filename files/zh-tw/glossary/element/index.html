--- title: 元素 slug: Glossary/Element tags: - HTML - 術語 translation_of: Glossary/Element ---

元素（element）是網頁結構的一部分。在 XML 與 HTML 中，元素可能包含資料清單、文字、圖片，或是也可能不包含東西。典型的元素會包含一個有些許屬性（attributes）的開始標籤（opening tag）、被包覆住的文字內容，以及一個結束標籤（closing tag）。

元素與標籤是不同的概念。在原始碼中，標籤用來標注元素開始與結束，而元素其實是 DOM 的一部分。DOM，指文件物件模型（document object model），會由 browser 將 DOM 呈現為網頁。
