IndexedDB 是一個用來在瀏覽器內儲存大型數據結構，並對其建立索引以高效查詢的 Web API。類似基於 SQL 的 關連式資料庫管理系統（RDBMS）， IndexedDB 是一套交易式資料庫系統。然而，它使用 JavaScript 物件而非固定欄位的資料表來記錄資料。
