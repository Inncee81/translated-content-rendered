<!DOCTYPE>會告知你的瀏覽器這個文件是用哪個版本的 HTML （或 XML）撰寫。Doctype 是一種宣告，而非 tag。你也可以把它想作「document type declaration」（文件類型宣告），或是縮寫的 「DTD」。
