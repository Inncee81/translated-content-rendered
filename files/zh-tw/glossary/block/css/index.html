--- title: 區塊 (CSS) slug: Glossary/Block/CSS tags: - Block - CSS - HTML - display - 術語表 translation_of: Glossary/Block/CSS ---

在網頁上一個區塊是一個 HTML元素(element)它會出現一個新的行，例：在前面的元素之下和後面的元素之前(通常稱爲塊級元素)。擧個例子, <p> 標簽就是一個默認的塊級元素，而 <a> 標簽則是一個内聯元素— 你可以在 HTML 源代碼放一些相鄰的鏈接，他們將會并列渲染在相同的行。

使用 display 屬性你可以隨意改變大多數元素的顯示方式，不論是内聯還是區塊(其中還有很多其他的選項); blocks 也同樣會受到定位方案和 position 屬性的影響。
