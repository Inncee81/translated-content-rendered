--- title: 陣列 slug: Glossary/array tags: - 術語表 - 陣列 translation_of: Glossary/array ---

陣列是個有序資料的集合（按程式語言不同，資料型態可以是原始型別或物件）。陣列是用來把複數的值存進一個變數裡。與只能存一個值的變數，呈現強烈的對比。

每一個陣列中的元素，都附有一個能讀相對應元素的數字，稱做數字索引（numeric index）。在 JavaScript 中，陣列的索引開始於數字 0 且能透過不同的方法來操作。

JavaScript 中的陣列：

var myArray = [1, 2, 3, 4];
var catNamesArray = ["Jacqueline", "Sophia", "Autumn"];
//Arrays in JavaScript can hold different types of data, as shown above.
