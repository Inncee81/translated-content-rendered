可縮放向量圖形（Scalable Vector Graphics, SVG）是基於 XML 語法的 2D 向量圖片格式。

W3C 在 1990 晚期啟動 SVG 開發，但它要在 IE 9 開始支援 SVG 後才開始發跡。目前所有主流瀏覽器都支援 SVG。

由於 SVG 基於 XML 語法，所以它可以透過 CSS 樣式化、並透過 JavaScript 互動。HTML5 目前能在 HTML 檔案內直接嵌入 SVG 標籤。

由於 SVG 是向量圖形，它因此可以無限縮放。也由於你可以建立介面元素（interface element）還有在任何螢幕尺寸縮放的圖形，它在響應式設計上也相當重要。SVG 也提供一組有用的工具，例如剪輯、遮罩、濾器、動畫（clipping, masking, filters, animation）。
