標準通用標記式語言（Standard Generalized Markup Language 或簡稱 SGML）是一套 ISO 所定義，用來規範宣告式標記語言的標準。

在 Web 環境當中，HTML 4、XHTML 以及 XML 都是知名的以 SGML 為基礎的語言。然而值得一提的是，HTML 自第五版起擁有了自己的剖析規則，不再是以 SGML 為基礎的語言。
