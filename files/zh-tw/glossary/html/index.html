    超文件標示語言（HTML，HyperText Markup Language）是用於指定網頁結構的描述性語言。

    簡歷

    在 1990 年，作為 Tim Berners-Lee 對網路所知的一部分，超文本的概念被定義起來了。隔年，Berners-Lee 以 SGML 的標記為基礎定型起來。IETF 於 1993 年規範了 HTML、並在 1995 年發布 2.0（第二版）。1994 年 Berners-Lee 成立了 W3C 以便發展網路。1996 年，W3C 接手 HTML 的工作，並在隔年發布了 HTML 3.2 Recommendation（HTML 3.2 建議）。在 1999 年發布了 HTML 4.0 並在 2000 年成為 ISO 標準。

    這段期間，偏好 XHTML 的 W3C 幾乎拋棄了 HTML，使得 2004 年一個稱作 WHATWG 的獨立團體成立起來。也因為 WHATWG 的努力，HTML5 的工作持續進行：兩個組織在 2008 年發布了第一份草稿、最終標準則在 2014 年實行。

    概念及語法

    HTML 是個以元素構成的純文字文件。元素通常被開啟與關閉的標籤包覆起來。每個標籤都以角括號（<>）作為開始與結束。不過，也有些不包覆任何文件的空標籤（empty or void tag），例如 <img>。

    HTML 文件的副檔名通常存為 .htm 或是 .html。它們會放在網頁伺服器上運作、並由瀏覽器渲染。
