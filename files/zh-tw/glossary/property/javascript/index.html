property（屬性）是典型的物件特徵，經常與資料結構聯結。屬性分為兩種類型：

  • 實體屬性（Instance properties）：保存的資料只於特定的物件實體中。
  • 靜態屬性（Static Properties）：保存的資料為所有該型別物件所共有。

屬性擁有一個名字（為字串）以及值（可以是基本資料型別、方法或是物件參考）。請注意，當我們說「一個屬性保存著物件」，其實是指「一個屬性保存著物件的參考（reference）」。這其中的差別在於，當你改變屬性中的值的時候，所指向的物件實體之值也會改變。
