Learn a new term:

CSRF
跨站請求偽造（Cross-Site Request Forgery, CSRF）是一種冒充信任用戶，來傳送非預期指令的攻擊。

網路技術包含了一長串的術語以及縮寫，這些會被使用在文件跟程式碼中。此術語表提供你需要知道的辭彙與縮寫的定義，以便能成功理解和建構網路。
