DOM(Document Object Model, DOM) 是用來表示和與任何HTML 或 XML 文件互動的 API。DOM是一個載入於browser中的文件模組，它的表示方式像樹的節點一樣, 每個節點代表著文件的一部分(如：element、 文字或留言)。

DOM是一個在Web中最常被使用的APIs，因為它在瀏覽器中允許程式碼執行存取和互動任何在文件中的節點。 節點可以被創建、移動和變更。事件監聽器可以被新增至節點中，當發生特定事件時會被觸發。

DOM最初並沒有被定義—它是來自於當瀏覽器開始啟用JavaScript時出現。這個老舊的DOM有時候會稱DOM 0。此時此刻, WHATWG推動著DOM現存的標準.
