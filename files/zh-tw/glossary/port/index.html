對使用 IP address 連線到網路的電腦來說，通訊埠是傳輸時的端點。我們可用不同的數字（通訊埠號）來分辨不同的通訊埠，而小於 1024 號的每個埠通常都有自己預設使用的 protocol。

舉例來說，HTTP 通訊協定的預設埠號是 80，而 HTTPS 通訊協定預設的埠號是 443，這麼 HTTP 伺服器預設就會在這些通訊埠上等待請求。網際網路上的每個通訊協定都會有自己的預設通訊埠: SMTP (25)、POP3 (110)、IMAP (143)、IRC (194) 等等。
