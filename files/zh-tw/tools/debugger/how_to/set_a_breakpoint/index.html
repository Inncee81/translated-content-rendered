你可以使用以下任一種方法來設置中斷點；

  • 在原始碼視窗中，點選你想要設置中斷的行號來設置中斷點。
  • 在原始碼視窗，當你位於你想要中斷的那一行上時，啟動 context menu，然後選擇 "新增中斷點"
  • 在 原始碼視窗，標記出你想要中斷的那一行，然後按下 Ctrl+B (Windows/Linux) 或是 Command+B (Mac OS X)

以下影片使用 context menu 來設置一個中斷點:

每個中斷點會在除錯器中的兩個地方顯示；

  • 中斷清單中會顯示檔名以及中斷點所在的行號。
  • 在原始碼視窗 中，如果中斷點是有條件的，那麼那一行將會以藍色或橘色的箭頭標記。
