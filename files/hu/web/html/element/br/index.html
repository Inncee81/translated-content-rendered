Összefoglalás

A HTML <br> elem (vagy HTML sortörés elem) egy sortörést hoz létre a szövegben (kocsi-vissza). Hasznos versek és olyan címek írásakor, ahol a sorok felosztása fontos.

Ne használja a <br> elemet a szöveg sorai között lévő térköz növelésére. Használja a CSS margin tulajdonságot vagy a <p> elemet.

  • Tartalom kategóriák Folyam tartalom, kifejezés tartalom.
  • Engedélyezett tartalom Semmi, ez egy üres elem.
  • Címke elhagyás Lennie kell kezdő címkének és tilos a záró címke. Az XHTML dokumentumokban így írja ezt az elemet: <br />.
  • Engedélyezett szülő elemek Bármely elem, amely elfogadja a kifejezés tartalmat.
  • DOM felület HTMLBRElement

Attribútumok

Ez az elem a globális attribútumokkal rendelkezik.

clear Deprecated since HTML4.01 Obsolete since HTML5
Azt jelzi, hogy hol kezdődjön a következő sor a törés után.

Használati megjegyzés: Ez az attribútum elavult a HTML5 óta és a szerzők nem használhatják. Használja helyette a CSS clear tulajdonságot.
