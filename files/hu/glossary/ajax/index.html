Az Ajax (Asynchronous JavaScript And XML), egy programozási gyakorlat komplex és dinamikus weblapok építésére egy XMLHttpRequest nevezetű technológiát használva.

Az Ajax lehetővé teszi a DOM egy részének a frissítését egy HTML oldalon, teljes oldalfrissítés szükségessége nélkül. Az Ajax lehetővé teszi az aszinkronos munkát, ami azt jelenti, hogy amíg az oldal célzott része megpróbál újratölteni, addig az oldal többi része tovább fut (ezzel ellentétben a szinkronos módszer blokkolja a kód további futását, amíg az oldal célzott része újra nem töltött).

Az interaktív weboldalakkal és modern internetes szabványokkal, az Ajax-ot szépen lassan felváltják a JavaScript keretrendszereken belüli függvények és az eredeti Fetch API Szabvány.
