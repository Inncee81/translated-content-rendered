Üdvözöljük! Az oldal meglátogatásával megtette az első lépést, hogy az MDN közreműködője legyen.

Az itt található útmutatók lefedik az MDN-en közreműködés minden részét, beleértve a stílus útmutatókat, a szerkesztőnk és az eszközök útmutatóját, és így tovább. Kérjük, olvassa el (és tartsa be) a Mozilla feltételeit mielőtt bármilyen oldalt szerkesztene vagy létrehozna.

Ha eddig még nem működött közre az MDN-en, akkor a Kezdő lépések útmutató segíthet a feladat kiválasztásában.

MDN content and style guides
These guides provide details on how MDN documentation should be written and formatted, as well as how our code samples and other content should be presented.
MDN editor guide
The WYSIWYG (what-you-see-is-what-you-get) editor offered by the Mozilla Developer Network wiki makes it easy to contribute new content.  The MDN editor guide provides some information on how to use the editor, as well as some information about helpful features that may improve your productivity.
