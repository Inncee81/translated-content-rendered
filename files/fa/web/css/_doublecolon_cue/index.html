شبه عنصر cue:: یک عنصر انتخاب شده با نشانه‌های WebVTT را تطابق می‌دهد. این شبه عنصر می‌تواند برای آرایش زیرنویس و دیگر موارد محتوای تصویری با VTT tracks مورد استفاده قرار گیرد.

::cue {
  color: yellow;
  font-weight: bold;
}

Allowable properties

Only a small subset of CSS properties can be used in a rule with ::cue in its selector:

  • color
  • opacity
  • visibility
  • text-decoration and its longhand properties
  • text-shadow
  • background and its longhand properties
  • outline and its longhand properties
  • font and its longhand properties, including line-height
  • white-space
  • text-combine-upright
  • ruby-position

The properties are applied to the entire set of cues as if they were a single unit. The only exception is that background and its shorthand properties apply to each cue individually, to avoid creating boxes and obscuring unexpectedly large areas of the media.

Syntax

::cue | ::cue( <selector> )

Example

The following CSS sets the cue style so that the text is white and the background is a translucent black box.

::cue {
  color: #fff;
  background-color: rgba(0, 0, 0, 0.6);
}
