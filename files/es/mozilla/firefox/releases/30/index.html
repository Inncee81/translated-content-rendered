Cambios para los Desarrolladores Web

Herramientas para Desarrolladores

  • Ha sido implementado el Modelo de Caja Resaltado (Box Model Highlighter) (error 663778).
  • En cualquier parte el nodo DOM aparece en la salida de la consola, se resalta cuando se encima la salida de esa consola (error 757866). Sucede lo mismo con todas las funciones y objetos JS, son resaltados en la salida de la consola (error 584733). Más información acerca de las mejoras de la consola pueden econtrarse en el siguiente post del blog.
  • Sopoerte para Console.count() ha sido agregado (error 922208).

CSS

  • La propiedad background-blend-mode ha sido habilitada por defecto (error 970600).
  • La propiedad no estándar  overflow-clip-box ha sido implementada para utilizarse solamente en las hojas de estilo UA (error 966992).
  • La propiedad line-height afecta ahora a las entradas de lineas de texto sencillas  (<input type=text|password|email|search|tel|url|unknown> types) aunque no puede reducir su tamaño por debajo de una altura de la línea de 1.0 (error 349259).
  • La propiedad line-height ahora también afecta el type=button, sin restricciones (error 697451).
  • El cambio al nombre de los keyframes no afecta los elementos actuales (error 978648).

HTML

Sin cambios.

JavaScript

  • New ES6-compatible array comprehensions [for (item of iterable) item] and generator comprehensions (for (item of iterable) item) have been implemented (error 979865).
  • Typed arrays are now extensible and support new named properties (error 695438).
  • The Error.prototype.stack property now contains column numbers (error 762556) and has been improved when using Function() and eval() calls. This can help you to better debug minified or generated JavaScript code.
  • The Promise.cast() method has been renamed to Promise.resolve() (error 966348).

Interfaces/APIs/DOM

  • Navigator.sendBeacon has been implemented, easing telemetry collection (error 936340).
  • Added a relList property returning a DOMTokenList to HTMLLinkElement, HTMLAreaElement and HTMLAnchorElement (error 968637).
  • As per the latest specification, the first argument of OscillatorNode.start and OscillatorNode.stop is now optional and defaults to 0 (error 982541).
  • The method Navigator.requestWakeLock() and the non-standard MozWakeLock are no longer available from the Web on Desktop (error 963366).
  • The DOM_VK_ENTER constant has been removed from KeyboardEvent (error 969247).
  • Web components' Document.register has been adapted to follow the behavior described in the latest version of the specification (error 856140).
  • Basic support for Hit regions on canvas has been added: the methods CanvasRenderingContext2D.addHitRegion() and CanvasRenderingContext2D.removeHitRegion() have been added. These are disabled by default; to activate them set the preference canvas.hitregions.enabled to true. (error 966591)
  • The non-standard, and deprecated since Firefox 15, Blob.mozSlice is no longer supported (error 961804).
  • The non-standard ArchiveReader and ArchiveRequest are no longer exposed to the Web (error 968883).
  • WebIDL constructors cannot be called as functions anymore. They need to be preceded by the keyword new. (error 916644)
  • Added support for a new value (alpha) for the second, optional, parameter of the Canvas.getContext() method allowing to define if alpha blending must be stored or not for this context. When not, the per-pixel alpha value in this store is always 1.0. This allows the back-end to implement a fast-track. (error 982480)
  • GlobalWorkerScope.consle now returns for the regular Console; WorkerConsole has been removed (error 965860).

MathML

No change.

SVG

  • <feDropShadow>, and its interface SVGFEDropShadowElement, from the Filter Effects Module are now supported (error 964200).

Audio/Video

  • On Linux, Gstreamer 1.0 is now supported (instead of 0.10) (error 806917).

Security

No change.

Changes for add-on and Mozilla developers

  • The interface nsIDOMWindowUtils now supports the Boolean attribute audioMuted and audioVolume, a float in the range [0.0 ,  1.0], allowing to control the sound produced by a window (that is any tab or iframe). There is no UI for this, but is available to add-ons. (error 923247)
