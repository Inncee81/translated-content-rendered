El encabezado Age contiene el tiempo medido en segundos que el objeto ha estado en la memoria caché de servidor proxy.
 
El encabezado Age suele estar seteado en 0 (cero). Si Age : 0 probablemente en ese instante se acaba de obtener la respuesta del servidor de origen, de lo contrario, por lo general esto se calcula como la diferencia entre la fecha actual del proxy y la fecha dada por el parámetro en cabecera "Date" (Date ) incluído en la respuesta HTTP.
 
Tipo de Cabecera Response header
Nombre de Cabecera Prohibido Forbidden header name no

Sintaxis

Age: <segundos>

Directivas

<segundos>

Número entero no negativo, que representa el tiempo en segundos que el objeto ha almacenado en la caché del proxy.

Ejemplos

Age: 24

Especificaciones

Specification Title
RFC 7234, section 5.1: Age Hypertext Transfer Protocol (HTTP/1.1): Caching
