Esta  es una lista alfabetica de atributos de MathML. Más detalles de cada atributo está disponible en cada página de los elementos.

Notas:

  • Los elementos MathML <mstyle> y <math>   aceptan todos los atributos de todos los elementos de presentacion.
  • Vea los valores de las notas sobre los valores y unidades en MathML.

accentunder <munder>, <munderover> Un valor booleano que especifíca si el operador debe ser tratado como un acento.
actiontype <maction> Un valor de cadena que especifica la acción pasada al elemento.

No implementado columnspacing <mtable> Especifica el espacio entre las columnas de la tabla.

No implementado decimalpoint <mstyle> Si el valor de decimalpoint es usado para especificar la alineación, este atributo especifica el caracter del punto de alineación dentro de las columnas de <mstack> y <mtable> .
denomalign <mfrac> La alineación del denominador bajo la fracción.
depth <mpadded> Establece o incrementa la profundidad. Ver longitud.
dir <math>, <mi>, <mo>, <mrow>, <ms>, <mtext> La dirección del texto. Posibles valores son ltr (izquierda a derecha) o rtl (derecha a izquierda).
display <math> Especifica el modo de renderizado. Los valores block e inline son permitidos.
displaystyle <mstyle>, <mtable>

Un valor booleano que especifica si un espacio más vertical es usado para las ecuaciones mostradas o, si se establece como false, un diseño más compacto se utiliza para mostrar las formulas.

No implementado equalcolumns <mtable>

No implementado equalrows <mtable> Un valor booleano que indica si se debe obligar a las filas a tener la misma altura total.
fence <mo>

Un valor booleano que indica si el operador está cercado (como parentesis). No hay efecto visual de este atributo.
No implementado form <mo>

Especifica el rol del operador en una expresión cerrada.
frame <mtable>

Especifica bordes de toda una <mtable>. Posibles valores son: none (default), solid y dashed.
No implementado framespacing <mtable> Especifica espacio adicional agregado entre la tabla y el frame.

mathbackground All El colo de fondo. Puedes usar #rgb, #rrggbb y nombres de color HTML.
mathcolor All El color de texto. Puedes usar  #rgb, #rrggbb y nombres de color HTML.
mathsize <mi>, <mn>, <mo>, <ms>, <mtext> El tamaño del contenido. Comenzando con Gecko 20.0 (Firefox 20 / Thunderbird 20 / SeaMonkey 2.17) valores sin unidades están permitidos y se interpretan como N veces el tamaño predeterminado.
mathvariant <mi>, <mn>, <mo>, <ms>, <mtext> La clase logica del identificador, la cual varia en tipografia.
maxsize <mo> El tamaño máximo del operador.
No implementado minlabelspacing <mtable> Un valor de longitud espcificando el minimo espacio entre una etiqueta y la celda adjacente en la fila.
minsize <mo> El tamaño mínimo del operador.
movablelimits <mo> Especifica si los guiones bajos y altos se mueven a las posiciones de sub-índice o super-índice.
notation <menclose> Una lista de anotaciones, separado por espacios en blanco, para aplicar a los elementos secundarios.
numalign <mfrac> La alineación del numerador en la fracción.
open <mfenced> Una cadena para el delimitador de apertura.
No implementado overflow <math>

Si una expresión es demasiado larga para encajar en el ancho definido, este atributo establece el tratamiendo preferido.
No implementado position <msgroup>, <msrow>, <mscarries>, <msline>  
rowalign <mtable>, <mtd>, <mtr> Especifica la alineación vertical de las celdas de la tabla.
rowlines <mtable> Especifica los bordes de las filas de la tabla.
rowspacing <mtable> Especifica el espacio entre las filas de la tabla.
rowspan <mtd> Un valor entero no negativo que indica el númerp de filas que se extiende las celdas.
rspace <mo> La cantidad de espacios luego del operador.
rquote <ms> La marca de cierre de cita que encierra el contenido. El valor predeterminado es "&quot;".
scriptlevel <mstyle> Controles sobre el tamaño de la fuente. Cuando mayor sea el nivel del script, menor el tamaño de la fuente.
scriptminsize <mstyle> Especifica el minimo tamaño de fuente permitidos debido a cambios en scriptlevel.
Comenzando con Gecko 20.0 (Firefox 20 / Thunderbird 20 / SeaMonkey 2.17) valores sin unidades y porcentajes son permitidos y se interpretan como multiplos de "8pt".
scriptsizemultiplier <mstyle> Especifica el multiplicador a ser utilizado para ajustar el tamaño de fuente debidp a cambios en scriptlevel.
selection <maction> El elemento secundario al cual se dirige la acción.
separator <mo> No existe efecto visual para este elemento, pero especifica si el operador es un separador (como las comas).
separators <mfenced> Una secuencia de ceros o más caracteres a ser usados por diferentes separadores.
No implementado shift <msgroup>  
No implementado side <mtable> Especifica la posición donde los elementos<mlabeledtr> deberían ser posicionados.
No implementado src <mglyph> La ubicación de la imagen.
No implementado stackalign <mstack>  
stretchy <mo> Especifica si el operador se extiende hasta el tamaño del elemento adyacente.
subscriptshift <mmultiscripts>, <msub>, <msubsup> El espacio mínimo por el que se desplace el subíndice por debajo de la línea de base de la expresión.
supscriptshift <mmultiscripts>, <msup>, <msubsup> El espacio mínimo por el que se desplace el superíndice por encima de la línea de base de la expresión.
symmetric <mo>

Si stretchy es true, este  atributo especifica si el operador debe estar simetrico verticalmente al eje imaginario de matemáticas (linea centrada de fracción).
voffset <mpadded> Ajusta la posición vertical del contenido secundario.
width <mglyph>, <mpadded>, <mspace>, <mtable> Especifica el ancho deseado. Ver longitudes para posibles valores.
xlink:href All Puede ser usado para establecer un hyperlink para determinada URL. Sin embargo, se ainma a los autores a usar el atributo href en su lugar.
xmlns <math> Especifica la URL para el espacio de nombres MathML (http://www.w3.org/1998/Math/MathML)
