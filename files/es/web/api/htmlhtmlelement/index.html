The HTMLHtmlElement interface serves as the root node for a given HTMLÂ document.Â  This object inherits the properties and methods described in the HTMLElement interface.

You can retrieve the HTMLHtmlElement object for a given document by reading the value of the document.documentElement property.

Properties

Inherits properties from its parent, HTMLElement.

HTMLHtmlElement.versionÂ 
Is a DOMString representing the version of the HTML Document Type Definition (DTD) that governs this document. This property should not be used any more as it is non-conforming. Simply omit it.

Methods

No specific method; inherits methods from its parent, HTMLElement.
