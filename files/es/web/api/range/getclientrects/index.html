Esta es una tecnología experimental
Comprueba la Tabla de compabilidad de navegadores cuidadosamente antes de usarla en producción.

El método Range.getClientRects() regresa una lista de objetos DOMRect los cuales representan el área de la pantalla ocupada por el rango. El resultado es generado al agregar los resultados de las llamadas a Element.getClientRects() para cada uno de los elementos dentro del rango.

Sintaxis

rectList = range.getClientRects()

Ejemplo

range = document.createRange();
range.selectNode(document.getElementsByTagName("div").item(0));
rectList = range.getClientRects();
