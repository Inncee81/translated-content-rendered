La propiedad de solo lectura timeOrigin de la inferfaz Performance devuelve una marca de tiempo de alta precisión del inicio de medida de rendimiento.

Nota: Esta característica está disponible en Web Workers.

Sintaxis

var timeOrigin = performance.timeOrigin

Valor

Una marca de tiempo de alta precisión.

Especificaciones

Especificación Estado
High Resolution Time Level 2
La definición de 'timeOrigin' en esta especificación.		 Recommendation
