Desaprobado
This feature has been removed from the Web standards. Though some browsers may still support it, it is in the process of being dropped. Do not use it in old or new projects. Pages or Web apps using it may break at any time.

Esta propiedad está deprecada en Navigation Timing Level 2 specification. Por favor usa Performance.timeOrigin en vez esta..

La propiedad de solo lecutra Performance.timing de legado devulve un objeto PerformanceTiming que contienen información relacionada con el rendimiento en relación a la latencia.

Esta propiedad no está disponible en workers.

Sintaxis

var timingInfo = performance.timing;
