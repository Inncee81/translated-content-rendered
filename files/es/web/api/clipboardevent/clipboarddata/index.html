Esta es una tecnología experimental
Comprueba la Tabla de compabilidad de navegadores cuidadosamente antes de usarla en producción.

La propiedad ClipboardEvent.clipboardData, del tipo DataTransfer, puede ser usuada:

  • Para especificar qué datos mover al portapapeles cuando se produzcan los eventos cut y copy, normalmente a través de setData(format, data);
  • Para obtener la información del portapapeles a ser pegada cuando se produce el evento paste, normalmente a través de getData(format).

Para más información, mira la documentación de los eventos cut, copy, y paste.

Sintaxis

data = ClipboardEvent.clipboardData
