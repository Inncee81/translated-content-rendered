La propiedadsupportedContentEncodings de solo lectura (read-only) de la interfaz PushManager devuelve un array de los codigos de contenido que puede soportar y que pueden ser usados para encriptar los datos (payload) de un push-message.

Sintaxis

var encodings[] = PushManager.supportedContentEncodings

Valor

Un array de Strings
