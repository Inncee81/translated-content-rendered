Resumen

Retorna un boleano indicando si event.preventDefault() fue llamado en el evento o no.

Nota: Deberías usar este en lugar del que no es estándar, el método obsoleto getPreventDefault() (ver error 691151).

Sintaxis

bool = event.defaultPrevented

Ejemplo

 if (e.defaultPrevented) {
   /* El evento por defecto fue prevenido */
 }
