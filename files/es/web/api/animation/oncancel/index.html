La propiedad oncancel de la interfaz Animation de la  Web Animations API es el manejador de eventos para el evento cancel .

El evento cancel puede ser activado manualmente con Animation.cancel() cuando la animación entra en estado de reproducción "idle"(inactivo) desde otro estado, como cuando una animación se elimina de un elemento antes de que termine de reproducirse.

La creación de una nueva animación, inicialmente inactiva, no activa el evento cancel en la nueva animación.

Sintaxis

var cancelHandler = Animation.oncancel;

Animation.oncancel = cancelHandler;

Valor

Una función que será ejecutada cuando la animación sea cancelada, o null si no hay un manejador de eventos cancel.

Ejemplos

Si esta animación es cancelada, elimina su elemento.

animation.oncancel = animation.effect.target.remove();
