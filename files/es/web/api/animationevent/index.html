Propiedades

También hereda las propiedades de su padre Event.

AnimationEvent.animationName Read only
Es un DOMString que contiene el valor de la propiedad CSS animation-name asociado con la transición.
AnimationEvent.elapsedTime Read only
Es un número flotante que da la cantidad de tiempo que ha estado corriendo la animación, en segundos, cuando este evento es disparado, excluyendo cualquier vez que la aimación fue pausada.  Para un evento "animationstart", elapsedTime es 0.0 a ménos que hubiera un valor negativo para animation-delay, en tal caso sería disparado el evento con elapsedTime conteniendo (-1 * retraso).
AnimationEvent.pseudoElement Read only
Es un DOMString, empezando con '::', conteniendo el nombre del pseudo-element en el cual corre la animación. Si la animación no corre sobre un pseudo-element sino en el elemento, un string vacío: ''. 

Constructor

AnimationEvent()
Crea un evento AnimationEvent con los parametros dados.

Métodos

También hereda los métodos de su padre Event.

AnimationEvent.initAnimationEvent()
Inicializa un evento AnimationEvent creado utilizando el método obsoleto  Document.createEvent("AnimationEvent")
