Propiedades

Hereda las propiedades del padre, HTMLElement.

HTMLHeadElement.profile
Es un DOMString que representa las URI de uno o más perfiles de metadatos (separados por espacios en blanco).

Métodos

No hay un método especifico; hereda los métodos del padre, HTMLElement.
