Resumen

La propiedad icon indica la URL del icono a usar dentro de la notificacion.

El valor para esta propiedad es enviada durante la instanciacion Notification  definiendo la propiedad icon el objeto pasa al  Notification constructor.

Sintaxis

var icon = instanceOfNotification.icon;

Tipo

String que representa una URL válida.
