El metodo location.reload() carga de nuevo la URL actual, como lo hace el boton de Refresh de los navegadores.

La recarga puede estar bloqueada y arrojar un SECURITY_ERROR DOMException . Esto sucede si el origin de la llamada location.reload() difiere del origen de la página que posee el objeto { {domxref("Location")}}. Consulta Same-origin policy para mayor información.

Syntax

location.reload();

Ejemplo

HTML

<button id="reload">Click para recargar</button>

JavaScript

const reload = document.getElementById('reload');

reload.addEventListener('click', _ => { // el _ es para indicar la ausencia de parametros
    location.reload();
});
