La interfaz Crypto representa las características básicas de criptografía disponibles en el contexto actual. Permite acceder a un generador de números aleatorios criptográficamente fuerte y a primitivos criptográficos.

Un objeto con esta interfaz está disponible en el contexto de la Web a través de la propiedad Window.crypto.

Propiedades

Esta interfaz implementa las propiedades definidas en RandomSource.

Crypto.subtle Read only
Retorna un SubtleCrypto objeto proporcionando acceso a primitivos criptográficos comunes, como el hashing, el firmado, cifrado o descifrado.

Métodos

Esta interfaz implementa las propiedades definidas en RandomSource.

Crypto.getRandomValues()
Llena el pasado TypedArray valores aleatorio sólidos criptográficamente.
