El método readAsText se usa para leer el contenido de un tipo Blob o File. Cuando la operación de lectura se completa, el atributo readyState cambia a DONE, el evento loadend se dispara y el atributo result contiene el contenido del archivo como una cadena de texto.

Sintaxis

instanceOfFileReader.readAsText(blob[, encoding]);

Parametros

blob
Una variable de tipo Blob o File de la cual se leerán los datos.
encoding Optional
Una cadena de texto especificando la codificación de caracteres que se usará en los datos leidos. Por defecto, se utiliza UTF-8 si el parametro no se especifica
