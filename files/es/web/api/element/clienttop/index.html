Resumen

El ancho del borde superior de un elemento en pixel. No incluye ni elmargin-top , ni elpadding-top . clientTop es de sólo lectura.

Las aplicaciones "Gecko-based applications" implementan el uso de clientTop a partir de Gecko 1.9 (Firefox 3,ver error 111207). Esta propiedad no está disponible en Firefox 2 o anteriores.

Sintaxis

vartop = element.clientTop;
