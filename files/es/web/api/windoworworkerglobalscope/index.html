El mixin WindowOrWorkerGlobalScope describe diversas características comunes a las interfaces Window y WorkerGlobalScope. Cada una de estas interfaces pueden, por supuesto, añadir más funcionalidades sobre las listadas a continuación.

Nota: WindowOrWorkerGlobalScope es un mixin y no una interface; no se puede crear un objeto de tipo WindowOrWorkerGlobalScope.

Propiedades

Estas propiedades se definen en el mixin WindowOrWorkerGlobalScope, y son implementadas por Window y WorkerGlobalScope.

WindowOrWorkerGlobalScope.caches Read only
Devuelve el objeto CacheStorage, asociado al contexto actual. Este objeto habilita funcionalidades como guardar assets para su utilización offline, y generar respuestas personalizadas a las peticiones.
WindowOrWorkerGlobalScope.indexedDB Read only
Proporciona un mecanismo para que las aplicaciones puedan acceder asíncronamente a las capacidades de las bases de datos indexadas.; devuelve un objeto IDBFactory.
WindowOrWorkerGlobalScope.isSecureContext Read only
Devuelve un booleano indicando si el contexto actual es seguro (true)  o no (false).
WindowOrWorkerGlobalScope.origin Read only
Devuelve el origen del objeto global, serializado como un string. (Esto no parece estar implementado todavía en ningun navegador).

Métodos

Estas propiedades se definen en el mixin WindowOrWorkerGlobalScope, y son implementadas por Window y WorkerGlobalScope.

WindowOrWorkerGlobalScope.atob()
Decodifica un string de datos que ha sido codificado utlizando codificación en base-64.
WindowOrWorkerGlobalScope.btoa()
Crea un string ASCII codificado en base-64 desde un string de datos binarios.
WindowOrWorkerGlobalScope.clearInterval()
Procesa el set de ejecución repetida utilizando WindowOrWorkerGlobalScope.setInterval().
WindowOrWorkerGlobalScope.clearTimeout()
Cancela el retraso del set de ejecuión utilizando WindowOrWorkerGlobalScope.setTimeout().
WindowOrWorkerGlobalScope.createImageBitmap()
Acepta variedad de recursos de imagen, y devuelve una Promise que se resuelve como un ImageBitmap. Opcionalmente, la fuente se recorta como un rectángulo de pixels con origen en (sx, sy) con ancho sw, y altura sh.
WindowOrWorkerGlobalScope.fetch()
Comienza el proceso de búsqueda de un recurso desde la red.
WindowOrWorkerGlobalScope.setInterval()
Programa la repetición de la ejecución de una función dado un número de lapsos en milisegundos.
WindowOrWorkerGlobalScope.setTimeout()
Programa la ejecución de una función dada una cantidad de tiempo.
