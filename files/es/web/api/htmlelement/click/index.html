El método HTMLElement.click() simula el click de un ratón físico en un elemento HTML.

Cuando click() se usa en un elemento que soporte este método (como un elemento <input>), el elemento lanza el evento asociado al click. Luego, este evento iniciará otros eventos asociados al clicado en elementos más altos en la jerarquía del documento html (o de la cadena de eventos).

Sintaxis

elemento.click()
