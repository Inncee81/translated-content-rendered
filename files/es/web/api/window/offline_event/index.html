El evento offline se activa cuando el navegador ha pierdido acceso a la red y el valor de navigator.onLine cambia a false.

Propiedad Tipo Descripción

type Read only DOMString Tipo de evento.
bubbles Read only Boolean Si el evento normalmente burbujea o no.
cancelable Read only Boolean Si el evento normalmente es cancelable o no.
