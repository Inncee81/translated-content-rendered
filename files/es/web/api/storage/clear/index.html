El método clear() de la interfaz Storage, al invocarlo, elimina todos los registros del almacen local.

Sintaxis

storage.clear();

Parámetros

No recibe parámetros.

Devuelve

No devuelve ningún valor.

 

Ejemplo

 

La siguiente función crea tres elementos con datos dentro del almacen local, seguidamente, los elimina usando clear().

function populateStorage() {
  localStorage.setItem('bgcolor', 'red');
  localStorage.setItem('font', 'Helvetica');
  localStorage.setItem('image', 'myCat.png');

  localStorage.clear();
}
