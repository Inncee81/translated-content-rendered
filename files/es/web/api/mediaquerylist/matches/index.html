La propiedad matches de sólo lectura de la interfaz MediaQueryList es Boolean que devuelve true si el document coincide actualmente con el medio lista de consultas, o false si no.

Sintaxis

var matches = MediaQueryList.matches;

Value

A Boolean; returns true if the document currently matches the media query list, false if not.

Examples

var mql = window.matchMedia('(max-width: 600px)');

if(mql.matches) {
  // media query test returning true
}
