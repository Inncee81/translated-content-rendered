Resumen

La pseudo-clase CSS :-moz-first-node representa cualquier elemento que sea el primer nodo hijo de algún otro elemento. Se diferencia de :first-child en que  no selecciona al primer hijo si tiene texto (que no sea espacios en blanco) detrás de él.

Cualquier espacion en blanco al principio del elemento se ignora a la hora de determina cual elemento es :-moz-first-node.

Síntaxis

span:-moz-first-node { propiedades del estilo }

Ejemplo

CSS

span:-moz-first-node {
  background-color: lime;
}

HTML

<div>
  <span>:-moz-first-node</span>
  <span>:-moz-last-node</span>
</div>
