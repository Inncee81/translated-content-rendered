Sumario

La propiedad CSS quotes indica cómo debe renderizar las citas el navegador.

Valor inicialdepends on user agent
Applies toall elements
Heredableyes
Valor calculadocomo se especifica
Animation typediscrete

Sintaxis

/* Valor con palabras clave */
quotes: none;

/* Valor <string> */
quotes: "«" "»";           /* Especifica open-quote y close-quote a las marcas de cita francesas */
quotes: "«" "»" "‹" "›";   /* Especifica dos niveles de marcas de cita */

/* Valores globales */
quotes: inherit;
quotes: initial;
quotes: unset;

Valores

none
Los valores open-quote y close-quote de la propiedad content no renderiza marcas de cita.
[<string> <string>]+
Uno o más pares de valores <string> para open-quote y close-quote. El primer par representa el nivel más externo de citación, el segundo par es para el primer nivel anidado, el siguiente par para el siguiente nivel, y así sucesivamente.

Sintaxis formal

none | auto | [ <string> <string> ]+

Ejemplos

q { quotes: '"' '"' "'" "'" }
q:before { content: open-quote }
q:after  { content: close-quote }

Notas

Desde Firefox 3.5, el valor inicial de la propiedad de las citas se puede leer usando -moz-initial. Esto no es posible en versiones anteriores de Firefox.
