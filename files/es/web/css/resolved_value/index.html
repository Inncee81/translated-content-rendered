El valor resuelto (resolved value) de una propiedad CSS es el valor devuelto por getComputedStyle(). Para la mayoría de las propiedades, esto es el valor calculado, pero para algunas propiedades antiguas (incluyendo width y height), éste es el valor usado. Véase el enlace a la especificación a continuación para más detalles por propiedad.
