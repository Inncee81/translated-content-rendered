Resumen

La propiedad CSS background-blend-mode describe cómo las imágenes de fondo y el color de fondo del elemento deben mezclar entre sí. 

Los modos de fusión deben ser definidos en el mismo orden que la propiedad CSS background-image. Si la cantidad de modos de fusión y de imagenes de fondo no son iguales, se repetirán y/o truncarán hasta igualar las cantidades.

Valor inicialnormal
Applies toAll elements. In SVG, it applies to container elements, graphics elements, and graphics referencing elements.. It also applies to ::first-letter and ::first-line.

Sintaxis

/* Un valor */
background-blend-mode: normal;

/* Dos valores, uno por fondo */
background-blend-mode: darken, luminosity;

/* Valores globales */
background-blend-mode: initial;
background-blend-mode: inherit;
background-blend-mode: unset;

Valores

<blend-mode>
Es un <blend-mode> que denota el modo de fusión que se aplicará. Puede haber varios valores, separados por coma.
