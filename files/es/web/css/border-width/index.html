Resumen

La propiedad border-width define el ancho del borde.

  • valor inicial: medio
  • Se aplica a: todos los elementos
  • herencia: no
  • Porcentajes: N/A
  • Medio: visual
  • valor calculado: largo absoluto o '0' si el estilo del borde es ninguno ('none') o 'hidden'.

Sintaxis

 border-width: <border-width-value> {1,4} | heredado

Valores

<border-width-value>
thin | medium | thick | largo
thin
Un borde fino.
medium
Un borde mediano.
thick
Un borde grueso.
<length>
El grosor de un borde es un valor explícito por lo cual el ancho no puede ser nunca negativo.
Nota: Un valor em también es reconocido y soportado.

Ejemplos

Ver el ejemplo en vivo

    border-style: solid;
    border-color: #000;
}
