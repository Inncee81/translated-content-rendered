La propiedad de CSS  border-inline-start-color define el color del borde lógico inicial en línea de un elemento, que se asigna al estilo de borde físico dependiendo del modo de escritura, la direccionalidad y la orientación del texto del elemento. Esto corresponde a las propiedades border-top-color y border-bottom-color, o border-left-color, y border-right-color dependiendo de los valores definidos por writing-mode, direction, y text-orientation.

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

Sintaxis

border-inline-start-color: red;
border-inline-start-color: #ee4141;

Propiedades relacionadas son border-block-start-color, border-block-end-color, y border-inline-end-color, que define los otros colores del borde del elemento.

Ejemplo

Contenido HTML

<div>
  <p class="exampleText">Example text</p>
</div>

Contenido CSS

div {
  background-color: yellow;
  width: 120px;
  height: 120px;
}

.exampleText {
  writing-mode: vertical-lr;
  border: 10px solid blue;
  border-inline-start-color: red;
}
