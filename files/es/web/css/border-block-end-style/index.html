Esta es una tecnología experimental
Comprueba la Tabla de compabilidad de navegadores cuidadosamente antes de usarla en producción.

La propiedad CSS border-block-end-style define el estilo del borde del final lógico del bloque de un elemento, que se mapea a un estilo de borde físico, dependiendo el modo de escritura, direccionalidad y orientación de texto del elemento. Corresponde a las propiedades border-top-style, border-right-style, border-bottom-style, o border-left-style, dependiendo de los valores definidos para writing-mode, direction, y text-orientation.

border-block-end-style: dashed;
border-block-end-style: dotted;
border-block-end-style: groove;

Está relacionada a border-block-start-style, border-inline-start-style, y border-inline-end-style, que definen las otras propiedades de estilos de borde del elemento.

div {

}

.exampleText {
  writing-mode: vertical-lr;
  border: 5px solid blue;
  border-block-end-style: dashed;
}
