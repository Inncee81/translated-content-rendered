Esta es una tecnología experimental
Comprueba la Tabla de compabilidad de navegadores cuidadosamente antes de usarla en producción.

La propiedad CSS border-block-end-color define el color del borde del final lógico de un elemento, que se mapea a un color de borde físico, dependiendo el modo de escritura, direccionalidad y orientación de texto del elemento. Corresponde a las propiedades border-top-color, border-right-color, border-bottom-color, o border-left-color, dependiendo de los valores definidos para writing-mode, direction, y text-orientation.

border-block-end-color: yellow;
border-block-end-color: #F5F6F7;

Está relacionada con border-block-start-color, border-inline-start-color, y border-inline-end-color, que definen las otras propiedades de color de borde del elemento.

}

.exampleText {
  writing-mode: vertical-lr;
  border: 10px solid blue;
  border-block-end-color: red;
}
