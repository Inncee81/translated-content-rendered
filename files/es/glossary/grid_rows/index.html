--- title: Grid Row slug: Glossary/Grid_Rows translation_of: Glossary/Grid_Rows ---

Una grid row es una pista horizontal en un CSS Grid Layout, es el espacio entre dos líneas de cuadrícula horizontales. Se define por la propiedad grid-template-rows o con la propiedad shorthand grid o grid-template.

Además, se pueden crear filas en la cuadrícula implícita cuando los elementos se colocan fuera de las filas creadas en la cuadrícula explícita. Estas filas serán de tamaño automático por defecto, o pueden tener un tamaño especificado con la propiedad grid-auto-rows.

Cuando se trabaja con alineación en CSS Grid Layout, el eje por el que se ejecutan las filas se conoce como eje en línea, o fila.
