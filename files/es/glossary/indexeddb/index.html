--- title: IndexedDB slug: Glossary/IndexedDB translation_of: Glossary/IndexedDB ---

IndexedDB es una API Web diseñada para almacenar estructuras de datos grandes en los navegadores e indexarlas para realizar búsquedas de alto rendimiento. Al igual que los sistemas RDBMS basados en SQL, IndexedDB es un sistema de base de datos transaccional. Sin embargo, usa objetos de JavaScript en vez de columnas fijas para almacenar los datos.
