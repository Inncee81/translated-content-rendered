--- title: Elemento vacío slug: Glossary/Empty_element translation_of: Glossary/Empty_element ---

Un elemento vacío es un elemento de HTML, SVG, o MathML que no puede tener nodos secundarios (es decir, elementos anidados o nodos de texto).

Las especificaciones HTML, SVG, y MathML definen con precisión lo que cada elemento puede contener. Muchas combinaciones no tienen sentido semántico, por ejemplo un elemento <audio> anidado dentro de un elemento <hr>.

En HTML, el uso de una etiqueta de cierre en un elemento vacío normalmente no es válido. Por ejemplo, <input type="text"></input> es un elemento HTML no válido.

Los elementos vacíos en HTML son los siguientes:

  • <area>
  • <base>
  • <br>
  • <col>
  • <embed>
  • <hr>
  • <img>
  • <input>
  • <keygen>(HTML 5.2 Draft removed)
  • <link>
  • <meta>
  • <param>
  • <source>
  • <track>
  • <wbr>
