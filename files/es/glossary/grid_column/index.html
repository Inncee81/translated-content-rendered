--- title: Grid Column slug: Glossary/Grid_Column translation_of: Glossary/Grid_Column ---

Una columna de grid es una pista vertical en un CSS Grid Layout, y es el espacio entre dos líneas verticales de cuadrícula. Está definido por la propiedad grid-template-columns o por las propiedades abreviadas grid o grid-template.

Además, se pueden crear columnas en la cuadrícula implícita cuando los elementos se colocan fuera de las columnas creadas en la cuadrícula explícita. Estas columnas serán de tamaño automático por defecto, o pueden tener un tamaño especificado con la propiedad grid-auto-columns.

Cuando se utiliza la alineación en CSS Grid Layout, el eje sobre el que corren las columnas es conocido como el eje del bloque o de la columna.
