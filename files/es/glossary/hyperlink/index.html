Los Hipervínculos ó enlaces permiten conectar entre sí datos ó páginas web. En HTML, los elementos <a> representan hipervínculos que tienen como origen un elemento de la página web (por ejemplo cadenas de texto o imágenes), y que pueden tener como destino un elemento de otro sitio web (incluso pueden enlazar a otro punto de la misma página).
