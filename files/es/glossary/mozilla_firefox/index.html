Mozilla Firefox es un navegador gratuito de codigo abierto cuyo desarrollo es supervisado por Mozilla Corporation. Firefox functiona sobre Windows, OS X, Linus y Android.

First released in November 2004, Firefox is completely customizable with themes, plug-ins, and add-ons.  Firefox uses Gecko to render webpages, and implements both current and upcoming Web standards.

Lanzado en Noviembre del 2014, Firefox es completamente personalizable con temas y complementos. Firefox usa Gecko para renderizar paginas webs, e implementa tanto actuales como próximos estandares Web.
