WebSocket es un  protocolo que permite establecer conexiones  TCP entre el servidor y el cliente, permitiendo así el transporte de datos en cualquier momento.

Cualquier cliente o servidor de aplicaciones puede usar WebSockets, pero principalmente es usado por browsers y el servidor web. WebScoket permite enviar datos al cliente sin la necesidad de recibir ningún tipo de notificación, permitiendo la actualización dinámica del contenido.
