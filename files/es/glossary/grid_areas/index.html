--- title: Grid Areas slug: Glossary/Grid_Areas translation_of: Glossary/Grid_Areas ---

Un grid area es una o más grid cells que forman un área rectangular en la cuadrícula. Los grid areas se crean cuando se coloca un ítem usando disposición basada en líneas o cuando se definen áreas usando grid areas con nombre.

Las Grid areas deben ser de naturaleza rectangular; no es posible crear, por ejemplo, un grid area en forma de T o L.

En el siguiente ejemplo, tengo un contenedor de cuadrícula con dos ítems. Los he nombrado con la propiedad grid-area y luego los he distribuido en la cuadrícula usando grid-template-areas. Esto crea dos áreas de cuadrícula, una que cubre cuatro celdas de la cuadrícula, la otra dos.

* {box-sizing: border-box;}

.wrapper {
    border: 2px solid #f76707;
    border-radius: 5px;
    background-color: #fff4e6;
}

.wrapper > div {
    border: 2px solid #ffa94d;
    border-radius: 5px;
    background-color: #ffd8a8;
    padding: 1em;
    color: #d9480f;
}

.wrapper {
  display: grid;
  grid-template-columns: repeat(3,1fr);
  grid-template-rows: 100px 100px;
  grid-template-areas:
    "a a b"
    "a a b";
}
.item1 {
  grid-area: a;
}
.item2 {
  grid-area: b;
}

<div class="wrapper">
   <div class="item1">Item</div>
   <div class="item2">Item</div>
</div>
