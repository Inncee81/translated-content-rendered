WebSocket.onclose özelliği bağlantının readyState özelliği CLOSED olduğunda çağırılan bir EventHandler'dır. CloseEvent ile birlikte çağırılır.

Sözdizim

aWebSocket.onclose = function(event) {
  console.log("WebSocket is closed now.");
};

Değer

Bir EventListener.
