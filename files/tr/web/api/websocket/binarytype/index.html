WebSocket.binaryType özelliği, bağlantıda iletilen verinin ikilik veri tipini verir.

Sözdizim

var binaryType = aWebSocket.binaryType;

Değer

Bir DOMString:

"blob"
Eğer Blob objeleri kullanıldıysa.
"arraybuffer"
Eğer ArrayBuffer objeleri kullanıldıysa.
