Comment () konstruktoru yeni yaradılmış Comment obyektini mətn məzmunu olaraq parametrdə verilən isteğe bağlı DOMString qaytarır.

Syntax

comment1 = new Comment(); // Create an empty comment
comment2 = new Comment("This is a comment");

ax

comment1 = new Comment(); // Create an empty comment

Example

var comment = new Comment("Test");
