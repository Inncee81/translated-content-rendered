Опис

Псевдо-клас :root вказує на кореневий елемент дерева документа. Якщо застосувати його до HTML, :root відображає <html> і є ідентичним селектору html, та не зважаючи на це, його вага є більшою.

Синтаксис

:root

Приклад

Використання :root може бути корисним при декларуванні глобальних змінних CSS

:root {
  --main-color: hotpink;
  --pane-padding: 5px 42px;
}
