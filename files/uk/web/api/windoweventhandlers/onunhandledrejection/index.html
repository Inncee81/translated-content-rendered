Властивість onunhandledrejection домішки WindowEventHandlers - це EventHandler для обробки подій unhandledrejection. Ці події виникають для необроблених відхилень об'єктів Promise.

Синтаксис

window.onunhandledrejection = function;

Значення

function - це EventHandler або функція, яку треба викликати, коли вікно отримує події unhandledrejection. Обробник подій отримує вхідний параметр як PromiseRejectionEvent.

Приклади

Цей приклад просто виводить у консоль значення reason (причина) необроблених відхилень.

window.onunhandledrejection = function(e) {
  console.log(e.reason);
}
