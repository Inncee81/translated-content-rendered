Властивість onrejectionhandled домішки WindowEventHandlers - це EventHandler для обробки подій rejectionhandled. Ці події виникають, коли відхиляються об'єкти Promise.

Синтаксис

window.addEventListener("rejectionhandled", function(event) { ... });
window.onrejectionhandled = function(event) { ...};

Приклад

window.onrejectionhandled = function(e) {
  console.log(e.reason);
}
