Ласкаво просимо! Відвідавши цю сторінку, ви зробили перший крок до того, щоб стати учасником MDN!

Наведені тут вказівки охоплюють всі питання щодо участі в творенні MDN, зокрема, настанови зі стилю, посібники з використання нашого редактора та інструментів тощо. Будь ласка, зверніть увагу, що перш ніж редагувати або створювати будь-які сторінки, Ви маєте ознайомитись та погодитись з умовами Mozilla.

Якщо ви раніше не допомагали MDN, зверніться спершу до статті «З чого почати на MDN» — вона допоможе Вам з вибором першого завдання і його виконанням.

MDN tools
MDN offers a number of features that make it easier to track progress, manage content, and keep up with the latest changes to the site.
Документові структури
На сторінках MDN можна знайти чимало різних повторюваних структур, що призначені для послідовного й упорядкованого викладення інформації у статтях MDN. Тут наведено статті, що описують ці структури, тож ви, як автор MDN, можете їх впізнавати, вживати й змінювати відповідним чином для документів, які ви створюєте, редагуєте чи перекладаєте.
Локалізація MDN
Локалізація на MDN охоплює три складові, кожна з яких потребує окремих засобів та привілеїв доступу.
Покрокові путівники
Кожна з наведених статей являє собою покроковий путівник для досягнення певної мети пов'язаної із внеском до MDN.
Посібник по інтерфейсу користувача редактора MDN
Редактор WYSIWYG (what-you-see-is-what-you-get — що ти бачиш те ти отримаєш) для вікі MDN Web Docs дозволяє легко створювати новий вміст. У цьому посібнику показано, як використовувати редактор і підвищити вашу продуктивність. Будь ласка, прочитайте та дотримуйтесь умов Mozilla, перш ніж редагувати або створювати нові сторінки.
Путівники
Перелічені тут путівники наводять докладні відомості щодо того, як слід писати й оформлювати документацію MDN, як мають виглядати зразки коду та інший вміст.
