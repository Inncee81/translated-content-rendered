Властивість у JavaScript - це характеристика об'єкта, що часто описує атрибути, пов'язані зі структурою даних.

Існує два види властивостей: 

  • Властивості екземпляра містять дані, специфічні для даного екземпляра об'єкта. 
  • Статичні властивості містять дані, спільні для усіх екземплярів об'єкта.

У властивості є ім'я (рядок) та значення (проста величина, метод, або посилання на об'єкт). Зауважте, що, коли ми кажемо "властивість містить об'єкт", це скорочення від "властивість містить посилання на об'єкт". Ця відмінність важлива, тому що сам об'єкт лишається незмінним, коли ви змінюєте значення властивості.
