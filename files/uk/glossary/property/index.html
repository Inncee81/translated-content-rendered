Термін властивість може мати декілька значень в залежності від контексту. Він може означати:

Властивість (CSS)
Властивість CSS - це характеристика (як колір), чия величина визначає один аспект того, як веб-переглядач має відображати елемент.
Властивість (JavaScript)
Властивість у JavaScript - це характеристика об'єкта, що часто описує атрибути, пов'язані зі структурою даних.
