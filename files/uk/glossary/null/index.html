У комп'ютерних науках величина null представляє собою посилання, що вказує, як правило, навмисно, на неіснуючий або недійсний об'єкт чи адресу. Значення null-посилання різниться у реалізаціях різних мов.

У мові JavaScript null є однією з простих величин.
