AJAX (Asynchronous JavaScript And XML) є практикою програмування з комбінування HTML, CSS, JavaScript, моделі DOM, і XMLHttpRequest об'єкт для побудови більш складних та динамічних сторінок. AJAX дозволяє оновлювати лише частини веб-сторінок замість того, щоб перезавантажувати сторінку повністю. Також AJAX дає можливість працювати асинхронно: JavaScript продовжує працювати, поки частина сторінки намагається оновитися (на відміну від синхронного способу, який заблокує виконання будь-якого коду до закінчення перезавантаження сторінки).
