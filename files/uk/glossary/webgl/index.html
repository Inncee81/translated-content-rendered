WebGL (Web Graphics Library) - це JavaScript API, який відображає інтерактивну 2D і 3D графіку.

Khronos Group підтримує WebGL, який базується на OpenGL ES 2.0.

Ви можете викликати WebGL в межах елементу HTML <canvas>, який слугує, як поверхня відображення.

Всі основні веб-переглядачі зараз підтримують WebGL, але його підтримка також залежить від зовнішніх факторів (наприклад, підтримка графічним процесором пристрою).
