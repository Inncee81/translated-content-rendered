HTML (HyperText Markup Language, мова розмітки гіпертексту) — це декларативна мова, яка визначає структуру веб-сторінки.

Історія

У 1990, як частину його бачення Web, Тім Бернерс-Лі описав концепцію гіпертексту, яку формалізував наступного року через розмітку, що була основана головним чином на SGML. Спільнота IETF почала створювати специфікацію HTML у 1993, та після декількох чорнеток випустила версію 2.0 у 1995. У 1994 Бернерс-Лі створив організацію W3C для розробки Web. У 1996, W3C взяла на себе розробку стандарта HTML та випустила рекомендації HTML 3.2 роком пізніше. HTML 4.0 було випущено у 1999 році, який набув стандарту ISO у 2000.

У ті часи W3C майже покинула HTML на користь XHTML, закликаючи до утворення незалежної групи WHATWG in 2004. Завдяки WHATWG, роботу над стандартом HTML5 було продовжено: дві організації випустили першу чернетку у 2008 році та фіналізували стандарт у 2014.

Концепція та синтаксис

HTML-документ це текстовий документ, що складається з елементів. Сам елемент зазвичай складається з початкового та кінцевого теґа, втім є елементи (як-от <img>), яким не дозволено мати вміст — такі елементи або складаються з одного-єдиного теґа («поодинокого» або «самозакривного»), або кінцевий теґ розташовано одразу за початковим (без жодного вмісту між ними). Кожен теґ починається та закінчується кутовими дужками.

До теґів можна додавати атрибути, що передають додаткову інформацію, яка впливає на поведінку чи відображення елемента переглядачем:

Зазвичай HTML зберегається у файлах із розширенням .htm чи .html, обслуговується (віддається) веб-сервером, та може переглядатися будь-яким веб-переглядачем.
