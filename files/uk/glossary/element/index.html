Елемент - частина веб-сторінки. В XML та HTML, елемент може містити данні або фрагмент тексту або зображення, або можливо нічого. Типовий елемент включає в себе відкриваючий тег з деякими атрибутами, вкладений текстовий вміст і закриваючий тег.

Елементи та теги не одне й те саме. Теги починаються або закінчуються елементом у вихідному коді, тоді як елементи є частиною DOM, моделлю документа для відображення сторінки в browser.
