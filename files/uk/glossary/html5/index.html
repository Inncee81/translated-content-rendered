Останній стабільний реліз HTML, HTML5 перетворює HTML з простої мови для структурування документа на платформу для розробки повноцінних додатків. Серед інших функцій, HTML5 включає в себе нові елементи і  JavaScript APIs для розширення можливостей сховища, мультимедіа та апаратного забезпечення .
