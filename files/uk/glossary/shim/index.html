Шим (shim - прошарок, англ.) - це код, що використовується для виправлення поведінки кода, який вже існує, зазвичай, додаванням нового API, який дозволяє обійти проблему. Це відрізняє його від поліфіла, що впроваджує новий API, який не підтримується переглядачем початково.
