Scalable Vector Graphics (SVG, масштабована векторна графіка) — це формат для двовимірної векторної графіки, що базується на синтаксисі XML. Група W3C почала працювати над SVG наприкінці дев'яностих, але SVG здобув популярність лише коли було випущено Internet Explorer 9 із підтримкою SVG. Всі головні браузери зараз підтримують SVG.

Завтяки тому, що SVG базується на синтаксисі XML, його можна стилізувати за допомогою CSS та надавати йому інтерактивності, використовуючи JavaScript. HTML5 зараз дозволяє напряму включати SVG теги у документ HTML.

Так як SVG — це векторний формат графіки, те, що він може маштабуватися без втрати даних, робить його неоціненним у адаптивному веб-дизайні, тому що він дає можливість створювати елементи інтерфейсу, що можуть маштабуватися для будь-якого розміру екрана. Також SVG включає в себе корисні інструменти, такі як відсікання, маскування, фільтри та анімації.
