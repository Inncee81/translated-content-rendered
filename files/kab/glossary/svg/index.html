Scalable Vector Graphics (SVG) d amasal n tugna tamawayt 2D yezider ɣef taseddast  XML .

A W3C ibda leqdic innes ɣer taggara iseggasen 1990,maca SVG yuɣal yettwasen ass-mi Internet Explorer 9  yefeɣ-ed s tannalt SVG.meṛṛa igejdanen iminigen ttannalin tura SVG.

Azider  ɣef yiwet n taseddast XML, SVG yezmer ad yettuɣanib akked CSS u ad nesnulfu s leqdic JavaScript. HTML5 yettaǧǧayaɣ tura aseddu srid n SVG tags deg yiwen zzmamHTML.

akka s amasal n tugna tamawayt,udlifen SVG zemren ad mgaraden xila, Dizayn udmir, acku tzemreḍ ad snulfuḍ iferdisen n agrudem ed udlifen ig msasan deg yal tiddi n wagdil.SVG yettmuddud daɣen aṭas n ifecka i wulmen,am tawaksa,aggelmes,tistayin ed imrayen.
