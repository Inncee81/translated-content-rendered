Der Begriff Block hat mehrere, kontextabhängige Bedeutungen, darunter:

Block (CSS)
Ein Block auf einer Website ist ein HTML-element, das in einer neuen Zeile angezeigt wird, also unter dem vorherigen Element und über dem folgenden (gemeinhin bekannt als Blockelement). Beispielsweise ist ein <p> standardmäßig ein Blockelement, während ein <a> ein eingebundenes Element (engl. "inline element") ist — mehrere Links können so nebeneinander im HTML-Quelltext stehen und dann in der gerenderten Ausgabe auf derselben Zeile liegen.
Block (Scripting)
In JavaScript ist ein Block eine Sammlung zueinander gehöriger statements, eingeschlossen von geschwungenen Klammern ("{}"). Zum Beispiel können Sie einen Block mit Anweisungen nach einem if (condition)-Block setzen, wodurch Sie deutlich machen, dass der Interpreter den Code innerhalb dieses Blocks ausführen soll, wenn die Bedingung zu true evaluiert, oder ihn überspringen soll, wenn die Bedingung false ergibt.
