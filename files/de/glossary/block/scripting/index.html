In JavaScript ist ein Block eine Sammlung zueinander gehöriger Anweisungen, die von geschweiften Klammern ("{}") eingeschlossen sind. Zum Beispiel können Sie einen Block mit Anweisungen nach einem if (condition)-Block setzen, wodurch Sie deutlich machen, dass der Interpreter den Code innerhalb dieses Blocks ausführen soll, wenn die Bedingung zu true evaluiert, oder ihn überspringen soll, wenn die Bedingung false ergibt.
