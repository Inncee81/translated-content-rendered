Ein URI (Uniform Resource Identifier, deutsch: einheitlicher Bezeichner für Ressourcen) ist ein Textstring, der auf eine Ressource verweist. Die verbreitetsten sind URLs, die auf eine Ressource verweisen, indem sie ihren Ort im Internet angibt. Im Gegensatz dazu beziehen sich URNs auf Ressourcen mit einem Namen, der in einem bestimmten Namensraum gültig ist, wie die ISBN eines Buches.
