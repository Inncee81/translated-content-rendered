Hyperlinks verbinden Webseiten oder Datenelemente miteinander. In HTML definieren <a>-Elemente Hyperlinks von einem Punkt auf einer Webseite (wie einer Zeichenkette oder einem Bild) zu einem anderen Punkt auf einer ander Webseite (oder sogar auf der gleichen Seite).
