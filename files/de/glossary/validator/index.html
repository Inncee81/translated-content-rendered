Bei einem Validator handelt es sich um ein Programm, das Code auf Syntaxfehler hin prüft. Validatoren können für beliebige Formate und Programmiersprachen erstellt werden, wobei wir uns in unserem Kontext auf Tools beziehen, die HTML, CSS, and XML untersuchen.
