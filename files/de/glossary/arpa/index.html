--- title: ARPA slug: Glossary/ARPA translation_of: Glossary/ARPA ---

.arpa (address and routing parameter area) ist eine Top-Level-Domain, welche für Internetinfrastrukturzwecke, speziell Reverse DNS Lookup, verwendet wird (d.h., finden des Domain-Namens für eine gegebene IP Adresse).
