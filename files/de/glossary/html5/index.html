Die neueste stabile Version von HTML, HTML5 bringt HTML von einer einfachen Auszeichnungssprache zum Strukturieren eines Dokuments zu einer vollständigen App-Entwicklungsplattform. HTML5 enthält unter anderem neue Elemente und neue JavaScript APIs, um den Zugriff auf Speicher, Multimedia und Hardware zu verbessern.
