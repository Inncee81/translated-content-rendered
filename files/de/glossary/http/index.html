HTTP (HyperText Transfer Protocol) ist das Standard Protokoll, dass es ermöglicht, Dateien über das Web zu übertragen. HTTP ist textlich (die gesamte Kouumunikation geschieht in einfachem Text) und zustandslos (Informationen aus vorheriger Kommunikation gehen verloren).
