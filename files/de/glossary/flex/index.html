flex ist ein neuer, zur CSS-display-Eigenschaft hinzugefügter Wert. Wie auch inline-flex macht er das Element, auf das er angewendet wird, zu einem flex container und jedes Kind des Elements wird zu einem flex item. Die Elemente sind dann Teil des Flex-Layouts und alle der Eigenschaften, die im „CSS Flexible Box Layout Module“ definiert sind, können angewandt werden.

Die flex-Eigenschaft ist eine Kurzform der Flexbox-Eigenschaften flex-grow, flex-shrink und flex-basis.

Zusätzlich kann <flex> auf eine flexible Länge im CSS-Grid-Layout verweisen.
