WebRTC (Web Real-Time Communication (übersetzt: Web Echtzeit Kommunikation)) ist eine API, die zum Erstellen von Video-Chat-, Audiogespräch- und P2P-Dateiaustausch
Web-Anwendungen benutzt wird.

WebRTC besteht hauptsächtlich aus folgenden Teilen:

getUserMedia()
Gewährt Zugang zu einer Kamera und/oder Mikrofon des Geräts, dessen Signale an eine RTC-Verbindung übergeben werden können.
RTCPeerConnection
Eine Schnittstelle zum Konfigurieren von Video- oder Audio-Gespächen.
RTCDataChannel
Stellt eine Methode zum Erstellen von P2P Daten Kanälen zwischen Browsern bereit.
