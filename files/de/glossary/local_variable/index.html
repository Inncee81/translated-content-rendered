--- title: Local variable slug: Glossary/Local_variable tags: - Gültigkeitsbereich - Locale variable - Lokale Variable translation_of: Glossary/Local_variable ---

Eine variable deren Namen an ihren  value – Wert – nur innerhalb eines local scope – lokalen Gültigkeitsbereichs – gebunden ist.

Example

var global = 5; //ist eine globale Variable

function fun(){
    var local = 10; //ist eine lokale Variable
}
