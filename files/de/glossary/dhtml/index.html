DHTML (Dynamisches HTML) bezieht sich auf den Code in interaktiven Webseiten, die keine Plugins benötigen wie  Flash oder Java. DHTML verbindet die gemeinsame Funktionalität von HTML, CSS, dem DOM, und JavaScript.
