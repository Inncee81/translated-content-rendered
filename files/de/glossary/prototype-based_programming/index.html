Prototypbasierte Programmierung ist eine Variante der objektorientierten Programmierung, bei der keine Klassen verwendet werden. Objekte werden durch kopieren von bereits existierenden Objekten (Prototypen) erzeugt. Beim Kopieren werden alle Eigenschaften des Prototypen übernommen, diese können jedoch verändert und/oder ergänzt werden.

Prototypbasierte Programmierung: neues Objekt =  Kopie von existierendem Objekt  objektorientierten Programmierung: neues Objekt = Instanz einer Klasse
