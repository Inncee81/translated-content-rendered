HTTPS (HTTP Secure) ist eine verschlüsselte Version des HTTP-Protokolls. Meistens nutzt es SSL oder TLS, um die gesamte Kommunikation zwischen einem Client und einem Server zu verschlüsseln. Diese sichere Verbindung erlaubt Clients, sicher sensible Daten mit dem Server auszutauschen, zum Beispiel Banking-Aktivitäten oder Online-Shopping.
