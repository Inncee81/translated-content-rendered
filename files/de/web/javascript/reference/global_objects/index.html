Dieses Kapitel dokumentiert alle Objekte, die standardmäßig in einer JavaScript-Umgebung zur Verfügung stehen.

Der Ausdruck "Globales Objekt" (oder Standardobjekt) ist nicht zu verwechseln mit dem global object. Hier sind globale Objekte Objekte, die zu Objekten im globalen Zugriffsbereich (scope) referenzieren. Das global object selbst wird mit Hilfe des this-Operators im globalen Bereich angesprochen (sofern der Strict-Modus von ECMAScript 5 nicht verwendet wird, andernfalls sind sie undefined). Tatsächlich besteht der globale Bereich aus den Eigenschaften (properties) des global object (einschließlich geerbter Eigenschaften, falls vorhanden).

Andere Objekte im globalen Bereich wurden entweder vom Benutzerscript erstellt oder von der ausführenden Anwendung (meist der Browser) bereitgestellt. Die Anwendungsobjekte, welche im Browserumfeld (DOM) verfügbar sind, werden in der API-Referenz dokumentiert. Eine detailiertere Unterscheidung zwischen DOM- und Kern-JavaScript befindet sich im Kapitel Überblick JavaScript-Technologien.
