Die flags Eigenschaft gibt einen String mit den Flags des RegExp-Objektes zurück.

Der Code für dieses interaktive Beispiel ist in einem GitHub Repository gespeichert. Wenn du etwas zum Interaktiven-Beispiel-Project beitragen möchtest, klone bitte https://github.com/mdn/interactive-examples und sende uns Pull-Requests.

Eigenschaftsattribute von RegExp.prototype.flags
Schreibbar nein
Aufzählbar nein
Konfigurierbar ja

Beschreibung

Flags in der flags Eigenschaft sind alphabetisch sortiert (von links nach rechts, z.B. "gimuy").

Beispiele

Die flags benutzen

/foo/ig.flags;   // "gi"
/bar/myu.flags;  // "muy"

Polyfill

if (RegExp.prototype.flags === undefined) {
  Object.defineProperty(RegExp.prototype, 'flags', {
    configurable: true,
    get: function() {
      return this.toString().match(/[gimuy]*$/)[0];
    }
  });
}
