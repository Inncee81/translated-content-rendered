Beschreibung

Die valueOf() Methode gibt den primitiven Wert eines Date Objektes als Nummern-Datentyp (number) zurÃ¼ck. Diese Nummer entspricht der Anzahl der Millisekunden seit dem 1. Januar 1970 UTC.

Diese Methode hat den gleiche Funktionsumfang wie die Date.prototype.getTime() Methode.

Diese Methode wird intern von JavaScript aufgerufen und muss nicht explizit im Quelltext verwendet werden.

Beispiele

Einsatz von valueOf()

var myVar = x.valueOf();      // schreibt -424713600000 auf myVar
