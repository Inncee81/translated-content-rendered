Die valueOf() Methode gibt einen primitiven Wert des Boolean Objektes zurück.

The source for this interactive example is stored in a GitHub repository. If you'd like to contribute to the interactive examples project, please clone https://github.com/mdn/interactive-examples and send us a pull request.

Syntax

bool.valueOf()

Rückgabewert

Den primitiven Wert des gegebenen Boolean Objektes.

Beschreibung

Die valueOf Methode des Boolean Objektes gibt einen primitiven Wert eines Boolean Objekts oder Boolean Literals als Boolean Datentyp zurück.

Diese Methode wird intern von JavaScript aufgerufen und wird nicht in normalen Anwendungen benutzt.

Beispiele

Einsatz von valueOf

x = new Boolean();
myVar = x.valueOf(); // weist  myVar false zu
