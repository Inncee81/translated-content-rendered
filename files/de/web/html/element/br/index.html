Zusammenfassung

Das HTML Element line break <br> produziert einen Zeilenumbruch in (Fließ-)Text (Wagenrücklauf). Es ist z.B nützlich beim Notieren eines Gedichtes oder einer Adresse, wo die Aufteilung der Zeilen von Bedeutung ist.

Gebrauch

Zulässiger Inhalt Keiner, es handelt sich um ein "void" Element.
Tag Platzierung Muss ein Start-Tag und darf kein Ende-Tag besitzen. In XHTML Dokumenten wird das Element als <br/> notiert.
Zulässige Elternelemente Alle Elemente die formulierten Inhalt akzeptieren.
Normatives Dokument HTML5, section 4.6.23; HTML 4.01, section 9.3.2

Attribute

Wie alle anderen HTML Elemente besitzt dieses Element die globalen Attribute.

clear Unerwünscht HTML4.01 Veraltet seit HTML5
Zeigt an, wo die nächste Zeile nach dem Zeilenumbruch beginnt.

Gebrauchshinweis: Dieses Attribut ist veraltet in HTML5 und sollte nicht verwendet werden. Stattdessen sollte die CSS-Eigenschaft clear verwendet werden.

Beispiel

Mozilla Foundation<br>
1981 Landings Drive<br>
Building K<br>
Mountain View, CA 94043-0801<br>
USA

Ausgabe:

Mozilla Foundation
1981 Landings Drive
Building K
Mountain View, CA 94043-0801
USA

DOM_Interface

  • HTMLBRElement

Anmerkungen

<br> nicht verwenden um die Lücken zwischen Text zu vergrößern; stattdessen sollte die CSS margin Eigenschaft oder das <p> Element verwendet werden.
