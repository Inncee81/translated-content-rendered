Unerwünscht
Dieses Feature wurde aus den Webstandards entfernt. Obwohl manche Browser es immer noch unterstützen, wird es zukünftig wegfallen. Es sollte daher nicht mehr in alten oder neuen Projekten verwendet werden. Webseiten oder Webapps, die es benutzen, funktionieren möglicherweise irgendwann nicht mehr korrekt.

Übersicht

Das HTML Center Element (<center>) ist ein Block-Level Element, welches Paragraphen oder andere Block-Level Elemente und Inline Elemente enthalten kann. Der gesamte Inhalt des Elements wird horizontal zentriert, innerhalb des übergeordneten Elements (normalerweise das <body> Element). Dieses Element ist in HTML 4 (und XHTML 1) veraltet und wird nicht mehr angewendet, zu Gunsten der CSS text-align Eigenschaft, welche auf <div> Elemente oder individuell auf <p> Elemente angewendet werden kann.

DOM Schnittstelle

Dieses Element implementiert die HTMLElement Schnittstelle.

Implementierungs Anmerkung: Bis einschließlich Gecko 1.9.2, implementiert Firefox die HTMLSpanElement Schnittstelle für dieses Element.

Beispiel 1

<center>Dieser Text wird zentriert.
<p>Ebenso dieser Paragraph.</p></center>

Beispiel 2 (CSS Alternative)

<div style="text-align:center">Dieser Text wird zentriert.
<p>Ebenso dieser Paragraph.</p></div>

Beispiel 3 (CSS alternative)

<p style="text-align:center">Dieser Text wird zentriert.<br>
Ebenso dieser Paragraph.</p>

Anmerkung

Bei Zuweisung der text-align:center Eigenschaft zu einem <div> oder <p> Element, wird der Inhalt des Elements zentriert, während dessen Größe unverändert bleibt.
