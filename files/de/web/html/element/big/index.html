Veraltet
Dieses Feature ist veraltet. Obwohl es in manchen Browsern immer noch funktioniert, wird von seiner Benutzung abgeraten, da es jederzeit entfernt werden kann. Es sollte daher nicht mehr verwendet werden.

Übersicht

Das HTML Big Element (<big>) vergrößert die Schrift um einen Wert (zum Beispiel von small nach medium oder von large nach x-large) bis zur maximalen Schriftgröße des Browsers.

Hinweis zur Benutzung: Dadurch, dass dieses Element rein der Darstellung diente, wurde es in HTML5 entfernt und sollte nicht mehr verwendet werden. Stattdessen sollten Webentwickler die CSS Eigenschaften nutzen.

Attribute

Dieses Element hat nicht mehr als die globalen Attribute, wie jedes Element.

Beispiel 1

<p>
  Das ist der erste Satz. <big>Der zweite
  Satz wird größer dargestellt.</big>
</p>

Beispiel 2 (CSS Alternative)

<p>
  Das ist der erste Satz. <span style="font-size:1.2em">Der zweite
  Satz wird größer dargestellt.</span>
</p>

Ergebnis

Das ist der erste Satz. Der zweite Satz wird größer dargestellt.

DOM Schnittstelle

Dieses Element implementiert die HTMLElement Schnittstelle.

Hinweis zur Implementierung: Bis einschließlich Gecko 1.9.2 implementierte Firefox die HTMLSpanElement Schnittstelle für dieses Element.
