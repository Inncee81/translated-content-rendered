Zusammenfassung

Das HTML <body> Element repräsentiert den Inhalt eines HTML Dokuments. Es gibt nur ein <body> Element in einem Dokument.

Content categories Sectioning root.
Erlaubter Inhalt Flow content.
Tag Platzierung Der Starttag darf weggelassen werden, wenn das erste Ding dazwischen kein Leerzeichen, Kommentar, <script> Element oder <style> Element ist. Der Endtag kann weggelassen werden, wenn das Body-Element Inhalte oder einen Starttag hat und ihm nicht sofort ein Kommentar folgt.
Zulässige Elternelemente Es muss das zweite Element eines <html> Elements sein.
DOM Schnittstelle HTMLBodyElement
  • Das <body> Element implementiert die HTMLBodyElement Schnittstelle.
  • Sie können auf das Body-Element durch ein document.body Attribute zugreifen.

Attribute

Dieses Element beinhaltet die Globalen Attribute.

alink
Textfarbe für selektierte Links. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS color Objekt in Verbindung mit der :active Pseudoklasse.
background
URI eines Bilds, das als Hintergrundbild gesetzt wird. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS background Objekt auf dem Element.
Hinweis: Seit Gecko 7.0 (Firefox 7.0 / Thunderbird 7.0 / SeaMonkey 2.4) wird background nicht länger als eine URI aufgelöst; stattdessen wird es als einfacher String behandelt.
bgcolor
Hintergrundfarbe eines Dokumentes. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS background-color Objekt auf dem Element.
bottommargin
Der Abstand des Bodens des Bodys. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS margin-bottom Objekt auf dem Element.
leftmargin
Der Abstand der linken Seite des Bodys. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS margin-left Objekt auf dem Element.
link
Textfarbe eines unbesuchten Links. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS color Objekt in Verbindung mit der :link Pseudoklasse.
onafterprint HTML5
Funktion, die aufgerufen wird, wenn der Nutzer das Dokument gedruckt hat.
onbeforeprint HTML5
Funktion, die aufgerufen wird, wenn der Nutzer das Dokument drucken möchte.
onbeforeunload HTML5
Funktion, die aufgerufen wird, wenn das Dokument nicht geladen wird.
onblur HTML5
Funktion, die aufgerufen wird, wenn das Dokument nicht mehr angezeigt wird.
onerror HTML5
Funktion, die aufgerufen wird, wenn das Dokument nicht richtig geladen wird.
onfocus HTML5
Funktion, die aufgerufen wird, wenn das Dokument angezigt wird.
onhashchange HTML5
Funktion, die aufgerufen wird, wenn der Fragment-Identifizier-Teil (beginnend mit der Raute ('#')) der eigentlichen Adresse des Dokuments geändert wird.
onlanguagechange
Funktion, die aufgerufen wird, wenn die bevorzugte Sprache geändert wird.
onload HTML5
Funktion, die aufgerufen wird, wenn das Dokument fertig geldaden ist.
onmessage HTML5
Funktion, die aufgerufen wird, wenn das Dokument eine Nachricht erhalten hat.
onoffline HTML5
Funktion, die aufgerufen wird, wenn die Netzwerk-Kommunikation gescheitert ist.
ononline HTML5
Funktion, die aufgerufen wird, wenn die Netzwerk-Kommunikation wiederhergestellt wird.
onpopstate HTML5
Funktion, die aufgerufen wird, wenn der Nutzer den Sitzungsverlauf aufgerufen hat.
onredo HTML5
Funktion, die aufgerufen wird, wenn der Nutzer vorwärts eine Rückgängig-Aktion wiederholt hat.
onresize HTML5
Funktion, die aufgerufen wird, wenn die Größe des Dokumentes verändert wird.
onstorage HTML5
Funktion, die aufgerufen wird, wenn der Speicherbreich verändert wurde.
onundo HTML5
Funktion, die aufgerufen wird, wenn der Nutzer zurückgegangen ist.
onunload HTML5
Funktion, die aufgerufen wird, wenn das Dokument verschwindet.
rightmargin
Der Abstand der rehten Seite des Bodys. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS margin-right Objekt auf dem Element.
text
Vordergrundfarbe des Textes. TDiese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS color Objekt auf dem Element.
topmargin
Der Abstand von oben des Bodys. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS margin-top Objekt auf dem Element.
vlink
Textfarbe eines besuchten Links. Diese Methode ist fehlerhaft, benutzen Sie stattdessen das CSS color Objekt in Verbindung mit der :visited Pseudoklasse.

Spezifikation Status Anmerkung
HTML Living Standard
Die Definition von '<body>' in dieser Spezifikation.		 Lebender Standard Veränderte die Liste der fehlerhaften Features.
HTML5
Die Definition von '<body>' in dieser Spezifikation.		 Empfehlung Überholt die ehemaligen veralteten Attribute. Definiert das Verhalten der fehlerhaften und niemals standartisierten margintop, marginleft, marginright und marginbottom.
HTML 4.01 Specification
Die Definition von '<body>' in dieser Spezifikation.		 Empfehlung Überholt die alink, background, bgcolor, link, text und vlink Attribute.
