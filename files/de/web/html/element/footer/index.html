Übersicht

Das HTML <footer> Element repräsentiert eine Fußzeile für den nahestehendsten Sektionsinhalt oder das Sektionsquellen Element (also das nahestehendste Element wie zum Beispiel <article>, <aside>, <nav>, <section>, <blockquote>, <body>, <details>, <fieldset>, <figure>, <td>). Eine Fußzeile enthält typischerweise Informationen über den Autor der Sektion, die Copyright-Daten oder Links zu verwandten Dokumente.

Hinweis zur Benutzung:

  • Informationen über den Autor sollten in ein <address> Element eingebettet werden, welches wiederum in einem <footer> Element stehen kann.
  • Das <footer> Element ist kein Sektionsinhalt und führt deshalb auch keine neue Sektion in der Begrenzung ein.
Inhaltskategorien Fließender Inhalt, greifbarer Inhalt.
Erlaubter Inhalt Fließender Inhalt, jedoch kein <footer> oder <header> Nachfahre.
Tag Auslassung Keine, sowohl das Anfangs- als auch das End-Tag sind erforderlich.
Erlaubte Elternelemente Jedes Element, dass fließenden Inhalt erlaubt. Das <footer> Element muss kein Nachfahre eines <address>, <header> oder anderem <footer> Element sein.
DOM Schnittstelle HTMLElement

Attribute

Dieses Element schließt nur globale Attribute ein.

Beispiel

<footer>
  Hier ein paar Copyright Informationen oder stattdessen Informationen über den &lt;Artikel&gt;?
</footer>
