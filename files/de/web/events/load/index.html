--- title: load slug: Web/Events/load translation_of: Web/API/Window/load_event ---

Das load Ereignis wird ausgelöst, sobald eine Ressource und die von ihr abhängigen Ressourcen das Laden beendet haben.

General info

Spezifikation
DOM L3
Schnittstelle
UIEvent
bubbles
Nein
cancelable
Nein
Ziel
Window,Document,Element
Default Action
None.

Properties

Property Type Description
target Schreibgeschützt EventTarget The event target (the topmost target in the DOM tree).
type Schreibgeschützt DOMString The type of event.
bubbles Schreibgeschützt Boolean Whether the event normally bubbles or not.
cancelable Schreibgeschützt Boolean Whether the event is cancellable or not.
view Schreibgeschützt WindowProxy document.defaultView (window of the document)
detail Schreibgeschützt long (float) 0.

Example

<script>
  window.addEventListener("load", function(event) {
    console.log("Alle Ressourcen haben das Laden beendet!");
  });
</script>
