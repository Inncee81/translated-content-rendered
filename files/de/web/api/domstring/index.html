DOMString ist ein UTF-16 String. Obwohl JavaScript bereits solche Strings nutzt wird DOMString direkt zu einem String gemappt.

Wird null in einem Parameter oder einer Methode übergeben, so wandelt DOMString dies in den String "null" um.
