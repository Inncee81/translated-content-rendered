Das AesCbcParams Verzeichnis (dictionary) der Web Crypto API wird als algorithm Parameter an die Funktionen SubtleCrypto.encrypt(), SubtleCrypto.decrypt(), SubtleCrypto.wrapKey() und SubtleCrypto.unwrapKey() übergeben, wenn der Algorithmus AES-CBC verwendet wird.

Eigenschaften

name
A DOMString. Der Wert sollte auf AES-CBC gesetzt werden.
iv
Ein BufferSource. Der Initialisierungsvektor. Er muss 16 Bytes lang sein und sollte unvorhersehbar und am besten kryptografisch zufällig sein. Er muss aber nicht geheim sein, er kann unverschlüsselt mit dem Kryptogram übertragen werden.

Browser Kompatibilität

Browser mit Unterstützung für den "AES-CBC" Algorithmus werden die Methoden SubtleCrypto.encrypt(), SubtleCrypto.decrypt(), SubtleCrypto.wrapKey() und SubtleCrypto.unwrapKey() für diesen Typ unterstützen.
