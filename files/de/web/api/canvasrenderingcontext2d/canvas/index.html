Die CanvasRenderingContext2D.canvas Eigenschaft, Teil der Canvas API, ist eine Nurlese-Referenz auf das HTMLCanvasElement Objekt, das mit einem gegebenen Kontext assoziiert ist. Sie kann null sein, wenn kein assoziiertes <canvas> Element existiert.

Syntax

ctx.canvas;

Beispiele

Wenn dieses <canvas> Element gegeben ist:

<canvas id="canvas"></canvas>

... kannst du eine Referenz auf das Canvas-Element innerhalb von CanvasRenderingContext2D herstellen, indem du die canvas Eigenschaft verwendest.

var canvas = document.getElementById('canvas');
var ctx = canvas.getContext('2d');
ctx.canvas // HTMLCanvasElement
