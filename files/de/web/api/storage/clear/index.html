Die Methode clear() der Storage-Schnittstelle löscht sämtliche Datensätze des Objekts.

Syntax

storage.clear();

Rückgabewert

undefined.

Beispiele

Die folgende Funktion speichert drei Datensätze und löscht sie abschließend mit clear().

function populateStorage() {
  localStorage.setItem('bgcolor', 'red');
  localStorage.setItem('font', 'Helvetica');
  localStorage.setItem('image', 'miGato.png');

  localStorage.clear();
}

Hinweis: Weitere Beispiele sind im Web-Storage-Demo zu finden.
