Übersicht

Die LinkStyle Schnittstelle erlaubt es, das zu einem Knoten zugehöriges CSS Stylesheet anzusprechen.

LinkStyle ist eine abstrakte Schnittstelle und es kann kein Objekt dieses Typs erstellt werden; Sie wird von HTMLLinkElement und HTMLStyleElement Objekten implementiert.

Eigenschaften

Es gibt keine vererbte Eigenschaft.

LinkStyle.sheet Read only
Gibt das StyleSheet Objekt zurück, das zu dem angegebenen Element gehört, oder null, falls keines existiert.

Methoden

Diese Schnittstelle implementiert keine Methoden.
