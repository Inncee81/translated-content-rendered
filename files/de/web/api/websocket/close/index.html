Die Funktion WebSocket.close() beendet eine WebSocket Verbindung oder den Versuch eines Verbindungsaufbaus. Wenn diese schon geschlossen ist, passiert nichts weiter.

Syntax

WebSocket.close();

Parameter

code Optional
Ein numerisch Wert, der den Status angibt, warum die Verbindung geschlossen wird. Wird kein Wert spezifiziert, . Zur Referenz kann diese Liste mit Codes des CloseEvents für gültige Werte genommen werden.
reason Optional
Ein lesbarer Text, welcher erklärt, warum die Verbindung geschlossen wird. Dieser muss UTF-8 kodiert und darf nicht länger als 123 Bytes sein.

Fehler

INVALID_ACCESS_ERR
Ein ungültiger code wurde angegeben.
SYNTAX_ERR
Der reason Text ist zu lang oder enthält ungültige Zeichen.

Note: In Gecko unterstützte diese Funktion bis zu Version 8.0 (Firefox 8.0 / Thunderbird 8.0 / SeaMonkey 2.5) keine Parameter.
