Die Eigenschaft WebSocket.onclose ist ein EventHandler, der aufgerufen wird, wenn die Eigenschaft WebSocket.readyState zu CLOSED geändert bzw. die Verbindung geschlossen wird.

Als Parameter übergeben wird ein CloseEvent.

Syntax

aWebSocket.onclose = function(event) {
  console.log("WebSocket is closed now.");
};

Wert

Ein EventListener.
