Die Eigenschaft WebSocket.binaryType gibt den Typ von Binärdaten zurück, der übertragen wird.

Syntax

var binaryType = aWebSocket.binaryType;

Wert

Ein DOMString:

"blob"
Wenn Blob Objekte benutzt werden.
"arraybuffer"
Wenn ArrayBuffer Objekte benutzt werden.
