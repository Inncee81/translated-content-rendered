Übersicht

CSSPageRule ist eine Schnittstelle, die eine einzelne CSS @page Regel repräsentiert. Sie implementiert die CSSRule Schnittstelle mit dem Typwert 6 (CSSRule.PAGE_RULE).

Syntax

Die Syntax wird im WebIDL Format beschrieben.

interface CSSPageRule : CSSRule {
    attribute DOMString selectorText;
    readonly attribute CSSStyleDeclaration style;
};

Eigenschaften

Als eine CSSRule implementiert CSSPageRule auch die Eigenschaften dieser Schnittstelle. Sie hat die folgenden speziellen Eigenschaften:

CSSPageRule.selectorText
Repräsentiert den Text eines zur At-Regel gehörenden Seitenselektors.
CSSPageRule.style Read only
Gibt den zur At-Regel gehörenden Deklarationsblock zurück.

Methoden

Als eine CSSRule implementiert CSSPageRule auch die Methoden dieser Schnittstelle. Sie hat keine speziellen Methoden.
