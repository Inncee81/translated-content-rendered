Übersicht

Die all CSS Kurzform Eigenschaft setzt alle Eigenschaften mit Ausnahme von unicode-bidi und direction auf ihren ursprünglichen oder vererbten Wert zurück.

InitialwertEs gibt keinen praktischen Initialwert dafür.
Anwendbar aufalle Elemente
VererbtNein
Berechneter Wertwie der angegebene Wert wird er auf alle Eigenschaften angewandt, für die dies eine Kurzschreibweise ist.
Animationstypwie jede der Kurzschreibweisen Eigenschaften (alle Eigenschaften außer unicode-bidi und direction)

Syntax

all: initial;
all: inherit;
all: unset;

Werte

initial
Dieses Schlüsselwort gibt an, dass alle dem Element oder dem Elternelement zugewiesenen Eigenschaften auf deren ursprünglichen Wert geändert werden sollen. unicode-bidi und direction Werte sind davon nicht betroffen.
inherit
Dieses Schlüsselwort gibt an, dass alle dem Element oder dem Elternelement zugewiesenen Eigenschaften auf den Wert des Elternelements geändert werden sollen. unicode-bidi und direction Werte sind davon nicht betroffen.
unset
Dieses Schlüsselwort gibt an, dass alle dem Element oder dem Elternelement zugewiesenen Eigenschaften auf den Wert des Elternelements geändert werden sollen, falls sie vererbbar sind, ansonsten auf ihren ursprünglichen Wert. unicode-bidi und direction Werte sind davon nicht betroffen.

Formale Syntax

initial | inherit | unset | revert

Beispiele

HTML

<blockquote id="quote">Lorem ipsum dolor sit amet.</blockquote> Phasellus eget velit sagittis.

CSS

html {
  font-size: small;
  background-color: #F0F0F0;
  color: blue;
}

blockquote {
  background-color: skyblue;
  color: red;
}

Ergibt:

Keine all Eigenschaft

<blockquote id="quote">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</blockquote> Phasellus eget velit sagittis.
html { font-size: small; background-color: #F0F0F0; color:blue; }
blockquote { background-color: skyblue;  color: red; }

Das <blockquote> Element verwendet die Standarddarstellung des Browsers zusammen mit einer angepassten Hintergrund- und Textfarbe. Es verhält sich auch wie ein Blockelement: Der Text, der ihm folgt, wird unter ihm angezeigt.

all:unset

<blockquote id="quote">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</blockquote> Phasellus eget velit sagittis.
html { font-size: small; background-color: #F0F0F0; color:blue; }
blockquote { background-color: skyblue;  color: red; }
blockquote { all: unset; }

Das <blockquote> Element verwendet nicht die Standarddarstellung des Browsers: Es ist jetzt ein Inlineelement (ursprünglicher Wert), seine background-color Eigenschaft ist transparent (ursprünglicher Wert), aber seine font-size Eigenschaft ist immer noch small (vererbter Wert) und seine color Eigenschaft ist blue (vererbter Wert).

all:initial

<blockquote id="quote">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</blockquote> Phasellus eget velit sagittis.
html { font-size: small; background-color: #F0F0F0; color:blue; }
blockquote { background-color: skyblue;  color: red; }
blockquote { all: initial; }

Das <blockquote> Element verwendet nicht die Standarddarstellung des Browsers: Es ist jetzt ein Inlineelement (ursprünglicher Wert), seine background-color Eigenschaft ist transparent (ursprünglicher Wert), seine font-size Eigenschaft ist normal (ursprünglicher Wert) und seine color Eigenschaft ist black (ursprünglicher Wert).

all:inherit

<blockquote id="quote">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</blockquote> Phasellus eget velit sagittis.
html { font-size: small; background-color: #F0F0F0; color:blue; }
blockquote { background-color: skyblue; color: red; }
blockquote { all: inherit; }

Das <blockquote> Element verwendet nicht die Standarddarstellung des Browsers: Es ist jetzt ein Blockelement (vererbter Wert seines beinhaltenden <div>), seine background-color Eigenschaft ist transparent (vererbter Wert), seine font-size Eigenschaft ist small (vererbter Wert) und seine color Eigenschaft ist blue (vererbter Wert).
