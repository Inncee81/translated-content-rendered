Der <angle-percentage> CSS-Datentyp stellt einen Wert dar, der entweder <angle> oder ein percentage sein kann.

Syntax

Für <angle-percentage>-Werte sind alle Werte syntaktisch gültig, die auch bei <angle> oder percentage syntaktisch gültig sind.

Verwendung mit calc()

Wenn <angle-percentage> ein zulässiger Typ ist, dann wird aus einem Prozentwert ein einfacher <angle> errechnet. Daher kann <angle-percentage> in einem calc()-Ausdruck verwendet werden.
