Übersicht

Die column-rule-width CSS Eigenschaft definiert die Breite der Linie, die zwischen den Spalten in mehrspaltigen Layouts gezeichnet wird.

Initialwertmedium
Anwendbar aufmehrspaltige Elemente
VererbtNein
Berechneter Wertdie absolute Länge; 0 falls column-rule-style none oder hidden ist
AnimationstypLängenangabe

Syntax

/* Schlüsselwortwerte */
column-rule-width: thin;
column-rule-width: medium;
column-rule-width: thick;

/* Längenwerte */
column-rule-width: 1px;
column-rule-width: 2.5em;

/* Globale Werte */
column-rule-width: inherit;
column-rule-width: initial;
column-rule-width: unset;

Werte

<'border-width'> 
Ist ein border-width Wert oder eines der Schlüsselwörter thin, medium oder thick, und beschreibt die Breite der Linie, die zwei Spalten trennt.

Formale Syntax

<'border-width'>

Beispiel

#header {
  column-rule-width: thick;
}
