Übersicht

Die list-style Eigenschaft ist eine Kurzform für list-style-type, list-style-image und list-style-position.

Initialwertwie die jeweiligen Kurzschreibweisen:
  • list-style-type: disc
  • list-style-position: outside
  • list-style-image: none

Berechneter Wertwie die jeweiligen Kurzschreibweisen:
  • list-style-image: none oder das Bild mit absoluter URI
  • list-style-position: wie angegeben
  • list-style-type: wie angegeben
