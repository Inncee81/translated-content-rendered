Übersicht

Die CSS Eigenschaft column-count bestimmt die Spaltenanzahl innerhalb eines Elements.

Initialwertauto
Anwendbar aufBlockcontainer außer Tabellen umgebende Boxen
VererbtNein
Medienvisuell
Berechneter Wertwie angegeben
AnimationstypInteger
Kanonische Reihenfolgenach Grammatik

Syntax

Formal syntax: <integer> | auto

column-count: 3
column-count: auto

Werte

auto
Besagt, dass die Anzahl Spalten von anderen Werten Abhängig ist, wie z. B. column-width.
<number>
Ein positiver <integer>, welcher die Anzahl der Spalten genau festlegt. Falls der Wert von column-width nicht auto ist, stellt dieser Wert die maximale Anzahl an Spalten dar.

Beispiel

.content-box {
  border: 10px solid #000000;
  column-count:3;
}
