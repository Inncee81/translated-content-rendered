Übersicht

Die border-bottom-width Eigenschaft legt die Breite des unteren Rahmens fest.

Initialwertmedium
Anwendbar aufalle Elemente. Auch anwendbar auf ::first-letter.
VererbtNein
Berechneter Wertdie absolute Länge; 0 falls border-bottom-style none oder hidden ist
AnimationstypLängenangabe

Syntax

border-bottom-width: <line-width>
wobei 
<line-width> = <length> | thin | medium | thick

Werte

<Länge>
Eine Längenangabe die, die Rahmenstärke explizit angibt. Es sind keine negativen Werte erlaubt.
thin
Eine dünne Rahmenlinie (1px).
medium
Standardwert. Eine mitteldicke Rahmenlinie (3px).
thick
Eine dicke Rahmenlinie (5px).
inherit
Der Wert des Elternelements wird geerbt.

Beispiele

.beispielEins {
  border-bottom-width: 10px;
}

.beispielZwei {
  border-bottom-width: thin;
}
