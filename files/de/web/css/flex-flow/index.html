Summary

Die Kurzschreibweise flex-flow fasst die Eigenschaften flex-direction und flex-wrap zusammen.

Initialwertwie die jeweiligen Kurzschreibweisen:
  • flex-direction: row
  • flex-wrap: nowrap
Anwendbar aufflexible Container
VererbtNein
Berechneter Wertwie die jeweiligen Kurzschreibweisen:
  • flex-direction: wie angegeben
  • flex-wrap: wie angegeben
Animationstypdiskret

Siehe Using CSS flexible boxes für mehr Information.

Syntax

<'flex-direction'> || <'flex-wrap'>

Werte

Siehe flex-direction und flex-wrap.

Beispiele

element {
  /* Main-axis is the block direction with reversed main-start and main-end. Flex items are laid out in multiple lines */
  flex-flow: column-reverse wrap;
}
