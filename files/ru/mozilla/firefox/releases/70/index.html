This article provides information about the changes in Firefox 70 that will affect developers. Firefox 70 is the current Nightly version of Firefox, and will ship on October 22, 2019.

Changes for web developers

Developer tools

  • An icon will be displayed next to invalid or unsupported CSS rules in the Rules pane of the Page Inspector (баг 1306054).

Removals

HTML

No changes.

 Removals

CSS

  • [fill-stroke] [filter-effects] [css-color] percentage opacity. баг 1562086
  • [css-grid] grid-auto-columns/rows should accept multiple track-size values. баг 1339672
  • [css-text-decor-4] Implement text-decoration-skip-ink. баг 1411922
  • [css-display-3] Implement multi-keyword display values. (баг 1038294, баг 1105868 and баг 1557825)
  • [filter-effects-2] Implement backdrop-filter. баг 1178765
