Список изменения для веб - разработки

Инструменты разработчика

Нет изменений.

CSS

  • Реализовано @counter-style правило(баг 966166).
  • Больше нет необходимости использовать префикс -moz-ethiopic-numeric(баг 985825).
  • Добавлена поддержка disclosure-open/disclosure-closed стилей счётчиков(баг 982355).
  • CSS анимация с пустыми keyframes свойствами обрабатывает события по умолчанию(баг 1004377).

HTML

Нет измененеий.

JavaScript

  • Удален нестандартный метод Number.toInteger() (баг 1022396).
  • Методы Map.prototype.set(), WeakMap.prototype.set() and Set.prototype.add() можно использовать последовательно, больше не возвращают undefined в место этого возвращают объект. (баг 1031632).
  • Добавленна поддержка типа данных "Symbol" из стандарта ECMAScript 6 (баг 645416):
    • Symbol
    • Symbol.for()
    • Symbol.keyFor()
    • Object.getOwnPropertySymbols()
  • Параметры по умолчанию инициализированные до инициализации функции или в теле функции не могут быть использованны в качестве параметров по умолчанию (баг 1022962).

Interfaces/APIs/DOM

Нет изменений.

MathML

  • Используются Open Type MATH для задания констант fractions, stacks, radicals и скриптов (баг 961365).

SVG

Нет изменений.

Audio/Video

Нет изменений.

Нет изменений.

Изменения для плагинов и методов их разработки

Нет изменений.
