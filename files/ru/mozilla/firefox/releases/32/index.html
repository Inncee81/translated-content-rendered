Изменения для Web Разработчиков

Инструментарий разработчика

Основные изменения:

  • Редактор Web Audio
  • Автоподстановка и подсказки в Блокноте
  • Стили User agent в представлении Inspector Rules
  • Кнопка выбора элемента со страницы перемещена
  • Добавлено отображение размеров узла в информацонную панель Инспектора
  • Добавлена кнопка снятия снимка всей страницы
  • Добавлены HiDPI изображения в инструменты
  • Узлы, содержщие display:none в Инспекторе отображаются иначе

Все ошибки исправленные между версиями Firefox 31 и Firefox 32.

CSS

  • mix-blend-mode включено по умолчанию (баг 952643).
  • position:sticky включено по умолчанию в Release сборках (раньше было только в Nightly и Aurora) (баг 916315).
  • реализовано box-decoration-break: Левая/правая часть box-shadow должна быть отрисована только на первом/последнем элементе inline контейнера (баг 613659).
  • Реализация flex-grow/flex-shrink для преобразования между нулевыми и не нулевыми значениями (баг 996945).

HTML

  • Экспериментальная реализация свойства (включается параметром) <img> srcset. Чтобы включить его, установите значение dom.image.srcset.enable в true (баг 870021).

JavaScript

  • Реализованы новые встроенные методы ECMAScript 6:
    • Array.from() (баг 904723)
    • Array.prototype.copyWithin()
    • Number.isSafeInteger()

Интерфейсы/APIs/DOM

  • Реализованы свойство navigator.languages и событие languagechange (баг 889335).
  • В методы KeyboardEvent.getModifierState() и MouseEvent.getModifierState() добавлена поддержка виртуального модификатора Accel (баг 1009388).
  • Добавлена экспериментальная реализация свойства KeyboardEvent.code: она отключена в Release сборках (баг 865649).

MathML

  • Добавлена поддержка для <menclose> notation phasorangle.

SVG

Без изменений.

Audio/Video

Без изменений.

Безопасность

  • Привилегированный код теперь обладает рентгеновским зрением относительно экземпляров Object и Array в JavaScript.

Изменения для разработчиков дополнений и разработчиков Mozilla

Add-on SDK

Основные изменения

  • Добавлено свойство exclude в PageMod.
  • Добавлено свойство anonymous в Request.
  • Add-on Debugger теперь содержит Консоль и Блокнот.
