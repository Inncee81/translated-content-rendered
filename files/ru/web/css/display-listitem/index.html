Ключевое слово list-item приводит к тому, что элемент генерирует ::marker псевдо-элемент с определяемыми list-style свойствами (например, маркерной точкой) вместе с основным блоком указанного типа для его  содержимого.

Синтаксис

Единственное значение list-item приведет к тому, что элемент будет вести себя как элемент списка. Это можно использовать вместе с list-style-type и list-style-position.

Примечание:В браузерах, поддерживающих двухзначный синтаксис, если внутреннее значение не указано, по умолчанию оно будет flow. Если внешнее значение не указано, основной блок будет иметь внешний вид отображения (outer display type) block.

Пример

HTML


<div class="fake-list">I will display as a list item</div>

CSS


.fake-list {
  display: list-item;
  list-style-position: inside;
}
