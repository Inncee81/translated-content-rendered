Геттер size возвращает количество элементов в объекте Map.

Описание

Значение size - целое число, описывающее, сколько элементов имеет объект Map. У свойства size не определён сеттер: переопределить его нельзя.

Примеры

Использование size

var myMap = new Map();
myMap.set("a", "альфа");
myMap.set("b", "бета");
myMap.set("g", "гамма");

myMap.size // 3
