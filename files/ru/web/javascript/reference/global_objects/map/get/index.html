Метод get() возвращает указанный элемент объекта Map.

Синтаксис

myMap.get(key);

Параметры

key
Обязательный. Ключ элемента объекта Map.

Возвращаемое значение

Возвращает элемент связанный с указанным ключом или undefined, если ключ не был найден в объекте Map​​​​.

Примеры

Использование метода get()

var myMap = new Map();
myMap.set('bar', 'foo');

myMap.get('bar');  // Вернет "foo".
myMap.get('baz');  // Вернет undefined.
