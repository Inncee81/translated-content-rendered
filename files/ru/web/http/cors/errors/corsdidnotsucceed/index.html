Причина

Причина: Не удалось выполнить запрос CORS

Что произошло?

Запрос HTTP, использующий CORS не был выполнен, потому что подключение через HTTP не было выполнено либо на сетевом уровне или на уровне протокола. Ошибка не имеет прямого отношения к CORS, но все равно считается как фундаментальная.
