Атрибут stop-color указывает какой цвет использовать при остановке градиента. Значения currentColor и ICC цвета могут быть заданы так же, как в <paint> спецификации для атрибутов   fill и stroke.

Контекст использования

Категория Атрибут представления
Значение currentColor | <color> | <icccolor> | inherit
Начальное значение black
Анимация Да
Нормативный документ SVG 1.1 (2nd Edition)
