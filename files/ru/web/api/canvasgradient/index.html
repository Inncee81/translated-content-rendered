Интерфейс CanvasGradient представляет собой непрозрачный объект, описывающий градиент. Он возвращается методами CanvasRenderingContext2D.createLinearGradient() и CanvasRenderingContext2D.createRadialGradient().

Свойства

Representing an opaque object, there is no exposed property.

Методы

Не имеет наследованных методов.

CanvasGradient.addColorStop()
Добавляет в градиент новый маркер, определяемый смещенем и цветом. Если смещение входдит за пределы 0 и 1, генерируется INDEX_SIZE_ERR. Если не удааётся обработать цвет как CSS <color>, генерируется SYNTAX_ERR.
