HTMLAreaElement интерфейс предоставляющий специальные свойства и методы (помимо тех, постоянных объектов HTMLElement интерфейса, также доступных ему по наследству) для манипуляции расположением и представлением пространством элементов.

Свойства

Наследует свойства от его родителя, HTMLElement, и реализует те из URLUtils.

HTMLAreaElement.accessKey
Это DOMString содержащий единственный символ, который переключает фокус ввода для управления.
HTMLAreaElement.alt
Это DOMString которое отображает alt HTML атрибут, содержащий альтернативный текст для элемента.
HTMLAreaElement.coords
Это DOMString который отображает coords HTML атрибут, содержащий координаты активного участка региона.
HTMLAreaElement.download
Это DOMString показывает что связанный ресурс предназначен для загрузки, а не для отображения в браузере. Значение представляет собой предполагаемое имя файла. Если имя не является допустимым именем файла в данной ОС, браузер будет адаптировать его.
URLUtils.hash
Это DOMString содержащий идентификатор фрагмента (включая начальный hash mark (#)), если таковые имеются, в указанном URL.
URLUtils.host
Это DOMString содержащий имя хоста и порт (если это не порт по умолчанию) в указанном URL.
URLUtils.hostname
Это DOMString содержащий имя хоста в указанном URL.
URLUtils.href
Это DOMString содержание которого отображает href HTML атрибут, содержащий действительный URL связанного с ним ресурса.
HTMLAreaElement.hreflang
Это DOMString содержание которого отображает hreflang HTML аттрибут, показывающий язык связанного с ним ресурса.
HTMLAreaElement.media
Это DOMString содержание которого отображает media HTML аттрибут, показывающий целевое mediaсвязанного с ним ресурса.
HTMLAreaElement.noHref
Это Boolean флаг показывающий, если зона неактивна (true) или активна (false).
URLUtils.password
Это DOMString содержащий пароль указанный перед именем домена.
URLUtils.origin Только для чтения
Возвращает DOMString содержащий изначальный URL, то есть его схема, его домен, его порт.
URLUtils.pathname
Это DOMString содержащий состовную часть пути имени, если таковые имеются в указанном URL.
URLUtils.port
Это DOMString содержащий компонент порта , если таковой имеется в указанном URL.
URLUtils.protocol
Это DOMString содержащий компоненту протокола (в том числе двоеточия ':'), в указанном URL.
HTMLAreaElement.referrer
Это DOMString который отображает referrer HTML атрибут,  показывающий какой referrer используется когда извлекается связанны с ним ресурс.
HTMLAreaElement.rel
Это DOMString который отображает rel HTML атрибут,показывая отношения текущего документа со связанным с ним ресурсом.
HTMLAreaElement.relList Только для чтения
Возвращает DOMTokenList который отображает rel HTML атрибут, показывая отношения текущего документа к связанному с ним ресурса, как список токенов.
HTMLAreaElement.search
Это DOMString содержащий найденный элемент (включая начальный знак вопроса'?'), если таковой имеется, в указанном URL.
HTMLAreaElement.shape
Это DOMString который отображается shape HTML атрибут, включающий вид активного участка, ограниченный известными значениями.
HTMLAreaElement.tabIndex
Как long содержащий положение элемента в порядке табуляции.
HTMLAreaElement.target
Это  DOMString которое отображает target HTML атрибут, включая контекст просмотра, в котором открыт связанный с ним ресурс.
HTMLAreaElement.type
Это DOMString которое отображает type HTML атрибут, показывающий MIME тип связанного с ним ресурса.
URLUtils.username
Это DOMString содержащий имя пользователя. определеное перед доменным именем.

Методы

наследует методы от его родителя, HTMLElement и реализует те из URLUtils.

URLUtils.toString()
Возвращает DOMString содержащий весь URL скрипта, выполняемого в Worker. Это синоним для URLUtils.href.
