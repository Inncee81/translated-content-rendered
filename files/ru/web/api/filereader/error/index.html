Сводка

Возвращает ошибку, произошедшую во время чтения файла.

Синтакс

var error = instanceOfFileReader.error

Значение

DOMError содержит соответствующую ошибку. В Chrome 48+/Firefox 58+ значение возвращает DOMException так как DOMError был удалён из DOM-стандарта.
