Метод readAsText() используется для чтения содержимог указанного Blob или File. Когда операция закончится, readyState примет значение DONE  и будет вызвано событие loadend. В то же время, аттрибут result будет содержать данные файла как строку.

Syntax

instanceOfFileReader.readAsText(blob[, encoding]);

Parameters

blob
Blob или File которые следует прочитать.
encoding Необязательный
Строка, обозначающая кодировку используемую для возращаемых данных. По умолчанию используется UTF-8, если параметр не указан.
