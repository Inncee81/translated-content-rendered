Это экспериментальная технология
Так как спецификация этой технологии ещё не стабилизировалась, смотрите таблицу совместимости по поводу использования в различных браузерах. Также заметьте, что синтаксис и поведение экспериментальной технологии может измениться в будущих версиях браузеров, вслед за изменениями спецификации.

Интерфейс NavigatorPlugins содержит методы и свойства, связанные с установленными плагинами в браузере.

NavigatorPlugins не содержит в себе объектов другово типа, но другие интерфейсы, такие как Navigator реализуют его.

Свойства

NavigatorPlugins.mimeTypes Только для чтения
Возвращает MimeTypeArray листинг MIME типов поддерживаемые браузером.
NavigatorPlugins.plugins Только для чтения
Возвращает листинг плагинов PluginArray, установленных в браузере.

Методы

Интерфейс NavigatorPlugins не наследует никаких методов.

NavigatorPlugins.javaEnabled Только для чтения
Возвращает флаг Boolean показывающий включен ли java в браузере или нет.
