Свойство onblur из GlobalEventHandlers событий EventHandler для blur события. Оно доступно дляElement, Document и Window.

Событие blur возникает когда элемент теряет фокус.

Note: Обратным onblur является onfocus.

Syntax

target.onblur = functionRef;

Value

functionRef  - это имя функции или function expression. Функция получает объект FocusEvent в качестве единственного аргумента.

Example

В этом примере onblur и onfocus используются для изменения текста в элементе <input>.

HTML

<input type="text" value="CLICK HERE">

JavaScript

let input = document.querySelector('input');

input.onblur = inputBlur;
input.onfocus = inputFocus;

function inputBlur() {
  input.value = 'Focus has been lost';
}

function inputFocus() {
  input.value = 'Focus is here';
}

Result

Нажимайте внутри и вне поля формы. Содержимое будет изменятся.
