Свойства

Наследует свойства родителя, HTMLElement.

HTMLDivElement.align
Свойство типа DOMString указывающее выравнивание содержимого элемента относительно окружающего контекста. Принемает следующие значения "left", "right", "justify", "center".

Методы

Не имеет специальных методов; Наследует методы родителя, HTMLElement.
