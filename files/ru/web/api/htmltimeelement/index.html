Свойства

Наследует свойства от родителя HTMLElement.

HTMLTimeElement.dateTime
Это DOMString который отражает datetime HTML атрибут, содержащий понятный для машин формат даты и времени.

Методы

Нет определённого метода; наследует его у родителя, HTMLElement.
