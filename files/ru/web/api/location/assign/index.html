Метод Location.assign() запускает загрузку и отображение нового документа по указанному URL.

Если новый URL не удовлетворяет требованиям безопасности, выбрасывается DOMException с типом SECURITY_ERROR. Это случается, если домен скрипта, вызвающего метод, не совпадает с доменом страницы, изначально содержащимся в Location.

Если новый URL некорректен, выбрасывается DOMException с типом SYNTAX_ERROR.

Синтаксис

object.assign(url);

Параметры

url
DOMString, содержащий URL страницы, на которую нужно перейти.

Примеры

// Перейти на статью Location.reload
document.location.assign('https://developer.mozilla.org/ru/docs/Web/API/Location.reload');
