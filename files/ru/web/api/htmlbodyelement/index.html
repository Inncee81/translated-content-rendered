HTMLBodyElement интерфейс предоставляющий специальные свойства (помимо этого, они постоянно наследуют HTMLElement интерфейс) для манипуляции элементами тела.

Свойства

Наследует свойства о его родителя, HTMLElement и от WindowEventHandlers.

HTMLBodyElement.aLink
Это DOMString который представляет цвет активных гиперссылок.
HTMLBodyElement.background
Это DOMString который представляет собой описание расположения фонового изображения ресурса. Обратите внимание, что это не URI, хотя в некоторых старых версиях некоторых браузеров ожидается именно это.
HTMLBodyElement.bgColor
Это DOMString который представляет собой цвет фона для документа.
HTMLBodyElement.link
Это DOMString который представляет собой цвет непосещенных ссылок.
HTMLBodyElement.text
Это DOMString который представляет собой основной цвет текста.
HTMLBodyElement.vLink
Это DOMString который представляет собой цвет посещенных ссылок.

Методы

Не имеет специфичных методов; наследует методы от его родителя, HTMLElement и от WindowEventHandlers.

Обработчики событий

Не имеет специфичных обработчиков событий; наследует обработчики событий от его родителя, HTMLElement и от WindowEventHandlers.

WindowEventHandlers.onhashchange

WindowEventHandlers.onlanguagechange
Это EventHandler  представляющее собой код который будет вызван когда событие languagechange вызванно.
WindowEventHandlers.onmessage
Это EventHandler  представляющее собой код который будет вызван когда событие message вызванно.
WindowEventHandlers.onoffline
Это EventHandler  представляющее собой код который будет вызван когда событие offline вызванно.
WindowEventHandlers.ononline
Это EventHandler  представляющее собой код который будет вызван когда событие online вызванно.
WindowEventHandlers.onpagehide
Это EventHandler  представляющее собой код который будет вызван когда событие pagehide вызванно.
WindowEventHandlers.onpageshow
Это EventHandler  представляющее собой код который будет вызван когда событие pageshow вызванно.

Это EventHandler  представляющее собой код который будет вызван когда событие popstate вызванно.

Это EventHandler  представляющее собой код который будет вызван когда событие storage вызванно.
