Вышла из употребления
Эта возможность вышла из употребления. Хотя она может продолжать работать в некоторых браузерах, её использование не рекомендуется, поскольку она может быть удалена в любое время. Старайтесь избегать её использования.

Тип возвращаемый некоторыми API, который содержит список DOMString (строки).

Свойства

DOMStringList.length Только для чтения
Возвращает длинну списка.

Методы

DOMStringList.item()
Возвращает DOMString.
DOMStringList.contains()
Возвращает Boolean с указанием, есть ли данная строка в списке.
