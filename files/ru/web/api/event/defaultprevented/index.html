Описание

Возвращает boolean-значение, информирующее о том, был ли вызван event.preventDefault() в текущем обработчике события.

Note: Используйте это свойство вместо устаревшего метода getPreventDefault() (смотри баг 691151).

Синтаксис

var defaultWasPrevented = event.defaultPrevented;

Пример

 if (e.defaultPrevented) {
   /* по умолчанию не выполнится */
 }
