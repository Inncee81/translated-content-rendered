FormData.keys() метод возвращает iterator позволяя пройтись по всем ключам содержащимся в этом объекте. Ключи являются USVString объектами.

Примечание: Метод доступен в Web Workers.

Синтаксис

formData.keys();

Возвращаемые значения

Возвращает iterator.

Пример

// Create a test FormData object
var formData = new FormData();
formData.append('key1', 'value1');
formData.append('key2', 'value2');

// Display the keys
for (var key of formData.keys()) {
   console.log(key);
}

Результат:

key1
key2
