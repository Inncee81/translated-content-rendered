Secure context
This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

Свойство onconnect интерфейса USB является обработчиком события, который вызывается всегда, когда сопряжённое устройство подключается.

Синтаксис

USB.onconnect = connectFunction
