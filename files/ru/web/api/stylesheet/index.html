Объектная модель CSS

Объект, реализующий интерфейс StyleSheet, представляет отдельную таблицу стилей. Таблица стилей CSS далее будет представлена более специализированным интерфейсом  CSSStyleSheet.

Свойства

StyleSheet.disabled
Свойство имеет тип Boolean, принимающий одно из двух значений и определяющий, применяется или нет текущая таблица стилей.
StyleSheet.href Только для чтения
Свойство возвращает строку DOMString, указывающую расположение таблицы стилей. Свойство не изменяется.
StyleSheet.media Только для чтения
Свойство содержит список медиа-установокMediaList таблицы стилей и определяет медиа-устройства, для отображения на которых предназначена данная таблица. Свойство не изменяется.
StyleSheet.ownerNode Только для чтения
Свойство возвращает ссылку на объект (узел)Node, ссылающийся на данную таблицу стилей текущего документа. Свойство не изменяется.
StyleSheet.parentStyleSheet Только для чтения
Свойство возвращает ссылку на родительскую таблицу стилей StyleSheet, включаюущую данную таблицу стилей; если таковой нет, возвращает значение null. Свойство не изменяется.
StyleSheet.title Только для чтения
Свойство возвращает строку DOMString, указывающую на заголовок  текущей таблицы стилей. Свойство не изменяется.
StyleSheet.typeТолько для чтения
Свойство возвращает строку DOMString, указывающую на язык, используемый таблицей стилей. Свойство не изменяется.
