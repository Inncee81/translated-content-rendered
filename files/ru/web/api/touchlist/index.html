Интерфейс TouchList представляет собой список точек касаний сенсорной поверхности. Например, если пользователь коснулся сенсорной поверхности (такой как экран или трекпад) тремя пальцами, соответствующий объект TouchList будет содержать по одному объекту Touch для каждого пальца, всего три объекта.

Свойства

TouchList.length Только для чтения
Количество объектов Touch в TouchList.

Методы

TouchList.identifiedTouch()
Возвращает первый объект Touch из списка, чей идентификатор соответствует заданному значению.
TouchList.item()
Возвращает объект Touch, находящийся в списке по указанному индексу
