Свойство Gamepad.timestamp интерфейса Gamepad возвращает DOMHighResTimeStamp, представляющий собой время последнего обновления геймпада.

Идея состоит в том, чтобы позволить разработчику определить, были ли изменены данные для axes и button на аппаратном обеспечении. Значение должно быть установлено относительно атрибута navigationStart интерфейса PerformanceTiming. Значения монотонно увеличиваются, поэтому их можно сравнивать, чтобы определить порядок изменений, т.к. новые значения всегда будут больше или равны старым.

Примечание: В данный момент, это свойство нигде не поддерживается.

Синтаксис

readonly    attribute DOMHighResTimeStamp timestamp;

Пример

var gp = navigator.getGamepads()[0];
console.log(gp.timestamp);

Значение

DOMHighResTimeStamp.
