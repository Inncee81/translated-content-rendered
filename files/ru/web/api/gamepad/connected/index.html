Свойство Gamepad.connected интерфейса Gamepad возвращает логическое значение того, подключен ли всё ещё геймпад к системе.

Если геймпад подключен - значение равно true; если нет - false.

Синтаксис

 readonly    attribute boolean             connected;

Пример

var gp = navigator.getGamepads()[0];
console.log(gp.connected);

Значение

Boolean.
