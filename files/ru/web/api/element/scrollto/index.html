scrollTo() метод интерфейса  Element, осуществляет прокрутку по заданному набору координат внутри данного элемента. 

Синтаксис

element.scrollTo(x-coord, y-coord)
element.scrollTo(options)

Параметры

  • x-coord координаты пикселя по горизонтальной оси элемента, который необходимо отобразить вверху слева.
  • y-coord координаты пикселя по вертикальной оси элемента, который необходимо отобразить вверху слева.

- or -

  • options как объект ScrollToOptions, с тремя свойствами.

Примеры

element.scrollTo(0, 1000);

С помощью options:

element.scrollTo({
  top: 100,
  left: 100,
  behavior: 'smooth'
});
