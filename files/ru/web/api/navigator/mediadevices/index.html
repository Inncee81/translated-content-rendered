Свойство Navigator.mediaDevices только для чтения, возвращает объект MediaDevices, который предоставляет доступ к подключенным медиа-устройствам ввода, таким как камера и микрофон, а также совместное использование экрана.

Синтаксис

var mediaDevices = navigator.mediaDevices;

Возвращаемое значение

MediaDevices объект синглтон. Как правило, вы просто используете элементы этого объекта напрямую, например с помощью вызова navigator.mediaDevices.getUserMedia().
