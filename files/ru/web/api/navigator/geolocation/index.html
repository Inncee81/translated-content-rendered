Свойство Navigator.geolocation только для чтения, возвращающие объект Geolocation, который дает веб контенту доступ к месторасположению устройства. Это позволяет веб-сайту или приложению предложить индивидуальный результат основанный на месторасположении пользователя.

Примечание: По соображениям безопасности, когда веб-страница пытается получить доступ к информации о местоположении, пользователь получает уведомление с просьбой дать разрешение. Помните, что каждый браузер имеет свои собственные правила и методы для запроса этого разрешения.

Синтаксис

geo = navigator.geolocation
