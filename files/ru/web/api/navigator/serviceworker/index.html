Свойство только-для-чтения Navigator.serviceWorker возвращает объект ServiceWorkerContainer, который предоставляет доступ к регистрации, удалению, обновлению и взаимодействию с объектами ServiceWorker для соответствующего документа.

Синтаксис

var workerContainerInstance = navigator.serviceWorker;

Значение

Объект ServiceWorkerContainer.
