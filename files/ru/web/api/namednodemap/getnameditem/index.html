Метод getNamedItem() интерфейса NamedNodeMap возвращает значение Attr, соответствующее заданному имени, или значение null, если соответствующий атрибут отсутствует.

Синтаксис

myAttr = attrs.getNamedItem(name)

Parameters

  • name имя нужного атрибута
