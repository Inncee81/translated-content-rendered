Метод XMLHttpRequest.abort() прерывает уже отправленный запрос. Когда запрос прерывается, то его свойство readyState устанавливается в UNSENT (0), а status в 0.

Синтаксис

XMLHttpRequest.abort()

Параметры

Нет.

Возвращаемое значение

undefined

Пример

var xhr = new XMLHttpRequest(),
    method = "GET",
    url = "https://developer.mozilla.org/";
xhr.open(method, url, true);

xhr.send();

xhr.abort();
