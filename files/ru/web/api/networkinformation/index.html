Это экспериментальная технология
Так как спецификация этой технологии ещё не стабилизировалась, смотрите таблицу совместимости по поводу использования в различных браузерах. Также заметьте, что синтаксис и поведение экспериментальной технологии может измениться в будущих версиях браузеров, вслед за изменениями спецификации.

Интерфейс NetworkInformation содержит методы и свойства, связанные с типом сети браузера.

В нем нет объектов типа  NetworkInformation, но другие интерфейсы, такие как Navigator, реализуют его.

Свойства

Интерфейс NetworkInformation не наследует других свойств.

NetworkInformation.connection Только для чтения
Connection предоставляет информацию о сетевом соединении устройства.

Методы

Интерфейс NetworkInformation ничего не предоставляет и не наследует других методов.

Заметки для Gecko

  • Сетевое API может быть отключено с помощьюc dom.network.enabled.
  • Как и в Gecko 12.0, сетевое API реализованно только для Android. Поддержка для Linux (баг 712442) и Windows (баг 721306) систем находится в процессе разработки.
