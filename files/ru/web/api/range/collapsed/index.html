Доступное только для чтения свойство Range.collapsed возвращает флаг Boolean, который указывает, в одной ли и той же позиции находятся начало и конец диапазона Range. Оно возвращает true, если начало и конец граничных точех диапазона Range являются одной и той же точкой в DOM, false – в противном случае.

Свёрнутый диапазон Range является пустым, не имеет содержимого, определяет единую точку в древе DOM. Для того, чтобы свернуть диапазон, смотрите метод Range.collapse().

Syntax

isCollapsed = range.collapsed;

Example

var range = document.createRange();

range.setStart(startNode,startOffset);
range.setEnd(endNode,endOffset);
isCollapsed = range.collapsed;
