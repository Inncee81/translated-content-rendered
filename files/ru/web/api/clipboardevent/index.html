Это экспериментальная технология
Так как спецификация этой технологии ещё не стабилизировалась, смотрите таблицу совместимости по поводу использования в различных браузерах. Также заметьте, что синтаксис и поведение экспериментальной технологии может измениться в будущих версиях браузеров, вслед за изменениями спецификации.

Интерфейс ClipboardEvent предствляет события, предоставляющие информацию,  связанную с изменением буфера обмена, этими события являются cut, copy и paste.

Свойства

Интерфейс наследует свойства от родителя Event.

ClipboardEvent.clipboardData Только для чтения
Является DataTransfer объектом, который содержит данные, полученные от совершения пользователем операции cut, copy или paste, а также их MIME тип.

Конструктор

ClipboardEvent()
Создает ClipboardEvent событие с переданными параметрами.

Методы

Нет специальных методов; Этот интерфейс наследует методы от своего родителя Event.
