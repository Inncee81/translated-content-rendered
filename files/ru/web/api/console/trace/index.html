Выводит стэк трэйс в  Веб Консоль.

Примечание: Эта возможность доступна в Web Workers.

Посмотрите Стэк трэйсы в документации console с деталями и примерами.

Синтаксис

console.trace();

Пример

function foo() {
  function bar() {
    console.trace();
  }
  bar();
}

foo();

В консоли следующий трэйс будет отображен так:

bar
foo
<anonymous>
