Запускает таймер, который вы можете использовать для определения, сколько времени занимает выполнение той или иной операции. Вы должны назначить каждому таймеру уникальное имя и таким образом можете запустить до 10000 таймеров на одной странице. Когда вы вызовете console.timeEnd() с именем, которое использовалось при запуске таймера, браузер отобразит в консоли время (в миллисекундах), прошедшее с момента запуска таймера.

Примеры и более детальная информация находятся в разделе Таймеры документации к console

Примечание: Эта возможность доступна в Web Workers.

Синтаксис

console.time(label);

Параметры

label
Название нового таймера. Оно однозначно идентифицирует таймер. Используйте то же самое название таймера, когда вызываете console.timeEnd() для остановки и отображения времени выполнения таймера.
