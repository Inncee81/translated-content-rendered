Это экспериментальная технология
Так как спецификация этой технологии ещё не стабилизировалась, смотрите таблицу совместимости по поводу использования в различных браузерах. Также заметьте, что синтаксис и поведение экспериментальной технологии может измениться в будущих версиях браузеров, вслед за изменениями спецификации.

Интерфейс SVGAElement предоставляет доступ к свойствам <a> элемента а также к методам манипулирования ими.

Интерфейс наследует следующие интерфейсы: SVGElement, SVGURIReference, SVGTests, SVGLangSpace, SVGExternalResourcesRequired, SVGStylable, SVGTransformable.

Свойства

Этот интерфейс также наследует свойства от его родителя, SVGGraphicsElement и реализует свойства из SVGURIReference и HTMLHyperlinkElementUtils.

SVGAElement.download
См. HTMLAnchorElement.download.
SVGAElement.href
См. HTMLAnchorElement.href.
SVGAElement.hreflang
Является DOMString, который отражает атрибут hreflang, указывающий язык связанного ресурса.
SVGAElement.ping
Является DOMString, отражая атрибут ping, содержащий список URL-адресов, разделенных пробелами, к которым, когда выполняется гиперссылка, POST запрос с телом PING будет отправлен браузером (в фоновом режиме). Обычно используется для отслеживания.
SVGAElement.referrerPolicy
См. HTMLAnchorElement.referrerPolicy.
SVGAElement.rel
См. HTMLAnchorElement.rel.
SVGAElement.relList
См. HTMLAnchorElement.relList.
SVGAElement.target Только для чтения
соответствует аттрибуту target данного элемента. 
SVGAElement.text
Является DOMString синонимом свойства Node.textContent. 
SVGAElement.type
Is a that reflects theattribute, indicating the MIME type of the linked resource. 
Является DOMString, который отражает атрибут type , указывающий тип MIME связанного ресурса.

Методы

Этот интерфейс не имеет методов, но наследует методы от его родителя, SVGGraphicsElement. 

Пример

В примере ниже, атрибуту target элемента <a> присвоено значение _blank и при клике по ссылке, при соблюдении условий, выведется сообщение в модальном окне. 

var linkRef = document.querySelector('a');
linkRef.target ='_self';

linkRef.onclick = function(){
    if (linkRef.target == '_blank') {
        alert("BLANK !");
        linkRef.target = '_self';
    }
    else
        alert("SORRY! not _blank");
}
