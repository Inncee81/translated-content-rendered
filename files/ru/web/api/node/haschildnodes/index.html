Метод Node.hasChildNodes() возвращает Boolean значение показывающее имеет ли текущий Node дочерние узлы или нет.

Синтаксис

node.hasChildNodes()

Пример

Следующий пример удаляет первый дочерний узел внутри элемента с id "foo", если foo имеет дочерние узлы.

var foo = document.getElementById("foo");

if ( foo.hasChildNodes() ) {
  foo.removeChild( foo.childNodes[0] );
}
