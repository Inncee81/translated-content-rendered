Метод Node.lookupPrefix() возвращает DOMString содержащий префикс для данного пространства имен URI, если он присутствует, и null если нет. Когда возможно присутствие нескольких префиксов, результат зависит от реализации.

Из-за ошибки bug 312019, этот метод не работает с динамическим назначением пространства имен, которое установлено с тем же свойством Node.prefix.
