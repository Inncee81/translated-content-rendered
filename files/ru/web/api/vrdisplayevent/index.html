Интерфейс VRDisplayEvent API WebVR представляет объект  событий, связанных с WebVR (Подробнее - см. Список расширений окна WebVR)

Constructor

VRDisplayEvent.VRDisplayEvent()
Создает экземпляр объекта VRDisplayEvent.

Properties

VRDisplayEvent также наследует свойства от своего родительского объекта, Event.

VRDisplayEvent.display Только для чтения
 VRDisplay связанный с этим событием.
VRDisplayEvent.reason Только для чтения
Определяемая человеческим фактором причина, по которой произошло событие.

Examples

window.addEventListener('vrdisplaypresentchange', function(e) {
  console.log('Display ' + e.display.displayId + ' presentation has changed. Reason given: ' + e.reason + '.');
})
