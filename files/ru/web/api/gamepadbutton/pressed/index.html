Свойство GamepadButton.pressed интерфейса GamepadButton возвращает boolean, указыващий, нажата ли текущая кнопка (true), или нет (false).

Синтакс

var isPressed = navigator.getGamepads()[0].pressed;

Пример

var gp = navigator.getGamepads()[0]; // Get the first gamepad object

if(gp.buttons[0].pressed == true) {
  // respond to button being pressed
}

Значение

 boolean.
