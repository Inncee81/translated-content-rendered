CanvasCaptureMediaStreamTrackИнтерфейс представляет собой видеодорожку, содержащуюся в MediaStream, генерируемую из <canvas> после вызова HTMLCanvasElement.captureStream ().

Часть API захвата мультимедиа и потоков.

Свойства

Этот интерфейс наследует свойства своего родителя, MediaStreamTrack.

CanvasCaptureMediaStreamTrack.холст Только для чтения
Возвращает объект HTMLCanvasElement, поверхность которого захватывается в режиме реального времени.

Методы

Этот интерфейс наследует методы своего родителя, MediaStreamTrack.

CanvasCaptureMediaStreamTrack.requestFrame()
Вручную заставляет кадр быть захваченным и отправленным в поток. Это позволяет приложениям, которые хотят непосредственно указать время захвата кадра, сделать это, если они указали a frameRateиз 0 при вызове captureStream ().
