HTML (קיצור של HyperText Markup Language) היא 'שפה תיאורית' המציינת את מבנה דף האינטרנט.

קצת היסטוריה

בשנת 1990, כחלק מחזונו של ה-Web, טים ברנרס-לי הגדיר את המושג hypertext, לאחר שנה הגדיר זאת בעיקר באמצעות SGML. ה-IETF החל לציין רשמית HTML בשנת 1993, ואחרי כמה טיוטות שוחררה גרסה 2.0 בשנת 1995. בשנת 1994 ייסד ברנרס-לי את W3C כדי לפתח את ה-Web. בשנת 1996 השתלט W3C על עבודת ה-HTML ופרסם את המלצת ה-HTML 3.2 כעבור שנה. HTML 4.0 שוחרר בשנת 1999 והפך לסטנדרט ISO בשנת 2000.

באותה תקופה, W3C כמעט נטש את HTML לטובת XHTML, מה שהניע את הקמתה של קבוצה עצמאית שנקראת WHATWG ב-2004. תודות ל- WHATWG, העבודה על HTML5 המשיכה: שני הארגונים פרסמו את הטיוטה הראשונה בשנת 2008 ואת התקן הסופי בשנת 2014.

קונספט ותחביר

מסמך HTML הוא מסמך טקסט רגיל שמאורגן ומחולק באמצעות elements. אלמנטים מוקפים בפתיחה וסגירה תואמים - "תגיות" tags. כל תגית מתחילה ומסתיימת בסוגריים זוויתיים (<>). ישנם כמה תגיות ריקות או מבוטלות שלא יכולות לכלול טקסט כלשהו, ​​למשל <img>.

אפשר להרחיב את תגיות ה-HTML עם attributes, המספקים מידע ותכונות נוספות, שמשפיעים על האופן בו הדפדפן מפרש את האלמנט:

קובץ HTML נשמר בדרך כלל עם סיומת htm. או html., המוגשת על ידי web server, וניתנת לעיבוד והצגה על ידי כל Web browser.
