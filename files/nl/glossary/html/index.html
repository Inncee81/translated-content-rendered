    HTML (HyperText Markup Language) is een beschrijvende taal specifiek voor de structuur van een webpagina.

    Korte geschiedenis

    In 1990, als deel van zijn visie voor de Web definieerde Tim Berners-Lee het concept van hypertext. Hetgeen Berners-Lee het jaar daarop formaliseerde met een opmaak die vooral gebaseerd was op SGML. Officieel begon de IETF HTML te noemen in 1993 en na verschillende proefversies werd de versie 2.0 uitgebracht in 1995. In 1994 richtte Berners-Lee de W3C om het Web verder uit te bouwen. In 1996 nam W3C het HTML-werk over en publiceerde HTML 3.2 een jaar later. HTML 4.0 werd in 1999 uitgebracht en werd een ISO-standaard in 2000.

    Toen gaf W3C bijna HTML op voor de XHTML, dit leidde tot de stichting van de onafhankelijke groep WHATWG in 2004. Dankzij de WHATWG werd het werk aan HTML5 voortgezet: de twee organisaties brachten de eerste proefversie uit in 2008 en de definitieve versie in 2014.

    Concept en syntax

    Een HTML-document is een onbewerkt tekstdocument gestructureerd met elementenelements. Elementen zitten tussen "opening" (geopende) en "closing (sluitende) tags. Iedere tag begint met het teken 'kleiner dan' en eindigt met het teken 'groter dan'. Er bestaan enkele lege of 'void' tags die geen enkele tekst omvat, bijvoorbeeld <img>.

    Een HTML-document wordt normaal bewaard met een .htm of .html extensie op een web server en kan weergegeven worden door iedere Web browser.
