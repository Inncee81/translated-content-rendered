--- title: Waarde slug: Glossary/Value tags: - CodingScripting - woordenlijst translation_of: Glossary/Value ---

In de context van data of een object wrapper rond die data, de waarde is de primitieve waarde die de wrapper bevat. In de context van een variabele of eigenschap, de waarde kan zowel een primitieve als een object referentie zijn.
