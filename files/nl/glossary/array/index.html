Een array is een geordende collectie van data (primitieve of object). Elk data object van de array heeft een numerieke index gebaseerd op de positie in de array. Deze index kan gebruikt worden om de corresponderende waarde op te vragen. In JavaScript zijn arrays objecten die gemanipuleerd kunnen worden door middel van verschillende methoden.
