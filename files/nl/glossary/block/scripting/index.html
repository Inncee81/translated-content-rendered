--- title: Block (scripting) slug: Glossary/Block/Scripting translation_of: Glossary/Block/Scripting ---

In JavaScript is een block een verzameling van gerelateerde statements ingesloten in accolades ("{}"). U kan  bijvoorbeeld een block met instructies zetten na een if (condition) block, om aan te geven dat de interpreter de code in het block moet uitvoeren als de voorwaarde waar is, of dat het hele block moet overgeslagen worden als de woorwaarde onwaar is.
