DeÂ call()Â methode staat het toe dat een functie of methode van een object om te worden toegewezen en aangeroepen voor een ander object.

call kan gebruikt worden om constructors voor een object aan elkaar te ketenen, vergelijkbaar met de werkwijze in Java. In dit voorbeeld hebben we een anonieme functie, en gebruiken we call om deze aan te roepen voor elk object in een array.   { species: 'Whale', name: 'Fail' }

Call gebruiken om een functie aan te roepen en een context te geven aan 'this'.

In het onderstaande voorbeeld zal de waarde van this gebonden zijn aan het object obj wanneer we greet aanroepen.
