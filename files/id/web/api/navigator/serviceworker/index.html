Navigator.serviceWorker adalah properti yang hanya bisa dibaca, mengembalikan objek ServiceWorkerContainer untuk dokumen terkait, yang memberikan akses untuk registrasi, penghapusan, upgrade, dan berkomunikasi dengan ServiceWorker.

Sintaks

var workerContainerInstance = navigator.serviceWorker;

Nilai

ServiceWorkerContainer.
