Properti Math.E mewakili dasar logaritma alami, e,  mendekati 2.718.

Math.E=e≈2.718\mathtt{\mi{Math.E}} = e \approx 2.718

Sumber untuk contoh interaktif ini disimpan di repositori GitHub. Apabila anda ingin berkontribusi dalam proyek contoh interaktif tersebut, silahkan klon https://github.com/mdn/interactive-examples dan kirimi kami sebuah pull request.

Property attributes of Math.E
Writable no
Enumerable no
Configurable no

Deskripsi

Karena E adalah sebuah properti matematika statis, saudara selalu menggunakannya sebagai Math.E, daripada sebagai sebuah properti dari objek matematika yang saudara buat. (Math bukan sebuah konstruktor).

Contoh

Menggunakan Math.E

Fungsi berikut menampilkan nilai e:

function getNapier() {
  return Math.E;
}

getNapier(); // 2.718281828459045
