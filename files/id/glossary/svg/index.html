Scalable Vector Graphics (SVG) merupakan format gambar vektor 2D berbasis sintaks XML.

W3C mulai mengerjakan SVG pada akhir 1990s, namun SVG baru menjadi populer ketika Internet Explorer 9 keluar dengan dukungan SVG. Semua browsers sekarang mendukung SVG.

Berbasis pada sintaks XML, SVG dapat diberikan gaya dengan menggunakan CSS dan dibuat semakin interaktif dengan menggunakan JavaScript. HTML5 saat ini memungkinkan direct embedding pada tags SVG pada dokumen HTML.

Sebagai sebuah format gambar vektor, grafis SVG dapat dirubah skalanya tanpa batas, sehingga membuat SVG sangat berharga dalam responsive design, karena Anda dapat membuat antarmuka elemen dan grafis yang dapat menyesuaikan pada segala ukuran layar. SVG juga menyediakan banyak alat-alat yang berguna, seperti clipping, masking, filter, dan animasi.
