HTTP (HyperText Transfer Protocol) adalah protokol sederhana yang memunginkan transfer file di Web. HTTP bersifat tekstual (semua komunikasi dilakukan dengan teks biasa) dan stateless (tidak ada komunikasi yang menyadari komunikasi sebelumnya).
