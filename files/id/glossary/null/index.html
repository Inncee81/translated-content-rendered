Dalam ilmu komputer, nilai null mewakili referensi suatu nilai yang tidak ada, umumnya sengaja untuk meng-tiadakan, hampa, atau Invalid object atau pengalamatan. Banyak arti referensi null pada penerapan bahasa pemrogramman lainnya.

Dalam JavaScript, null adalah salahsatu dari Nilai Primitif.
