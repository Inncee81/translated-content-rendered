Podsumowanie

Własność CSS clip określa, jaka część elementu jest widoczna.

  • Wartość początkowa: auto
  • Stosowana do: elementy pozycjonowane absolutnie
  • Dziedziczona: nie
  • Procenty: n/a
  • Media: visual
  • Wartość wyliczona: Dla wartości prostokątnych, prostokąt składający się z czterech wyliczonych długości, w innych przypadku - jako określone

Składnia

clip: <shape> | auto | inherit

Wartości

<shape>
określa kształt w formie rect(<top>, <right>, <bottom>, <left>);
auto
 
inherit
 

Przykłady

  img.clip04 {
    clip: rect(10px, 20px, 20px, 10px);
  }
