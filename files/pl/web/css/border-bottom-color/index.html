Podsumowanie

border-bottom-color ustawia kolor dolnego obramowania elementu, poprzez wartość koloru lub słowo kluczowe transparent.

  • Wartość początkowa: własność color
  • Stosowana do: wszystkich elementów
  • Dziedziczona: nie
  • Procenty: niedostępne
  • Media: visual
  • Wartość wyliczona: jeśli wzięta z własności 'color', wtedy wartość wyliczona własności 'color'; w innym przypadku jako określone

Składnia

border-bottom-color: color | transparent | inherit

Wartości

color 
Kolor może być zdefiniowany jako szestnastkowa wartość RGB, zwykła wartość RGB lub poprzez użycie jednego z predefiniowanych słów określających kolory.
transparent 
Element nie ma własnego koloru, zamiast tego pokazuje kolor elementu pod spodem.

Powiązane własności

  • border-color
  • border-left-color
  • border-right-color
  • border-top-color

Przykłady

Zobacz przykład

element {
	width: 300px;
        padding: 15px;
        border-bottom-size: 1px;
        border-bottom-style: solid;
        border-bottom-color: #000;
}
