Podsumowanie

list-style-image określa plik graficzny, który ma zostać użyty jako znacznik elementu listy.

  • Wartość początkowa: none
  • Stosowana do: elementów z 'display: list-item'
  • Dziedziczona: tak
  • Procenty: niedostępne
  • Media: visual
  • Wartość wyliczona: bezwzględny URI lub none

Składnia

list-style-image: <uri> | none | inherit

Wartości

uri
Adres pliku graficznego.

Przykłady

HTML

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

CSS

ul {
  list-style-image: url("https://mdn.mozillademos.org/files/11981/starsolid.gif")
}
