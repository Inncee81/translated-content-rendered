Podsumowanie

Element <i>...</i> powoduje wyświetlenie tekstu pisanego kursywą.

Przykład

<p>To jest pierwsze wyrażenie. <i>Całe to wyrażenie jest napisane kursywą..</i></p>

Rezultat

To jest pierwsze wyrażenie. Całe to wyrażenie jest napisane kursywą.
