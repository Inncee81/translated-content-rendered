Standard Generalized Markup Language (SGML) jest specyfikacją ISO służącą do definiowania deklaratywnych języków znaczników.

W sieci, HTML 4, XHTML i XML są popularnymi językami opartymi na SGML. Warto zauważyć, że od piątej edycji HTML nie jest już oparty na SGML i ma własne reguły parsowania.
